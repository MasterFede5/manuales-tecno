# Manuales Tecno

Repositorio de trabajo para los manuales de Colegio Nuevo Tecno.

## Apertura rápida

El índice vigente de los primeros semestres está en:

```text
MANUALES_BASE_DEPURADA/01_fuente_principal_markdown/manuales/primeros-semestres-vigentes-index.html
```

Desde ese archivo se puede abrir cada manual en HTML o PDF.

## Estructura principal

```text
MANUALES_BASE_DEPURADA/
  00_temario_estructura/
  01_fuente_principal_markdown/
  02_referencia_gemini_markdown/

IMAGENES_RECUPERADAS_511_JPG/

```

## Notas

- Los manuales vigentes se encuentran dentro de `MANUALES_BASE_DEPURADA`.
- Las imágenes recuperadas se conservan como JPG individuales.
- El archivo `visuales.rar` no se sube porque supera el límite de GitHub; su contenido ya está preservado como imágenes individuales.
- Los PDFs y HTML principales están versionados para poder abrirlos desde otro equipo.
