# 8.5 Podcasting con IA

El podcast es **el formato perfecto para audiencia académica/profesional**: largo (30+ min), profundo, escuchable mientras manejas o caminas. Con IA puedes producirlo en 1/4 del tiempo. Y puedes hacerlo solo.

::concepto{titulo="Podcasting con IA"}
Tres modos:
- **Narrado solo con TTS** — escribes guion, ElevenLabs lo narra. Útil para episodio único o serie de cápsulas.
- **Conversacional NotebookLM** — generas podcast de 2 voces automático desde tus PDFs.
- **Híbrido humano + IA** — grabas tú, usas Descript para editar, IA para música y distribución.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Si te dedicas a tema técnico/educativo, un podcast te posiciona como autoridad. Antes requería socio, estudio, equipo. Hoy una persona con ElevenLabs ($5/mes) y Descript puede producir series profesionales.
::/interioriza

## Modo 1 — TTS puro (más rápido)

```
1. Escribe guion en ChatGPT/Claude (15 min).
2. ElevenLabs convierte a audio (5 min).
3. Suno genera música intro/outro (5 min).
4. Descript ensambla y exporta (5 min).
Tiempo total: 30 min para episodio de 10-20 min.
```

**Limitación.** Voz suena bien pero "menos humana" que tú real. Para serie corta funciona; para podcast personal pesado, mejor grabar.

## Modo 2 — NotebookLM Audio Overview

```
1. Cargas tus fuentes a NotebookLM (2 min).
2. Generate Audio Overview de 15-25 min (5 min de espera).
3. Descargas MP3.
Tiempo total: 7 minutos para episodio conversacional 2 voces.
```

**Limitación.** No personalizable a fondo. Pero **excelente para resumen de un libro o tema** estilo podcast educativo.

## Modo 3 — Híbrido humano + IA

```
1. Graba tú con celular o mic decente (20 min audio crudo).
2. Sube a Descript: transcript automático + Studio Sound.
3. Edita transcript: borra ehhs, tangentes (10 min).
4. Genera música intro/outro en Suno (5 min).
5. Multitrack si tienes invitado.
6. Exporta MP3 (1 min).
Tiempo total: 40 min para episodio 15-20 min.
```

**Ventaja.** Tu voz real, edición fácil. Recomendado para el podcast "tuyo de verdad".

## Distribución

Cada episodio publica simultáneamente en:
- **Spotify** (plataforma principal).
- **Apple Podcasts**.
- **YouTube** (con waveform visual + thumbnail estático).
- **Spotify for Podcasters** (gestión central, gratis).

## Aplicación al case study

Para tu tutor educativo:
- **Serie corta de cápsulas** (5 episodios de 10 min) sobre los fundamentos de IA Básica.
- **Resumen semanal** de tu materia (NotebookLM con tus PDFs de la semana).
- **Entrevistas** con compañeros expertos en algo.
