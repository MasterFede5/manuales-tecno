# 8.2 Storytelling y guion con IA

Una explicación pasiva aburre. Una historia bien contada captura. Esta sección te enseña las **estructuras narrativas** clásicas y cómo aplicarlas con IA a tu contenido educativo.

::concepto{titulo="Storytelling estructural"}
Cuatro estructuras útiles para contenido educativo:

1. **Three-Act Structure (Aristóteles)** — setup, conflicto, resolución.
2. **Hero's Journey (Campbell)** — llamado, prueba, transformación.
3. **PAS (Problem-Agitate-Solve)** — copy publicitario.
4. **AIDA (Attention-Interest-Desire-Action)** — funnel de venta.

Para video corto educativo (60-90s): **Hook → Concept → Aplicación → CTA**.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Tu cerebro está cableado para historias. Una explicación abstracta de Bernoulli se olvida en horas. La misma explicación dentro de "imagina que estás en tu coche F1 a 80 km/h..." se queda. Storytelling no es decoración — es la manera **eficiente** de transmitir información.
::/interioriza

## Three-Act Structure aplicado a video educativo de 60s

```
ACTO 1 (0-15 s) — SETUP / HOOK
"¿Por qué un avión Airbus de 575 toneladas no se cae?"
[Pregunta gancho que genera curiosidad]

ACTO 2 (15-45 s) — CONFLICTO / DESARROLLO
"Si la gravedad lo jala hacia abajo con miles de toneladas, ¿qué lo
mantiene arriba? Spoiler: no es magia, es Bernoulli."
[Concepto con tensión]

ACTO 3 (45-60 s) — RESOLUCIÓN
"El aire pasa más rápido encima del ala que abajo. Eso crea menor
presión arriba. La presión empuja hacia arriba. Listo: Bernoulli en
acción. ¿Quieres saber cómo aplica al alerón de un F1 al revés?
Sigue para más."
[Resolución + CTA]
```

## Prompt para guion con IA

```
Eres guionista educativo. Genera guion para video TikTok de 60 segundos
sobre [tema]. Audiencia: estudiantes 15-22 años.

Estructura:
- Hook (0-5 s): pregunta que genere curiosidad.
- Setup (5-15 s): contexto del problema.
- Desarrollo (15-45 s): explicación con analogía cotidiana.
- Resolución (45-55 s): respuesta clara.
- CTA (55-60 s): "comparte si te explotó la cabeza".

Tono: conversacional, sin jerga académica. Voz mexicana neutra.
Ritmo: cambio de idea cada 8 s. Lenguaje visual: describe imagen
para cada bloque.
```

## Hooks que funcionan

Los primeros 5 segundos deciden si alguien queda. Tipos de hook:

| Tipo | Ejemplo |
|---|---|
| **Pregunta** | "¿Por qué tu cerebro olvida lo que estudia?" |
| **Estadística shock** | "El 80% de tus apuntes los olvidas en 24 h" |
| **Promesa** | "Te enseño Bernoulli en 60 segundos" |
| **Contraintuitivo** | "Estudiar más no funciona. Esto sí." |
| **Anécdota** | "Reproduje a mano 30 papers científicos. Esto aprendí" |

> **Tip pro.** Reescribe tu hook 5 veces antes de publicar. Es la diferencia entre 100 vistas y 100,000.

## Storytelling para post LinkedIn

Estructura para post largo:

```
1. HOOK (1ª línea): pregunta o afirmación contraintuitiva.
2. STAKES (2ª-3ª): por qué esto te importa.
3. NARRATIVA (4-10 líneas): cuento con conflicto y resolución.
4. LECCIÓN: 3-5 bullets con aprendizajes.
5. CTA: pregunta a la audiencia.
```
