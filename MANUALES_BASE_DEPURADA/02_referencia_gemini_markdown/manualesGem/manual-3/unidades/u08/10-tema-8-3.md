# 8.3 Diseño con Canva Magic Studio y Adobe Express

Antes diseñar un post de Instagram requería conocer Photoshop. Hoy, con Canva Magic Studio o Adobe Express, **describes lo que quieres** y la IA lo genera. La barrera técnica desapareció.

::concepto{titulo="Canva Magic Studio y Adobe Express"}
**Canva Magic Studio.** Suite completa de funciones IA dentro de Canva:
- **Magic Write** — texto generativo.
- **Magic Design** — plantilla automática.
- **Magic Eraser** — quita objetos.
- **Magic Edit** — modifica zona específica.
- **Magic Resize** — un diseño para múltiples formatos.
- **Magic Switch** — cambia idioma + estilo.

**Adobe Express.** Equivalente Adobe, integrado con Firefly:
- Generación de imagen.
- Plantillas profesionales.
- Edición vectorial.
- **Licencia comercial limpia** (Firefly entrenado solo con Adobe Stock).
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Tu LinkedIn, presentación escolar, infografía de tarea, póster del proyecto. Todo se ve más profesional con diseño. Antes era barrera; hoy con AI cualquiera produce piezas pulidas. Quien NO usa estas herramientas envejece visualmente.
::/interioriza

## Canva Magic Studio en 5 minutos

**Acceso:** canva.com. Free tier abundante; Pro $13/mes.

**Pipeline rápido:**
```
1. Eliges tipo de pieza (post Instagram, presentación, infografía).
2. Magic Design: prompt corto → 5 plantillas generadas.
3. Eliges una. Magic Write para texto.
4. Imágenes: subes propias o generas con DALL·E integrado.
5. Magic Resize: misma pieza para Instagram, LinkedIn, Twitter.
6. Descargar en PNG/PDF.
```

## Adobe Express con Firefly

**Acceso:** express.adobe.com. Free; Premium $10/mes.

**Diferencias vs Canva:**
| Aspecto | Canva | Adobe Express |
|---|---|---|
| Plantillas | Ilimitadas, varias estilos | Más profesionales, pocas |
| AI imagen | DALL·E + Stable Diffusion | **Firefly (licencia limpia)** |
| Edición vectorial | Limitada | **Avanzada** |
| Mejor para | Estudiante, social | **Profesional, comercial** |

> **Recomendado.** Canva para social media casual. Adobe Express si tu pieza va a uso comercial real (cliente, tienda).

## Pipeline de identidad visual

Para que tu contenido se vea **coherente**:

```
1. Define BRAND KIT: 2 colores, 2 tipografías, 1 logo, 3 estilos foto.
2. Configura Brand Kit en Canva o Adobe.
3. Cada pieza nueva la inicias desde tu Brand Kit.
4. Resultado: 30 piezas que se ven obviamente "tuyas".
```

## Limitaciones

- **Imágenes generadas con sello AI** — algunos competidores las identifican. Para portafolio profesional, mejor fotos propias.
- **Plantillas reconocibles.** Si todos usan Canva default, todos se ven igual. Personaliza.
- **Free tier limitado en exportaciones.** Para publicación frecuente, paga Pro.
