## Banco de ejercicios — Unidad 08

> Banco de práctica para creación de contenido con IA: redacción, storytelling, diseño, edición de video, podcasting y pipeline integrado. Privilegia ejercicios de **redacción de hooks**, **decisión de herramienta** y **evaluación de outputs**. Resuelve antes de mirar la clave.

### Bloque A — Redacción (8.1)

::act-mcq{titulo="Redacción asistida por IA"}
1. Para mantener **tu voz auténtica** al redactar un blog con IA, lo más eficaz es:
   - [ ] Pedirle que sea "creativa"
   - [x] Darle 2-3 ejemplos de tu trabajo pasado como contexto
   - [ ] Usar el modelo más nuevo
   - [ ] Pagar plan Pro

2. Un buen "hook" para un post de redes:
   - [ ] Tiene 5 párrafos
   - [x] Cabe en una sola frase y genera curiosidad o tensión
   - [ ] Usa muchos hashtags
   - [ ] Empieza con "Hola amigos"
::/act-mcq

::act-fill{titulo="Anatomía AIDA aplicada"}
La estructura clásica AIDA tiene 4 partes:

1. **A** = _____________ (lo que captura)
2. **I** = _____________ (lo que sostiene)
3. **D** = _____________ (lo que conecta emocionalmente)
4. **A** = _____________ (lo que pides al lector hacer)

Para un blog de 1500 palabras, asignas aproximadamente _____________ palabras a A, _____________ a I, _____________ a D y _____________ a A. La voz de tu marca se mantiene si das _____________ de contexto al modelo y le pides _____________ específico.
::/act-fill

::act-case{titulo="Redacta 4 hooks distintos para el mismo tema" lineas=10}
Tema: "Por qué dormir 8 horas mejora tu rendimiento académico". Redacta 4 hooks (1 frase cada uno) con estilos distintos: a) **estadística sorprendente**, b) **pregunta provocadora**, c) **historia personal**, d) **mito a desmontar**.
::/act-case

### Bloque D — Edición de video (8.4)

::act-tf{titulo="CapCut AI vs Descript"}
1. CapCut tiene generador de subtítulos automáticos en 60+ idiomas. ( ) ____________
2. Descript permite editar video eliminando palabras del transcript. ( ) ____________
3. La función "Studio Sound" de Descript limpia ruido de fondo. ( ) ____________
4. Descript Overdub permite clonar tu voz para cubrir errores sin re-grabar. ( ) ____________
5. CapCut Pro es más caro que Descript Pro. ( ) ____________
::/act-tf

::act-order{titulo="Ordena el flujo de edición de un video corto"}
[ ] Importa los assets generados (audio, imágenes, B-roll)
[ ] Pega la voz como track principal
[ ] Coloca imágenes con cortes cada 8-10 s
[ ] Genera subtítulos automáticos y revisa
[ ] Ajusta música a -20 dB para no tapar voz
[ ] Aplica color grading o filtro consistente
[ ] Verifica duración objetivo (60 s ± 5)
[ ] Exporta MP4 1080p
::/act-order

## Clave de respuestas

**Bloque A — MCQ:** 1-b · 2-b. **Fill:** Atención · Interés · Deseo · Acción · 100 · 600 · 600 · 200 · 2-3 ejemplos · formato. **Caso (ejemplos):** a) "Si duermes menos de 7 horas, tu rendimiento equivale al de alguien con 0.5 de alcohol en sangre". b) "¿Sabes que estudiar 3 horas tras 5 horas de sueño rinde menos que estudiar 1 hora tras 8?". c) "El semestre que dejé de dormir bien fue el que reprobé Cálculo, y no fue casualidad". d) "Estudiar más horas no es estudiar mejor: dormir es la otra mitad del aprendizaje".

**Bloque D — V/F:** 1-V · 2-V · 3-V · 4-V · 5-F (Descript suele ser más caro). **Order:** importa → voz como track principal → imágenes cortes 8-10s → subtítulos → ajusta música → color grading → verifica duración → exporta.

**Bloque E — MCQ:** 1-b · 2-b.
