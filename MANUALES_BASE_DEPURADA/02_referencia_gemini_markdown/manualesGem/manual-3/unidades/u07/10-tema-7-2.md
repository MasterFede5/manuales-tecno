# 7.2 Gems de Gemini

**Gems** es la respuesta de Google a los GPTs. Mismo concepto: asistentes personalizados con instrucciones específicas. Diferencias: ecosistema, distribución y algunas limitaciones.

::concepto{titulo="Gem de Gemini"}
Un **Gem** es una versión personalizada de Gemini con system prompt fijo. Disponible en gemini.google.com.

**Componentes:**
1. Name + description.
2. Instructions (system prompt).
3. Capacidad opcional de cargar archivos.
4. Distribución limitada (no hay GPT Store equivalente).

**Acceso:** gratuito con cuenta Google + Gemini Advanced ($20/mes para versiones avanzadas).
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Si eres usuario fuerte de Workspace y prefieres Google sobre OpenAI, Gems te da el mismo concepto que GPTs sin cambiar de ecosistema. Y Gemini Advanced viene incluido con Google One Premium ($20/mes que también te da 2TB Drive), así que costos competitivos.
::/interioriza

## Cómo crear un Gem

```
1. Ve a gemini.google.com.
2. Menú izquierdo: "Gem manager".
3. "+ New Gem".
4. Llena: Name, Description, Instructions.
5. Opcionalmente carga archivos como knowledge.
6. Guarda y prueba.
```

## Diferencias clave vs GPTs de OpenAI

| Aspecto | GPTs (OpenAI) | Gems (Google) |
|---|---|---|
| Knowledge files | Hasta 20 archivos | Sí (con plan adecuado) |
| Capabilities | Web, DALL·E, Code | Web, integración Workspace |
| Actions/APIs | Sí | No directamente |
| Distribución pública | GPT Store | Limitada (no hay store público equivalente) |
| Compartir | Link, store | Vía Workspace |
| Multimodal | GPT-4o multimodal | Gemini multimodal nativo (mejor) |
| Costo | $20/mes ChatGPT Plus | $20/mes Gemini Advanced |

## Cuándo elegir Gem sobre GPT

✅ **Elige Gem si:**
- Vives en Google Workspace.
- Necesitas integración con Drive, Gmail, Docs.
- Tu organización ya paga Workspace Enterprise.
- Quieres ventana de contexto grande (2M tokens en Gemini 2.0 Pro).

❌ **Prefiere GPT si:**
- Necesitas distribución pública (GPT Store).
- Quieres conectar APIs externas via Actions.
- Tu audiencia ya usa ChatGPT.
