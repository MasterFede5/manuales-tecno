# 7.4 Copilot Studio (Microsoft) — introducción

**Copilot Studio** es el builder enterprise de Microsoft. Diferente a GPTs/Gems/Projects: en lugar de prompt engineering puro, usa **flowcharts visuales** de conversación con topics, entidades y triggers.

::concepto{titulo="Copilot Studio"}
**Copilot Studio** es la plataforma no-code de Microsoft para construir copilots personalizados.

**Características distintivas:**
1. **Flowchart visual** de conversación (no solo prompt).
2. **Topics** y **entities** — patrones de intención y datos.
3. **Integración nativa** con SharePoint, Teams, Power Platform.
4. **Autenticación corporativa** Azure AD/Entra.
5. **Acciones automatizadas** vía Power Automate.

**Acceso:** $200/mes per tenant + $0.05 por mensaje.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Si vas a hacer **carrera profesional** en empresas grandes, Copilot Studio será probablemente la herramienta que verás. Aprenderlo mínimamente te abre puerta laboral. Para uso personal/estudiantil **no es para ti** — es overkill.
::/interioriza

## Anatomía de un copilot en Studio

```
Copilot: "Asistente RH Empresa X"

Topics (intenciones):
- "Vacaciones" → flujo: pregunta días → consulta calendar → genera solicitud.
- "Recibo nómina" → flujo: autentica → pulla SharePoint → manda PDF.
- "Contacto manager" → flujo: pregunta departamento → busca en directorio.

Entities (datos):
- Department (lista de 30 departamentos).
- DateRange (fecha inicio + fin).

Authentication:
- Azure AD (todos los empleados).

Actions (Power Automate):
- Crear ticket en ServiceNow.
- Enviar email aprobación.
```

## Diferencias clave

| Aspecto | GPTs / Gems / Projects | **Copilot Studio** |
|---|---|---|
| Approach | Prompt engineering | Flowchart visual + prompts |
| Costo | $20/mes | **$200+/mes per tenant** |
| Audiencia | Individual/equipo | **Empresa grande** |
| Curva aprendizaje | Baja | **Alta** |
| Integraciones | Limitadas | **M365 nativas + Azure** |
| Compliance | Variable | **Enterprise grade (HIPAA, SOC2)** |

## Cuándo aprender Copilot Studio

✅ Si trabajas en empresa con M365 Enterprise.
✅ Si tu rol incluye automatización corporativa.
✅ Si quieres CV con experiencia "no-code AI".

❌ Si eres estudiante uso personal — usa GPT/Gem/Project.
❌ Si tu organización es pequeña — usa GPT Store o herramientas más ligeras.

## Recursos para aprender

- **Microsoft Learn** — ruta oficial gratis con certificación.
- **Power Platform Community** — tutoriales en YouTube.
- **GitHub: PVATemplates** — plantillas de copilots.
