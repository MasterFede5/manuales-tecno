# 7.3 Claude Projects con instrucciones personalizadas

Anthropic implementó **Projects** distinto a GPTs/Gems: en lugar de un asistente público distribuible, son **carpetas de chats** con instrucciones y knowledge compartidas. Filosofía más enterprise/equipo.

::concepto{titulo="Claude Projects"}
Un **Claude Project** es una colección de chats con:
- **System prompt** compartido (instrucciones).
- **Knowledge base** persistente (archivos y links).
- **Artifacts** integrados (código, HTML, SVG generados).
- **Sharing por usuario/equipo** (no público).

**Acceso:** Claude Pro ($20/mes), Team o Enterprise.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Si vas a colaborar con equipo de proyecto durante meses (tesis, startup, cliente), Claude Projects te da: instrucciones consistentes para todos los chats, knowledge accesible por todo el equipo, historial de conversaciones agrupado, sin tener que recrear cada vez. Filosofía distinta a "asistente público" — más "espacio de trabajo persistente".
::/interioriza

## Anatomía de un Project

```
Project: "Tesis IA en Educación"

System prompt:
Eres mi asesor de tesis. Estilo APA 7. Mantén coherencia con la
estructura definida. Critica constructivamente. No suavices feedback.

Knowledge:
- 15 papers fundamentales (PDFs).
- Outline de tesis (Doc).
- Notas de asesor reuniones (txt).
- Marco teórico draft (Doc).

Chats agrupados:
- Chat 1: Capítulo 1 - introducción
- Chat 2: Capítulo 2 - marco teórico
- Chat 3: Capítulo 3 - metodología
- Chat 4: Defensa - preguntas anticipadas
```

## Diferencias vs GPTs y Gems

| Aspecto | GPTs | Gems | **Claude Projects** |
|---|---|---|---|
| Distribución pública | GPT Store | No | **No (privado/equipo)** |
| Compartir con desconocidos | ✓ | Limitado | **No** |
| Knowledge persistente | ✓ | ✓ | **✓** |
| Historial agrupado | No | No | **Sí (chats por project)** |
| Artifacts | No nativo | No | **Sí (HTML, SVG, código)** |
| Filosofía | Asistente público | Asistente personal | **Workspace de equipo** |

## Casos de uso ideales para Projects

1. **Tesis personal** (6+ meses).
2. **Equipo pequeño** trabajando en proyecto.
3. **Asesor + alumnos** compartiendo knowledge.
4. **Startup** con knowledge de producto.

## No ideal para Projects

- Distribución pública (usa GPT).
- Asistente para uso casual de muchos usuarios anónimos.

## Cómo crear

```
1. claude.ai → Sidebar → Projects → "+ New Project".
2. Nombra el Project.
3. Edita Custom Instructions (system prompt).
4. Agrega Project Knowledge: drag files o paste text.
5. Crea chats dentro del Project — todos heredan instrucciones + knowledge.
6. Comparte (Pro: solo tú; Team: con miembros).
```

## Aplicación al case study

Para tu tutor IA personal, Claude Projects es ideal cuando:
- El tutor es **para ti específicamente**, no público.
- Vas a tener **muchas conversaciones** sobre la misma materia.
- Quieres mantener **knowledge actualizada** semestre tras semestre.

Combina con GPT publicado si quieres también ofrecer una versión "tutor estudiantes" pública.
