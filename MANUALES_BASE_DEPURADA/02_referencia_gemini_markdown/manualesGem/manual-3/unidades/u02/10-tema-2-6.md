# 2.6 Comparativa práctica: cuándo usar cada uno

Cuatro plataformas excelentes pero diferentes. Si las conoces solo de oídas vas a elegir mal — porque "la mejor IA" depende de **qué quieres hacer y dónde vives digitalmente**. Esta sección te da criterios duros para decidir.

::concepto{titulo="Criterios de selección"}
Para elegir entre ChatGPT, Claude, Gemini y Copilot, evalúa cinco dimensiones:

1. **Tarea principal** — ¿escritura larga, código, análisis de docs, multimedia?
2. **Ecosistema actual** — ¿Google Workspace, Microsoft 365, ninguno?
3. **Presupuesto** — ¿gratis, $20/mes, $30/mes?
4. **Privacidad** — ¿cuánta información personal puedes compartir?
5. **Funciones distintivas** — ¿necesitas Artifacts, Voice, Computer Use?

No hay un ganador absoluto. Hay un **ganador por caso de uso**.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Tomarte 30 minutos en elegir bien la plataforma te ahorra meses de "fricción" con la herramienta equivocada. Y como tu tutor virtual va a ser una conversación de **largo plazo** —semanas, meses, años—, el mal match temprano se acumula. Esta es la decisión más estratégica del manual.
::/interioriza

## Comparativa por tarea concreta

| Tarea | Mejor opción | Por qué |
|---|---|---|
| **Resumen de PDF largo** | Claude Opus | Ventana 1M, no pierde detalle |
| **Generar imagen** | ChatGPT (DALL·E 3) o Gemini (Imagen 3) | Integrados nativos |
| **Redactar correo formal** | El que esté más cerca de tu inbox (Gemini si Gmail, Copilot si Outlook) | Cero copy/paste |
| **Código y debugging** | Claude o ChatGPT con o3 | Modelos razonadores |
| **Brainstorming en voz** | ChatGPT Voice | Mejor latencia y voz natural |
| **Análisis de Excel** | Copilot Pro | Integración celda directa |
| **Investigar tema con citas** | Perplexity (no comparado aquí, ver U5) | Especializado |
| **Tutor de cálculo** | Claude (Pedagogical style) o ChatGPT con o3 | Razonamiento paso a paso |

## Estrategia recomendada para este manual

**Mínimo para todo el manual:**
- Una cuenta gratis de **Claude** (claude.ai) — usaremos mucho artifacts y projects.
- Una cuenta gratis de **ChatGPT** (chatgpt.com) — usaremos GPTs en U7.

**Recomendado si tienes presupuesto:**
- **Claude Pro** ($20/mes) si vas a redactar mucho.
- O **ChatGPT Plus** ($20/mes) si quieres multimodal completo.

**No necesitas las cuatro de paga.** Empieza con dos gratis, identifica cuál se adapta mejor a tu flujo, y arriba a la de paga que más uses.

## Mi recomendación final para tu tutor

Para el case study **"Mi tutor IA personal"**, te recomiendo:
1. **Claude Pro** como base principal (redacción + projects + artifacts).
2. **ChatGPT Free** como complemento para GPTs cuando lleguen las unidades 7-8.

Esa combinación cubre el 95 % de los casos por $20/mes.
