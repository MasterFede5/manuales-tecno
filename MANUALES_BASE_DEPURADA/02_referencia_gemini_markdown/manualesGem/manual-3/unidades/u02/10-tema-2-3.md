# 2.3 Claude: artifacts, projects, styles y Computer Use

Claude es la apuesta de **Anthropic**, fundada por ex-investigadores de OpenAI que se fueron por desacuerdos sobre seguridad. La plataforma destaca por su **redacción cuidada**, **ventana de contexto enorme** y herramientas como Artifacts y Projects que cambian la forma de trabajar.

::concepto{titulo="Claude como plataforma"}
**Claude** es la interfaz web/móvil de Anthropic. Modelos en 2025: Claude Opus 4.7 (top), Sonnet 4.6 (balanceado), Haiku 4.5 (rápido y barato).

**Tres niveles:**
- **Free** — Sonnet 4.6 con límites diarios.
- **Pro** ($20 USD/mes) — Opus 4.7 + Projects + Artifacts.
- **Team** ($30/usuario/mes) — Workspace compartido.

**Funciones distintivas:**
1. **Artifacts** — panel lateral con código/tablas/HTML/SVG editables en vivo.
2. **Projects** — agrupa chats con knowledge base compartida (system prompt + PDFs).
3. **Styles** — perfiles de tono predefinidos o personalizados.
4. **Computer Use** — Claude puede tomar control de tu pantalla y hacer clics.
5. **Claude Code** — variante terminal para programadores.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Si vas a redactar mucho —ensayos, reportes, propuestas— Claude probablemente te dará mejores resultados que ChatGPT. Si manejas documentos grandes, su ventana de 1M de tokens (Opus 4.7) te permite cargar libros completos. Si quieres organizar tu trabajo en "proyectos" (uno para tesis, otro para club, otro para tu negocio), Projects los separa con knowledge base independiente. Saber esto te ahorra herramientas de terceros y te da una experiencia más estructurada.
::/interioriza

## Artifacts: el "documento que cobra vida"

Cuando Claude genera algo "documentable" —código, tabla, HTML, SVG, markdown largo— lo abre en un **panel lateral** llamado Artifact. Ahí puedes:
- Verlo renderizado (HTML como página, SVG como imagen, markdown como texto formateado).
- Editarlo a mano.
- Pedirle al chat que lo modifique sin rehacer todo: "*haz el botón más grande y cambia el color a azul*".
- **Descargarlo** como archivo.
- **Compartirlo** con un link público.

> **Casos de uso.** Generar páginas de aterrizaje HTML completas. Diseñar diagramas SVG. Producir tablas comparativas. Hacer presentaciones HTML que se renderizan inmediatamente. Es la diferencia entre que el modelo "te dé instrucciones para programar X" y que **te entregue X funcionando**.

## Projects: knowledge base por contexto

Un **Project** es una carpeta de chats con un system prompt y archivos compartidos. Ejemplo de uso:
- **Project "Tesis Albatros"** — system prompt: "ayúdame con mi tesis sobre IA en educación, mantén tono académico". Knowledge: PDFs de papers, mi outline, draft actual.
- **Project "Tutor Cálculo"** — system prompt: "eres mi tutor de cálculo, explicas paso a paso". Knowledge: programa del curso, ejercicios resueltos.
- **Project "Marca personal"** — system prompt: "eres mi coach de comunicación, mi tono es directo y casual". Knowledge: posts anteriores, biografía.

Cada chat dentro del Project hereda el contexto sin que tengas que repetirlo.

> **Diferencia clave con GPTs.** Los Projects son **privados tuyos**; los GPTs se pueden publicar. Projects son ideales para uso personal/equipo; GPTs para distribución pública.

## Styles: cambia el tono sin rehacer prompt

En el menú de Claude puedes elegir:
- **Default** — neutral.
- **Formal** — académico/profesional.
- **Concise** — respuestas cortas y directas.
- **Explanatory** — pedagógico, paso a paso.
- **Custom** — escribes tus propias instrucciones de estilo y se aplican a todos tus chats.

> **Recomendado para tu tutor.** Crea un style "Tutor pedagógico" con instrucciones: "Explica cada paso, da ejemplos, anticipa malentendidos comunes, hazme preguntas para verificar comprensión". Tendrás esa personalidad en todos tus chats.

## Computer Use: Claude maneja tu pantalla

Función de **Claude 3.5 Sonnet** (octubre 2024) y siguientes: el modelo puede recibir capturas de tu pantalla, mover el mouse y escribir en aplicaciones. Es la primera versión de un **agente** que ejecuta tareas reales en tu computadora.

> **Caso de uso real.** "Reserva una mesa para 4 personas en un restaurante italiano cerca de aquí para el viernes". Claude abre el navegador, busca, navega, llena el formulario y reserva.

> **Caveat 2025.** Sigue en beta. Errores comunes (clica donde no debe, se distrae). Útil para tareas repetitivas con verificación humana, no para procesos críticos.

## Claude Code: programar sin VS Code

Variante para desarrolladores que se ejecuta en terminal. Lee tu repositorio, edita archivos, corre tests, ejecuta comandos shell. Pago aparte ($20-200/mes según uso).

## Plan recomendado

**Para tu tutor virtual:** Claude Pro ($20/mes) es excelente alternativa a ChatGPT Plus si:
- Vas a hacer redacción larga y cuidada.
- Manejas documentos grandes.
- Quieres organizar trabajo en Projects.
- Te interesa Artifacts para producir código/diseño.

Versión Free es muy usable para U2-U3. Para Computer Use y Claude Code, suscripciones aparte.
