# 2.2 ChatGPT: interfaz, modelos, voz, Canvas y GPTs

ChatGPT no es solo un chat. Es la plataforma con **más funciones integradas** del mercado: voz, generación de imagen, búsqueda web, GPTs personalizados, Canvas, plugins. Saber qué tiene y cómo usarlo te ahorra horas y te abre opciones que mucha gente desconoce.

::concepto{titulo="ChatGPT como plataforma"}
**ChatGPT** es la interfaz web/móvil de OpenAI sobre los modelos GPT (4o, 5, o1, o3). Tres niveles de servicio:

- **Free** — GPT-4o limitado por mensajes/hora.
- **Plus** ($20 USD/mes) — GPT-5 + o3 + más mensajes + DALL·E + voz avanzada.
- **Pro** ($200 USD/mes) — uso ilimitado + acceso temprano + razonadores top.

**Funciones principales (2025):**
1. **Selector de modelos** — eliges según tarea.
2. **Voz** — conversaciones bidireccionales en tiempo real.
3. **DALL·E 3** — genera imágenes desde texto.
4. **Browsing** — busca en internet en vivo.
5. **GPTs** — versiones personalizadas tuyas o de terceros.
6. **Canvas** — editor lateral para texto/código largo.
7. **Memoria** — opcional, persiste preferencias.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Si solo usas ChatGPT como "Google con párrafos", desperdicias el 80 % de su capacidad. Combina texto, imagen, voz, búsqueda web y archivos en una sola conversación. La diferencia entre un usuario casual y un usuario potente está en si conoces y usas estas funciones. Esta sección te entrena para ser usuario potente.
::/interioriza

## Selector de modelos: cuándo usar cada uno

| Modelo | Cuándo elegir |
|---|---|
| **GPT-4o** | Conversación general, multimodal, rápido |
| **GPT-5** | Tareas complejas: ensayos largos, código, análisis |
| **o1 / o3** | Razonamiento matemático, lógica, planning |
| **GPT-4o mini** | Muchas consultas baratas, tareas simples |

**Trampa común.** Usar siempre el "más caro y nuevo" no siempre es mejor: para preguntas casuales, GPT-4o mini responde igual y más rápido. Reserva o3 para problemas que **realmente** requieran razonamiento paso a paso.

## Modo voz (Voice Mode)

Tap al ícono de auriculares en móvil → conversación en tiempo real. Tres usos potentes:
1. **Práctica de idioma** — habla en inglés, te corrige.
2. **Brainstorming caminando** — sin manos.
3. **Tutor mientras estudias** — le dictas el problema, te explica de viva voz.

> **Tip pro.** Activa "Advanced Voice Mode" en Plus: voz natural con tono e interrupciones. Es lo más parecido a hablar con persona real.

## DALL·E 3: imágenes desde texto

Escribe "imagina una tabla periódica con tema espacial" → genera imagen. Buenas prácticas:
- Sé específico: estilo, colores, composición.
- Usa imágenes generadas como **moodboards** para presentaciones.
- Para imágenes editables comerciales, usa Adobe Firefly (mismo costo, licencia más clara).

## Búsqueda web (Browsing)

Cuando preguntas algo después del corte de entrenamiento, ChatGPT puede buscar en vivo. Activación: ícono "🌐 Search" en el input. Ejemplo: "¿cuáles son las noticias de IA de esta semana?". El modelo navega, lee y responde con citas.

> **Limitación.** No siempre busca cuando debería. Si quieres garantizar búsqueda, ponlo explícito: "*Busca en internet…*" o usa **Perplexity** (especializado en búsqueda con IA, ver U5).

## GPTs personalizados

Un GPT es una **versión configurada** de ChatGPT con instrucciones, archivos y herramientas tuyas. Lo creas en *Explore GPTs → Create*. Define:
- Nombre y avatar.
- **Instrucciones** (system prompt: "eres tutor de cálculo I, explicas paso a paso, …").
- **Conocimiento** (archivos PDF/docx que el GPT puede consultar).
- **Capacidades** (búsqueda web, DALL·E, code interpreter).

> **Importante.** Los GPTs **NO entrenan al modelo base**. Solo le dan contexto adicional cuando alguien lo invoca. Es como darle un "uniforme" y "manuales".

Cubrirás GPTs en profundidad en U7 (es el pilar de tu tutor virtual).

## Canvas: editor lateral

Para textos o código largos, ChatGPT puede abrir un **panel lateral** estilo Google Docs donde:
- Puedes editar el texto manualmente y pedirle al modelo que lo retoque.
- Hay shortcuts: "shorten", "polish", "translate", "add code comments".
- El modelo solo modifica lo que seleccionas, no rehace todo.

Es muy útil para ensayos, blog posts, código que vas iterando.
