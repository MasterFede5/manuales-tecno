# 6.4 Documentos largos: Claude Projects, ChatGPT Canvas y Notion AI

Cuando escribes un documento de 10+ páginas (tesis, ensayo grande, reporte profesional), el chat clásico se vuelve incómodo: pierdes contexto, te toca repegar versiones, no puedes editar partes específicas sin rehacer todo. Tres herramientas resuelven esto: **Claude Projects**, **ChatGPT Canvas** y **Notion AI**.

::concepto{titulo="IA para documentos largos"}
Tres approaches:
- **Claude Projects.** Carpeta con system prompt + knowledge base + chats agrupados.
- **ChatGPT Canvas.** Editor lateral en chat; modificación quirúrgica de texto/código.
- **Notion AI.** AI nativa dentro de Notion; bloques /ai expanden o transforman.

Funciones clave:
1. **Persistencia** del contexto entre sesiones.
2. **Edición quirúrgica** sin rehacer todo.
3. **Knowledge base** propia.
4. **Iteración fluida**.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Tu próxima tesis o trabajo grande dura semanas. Sin estas herramientas, cada vez que vuelves al documento copias 10 párrafos para "darle contexto" al chat. Con Projects/Canvas/Notion AI, el contexto **vive** y avanzas. Diferencia entre proyecto que termines vs proyecto que abandones.
::/interioriza

## Claude Projects (la mejor opción para documentos largos)

**Acceso:** claude.ai con plan Pro ($20/mes).

**Cómo funciona:**
1. Creas un Project: "Tesis IA en educación".
2. Configura **system prompt**: "*eres mi asesor de tesis. Estilo APA 7. Tono académico. Conoce los archivos cargados.*"
3. Cargas **knowledge base**: PDFs de fuentes, tu outline, draft actual, notas de asesor.
4. **Cada chat** dentro del Project hereda system prompt + knowledge.
5. Puedes tener 30 chats sobre distintos capítulos sin perder contexto.

**Ventajas únicas:**
- Ventana de contexto 200K-1M tokens.
- Knowledge base persistente.
- Compartir Project con tu equipo o asesor.
- **Artifacts** activados — generación de tablas, gráficos, código embebido.

> **Caso de uso real.** Tesis de 6 meses. Project con: marco teórico (10 papers), outline, drafts por capítulo, notas de asesor. Cada vez que vuelves a trabajar, abres un chat nuevo y todo el contexto está. Asesor puede unirse al Project para feedback.

## ChatGPT Canvas

**Acceso:** ChatGPT Plus ($20/mes).

**Cómo funciona:**
1. En un chat, pides documento largo: "*genera ensayo de 1500 palabras sobre X*".
2. ChatGPT abre **panel lateral Canvas** con el documento.
3. **Editas a mano** el documento o le pides cambios al chat.
4. **Selecciona párrafo** + comando: "Shorten", "Polish", "Translate", "Add citations", "Adjust tone".
5. ChatGPT modifica solo lo seleccionado sin rehacer todo.

**Ventajas únicas:**
- Quirúrgica edición.
- Versiones automáticas.
- También funciona con código.

> **Caso de uso real.** Estás iterando un ensayo. Seleccionas el párrafo de la introducción, pides "más enganchador". ChatGPT modifica solo ese párrafo. El resto se mantiene intacto. **Sin Canvas tendrías que rehacer todo cada vez**.

## Notion AI

**Acceso:** notion.com. AI features $10/u/mes adicional al plan Notion.

**Cómo funciona:**
- **Comando /ai** dentro de cualquier bloque.
- "Expande este bullet en párrafo".
- "Resume esta página en 5 puntos".
- "Traduce a inglés".
- "Genera plan de acción a partir de este texto".

**Ventajas únicas:**
- Vive dentro de Notion (donde ya tienes notas).
- Knowledge base = todo tu Notion.
- Q&A: "¿qué dice mi nota sobre X?".

> **Caso de uso real.** Tu segundo cerebro es Notion. Tienes 200 notas. Notion AI puede responder preguntas usando todas tus notas como contexto: "*¿qué dije sobre Bernoulli en mis apuntes de física?*".

## Cuándo usar cada uno

| Caso | Herramienta |
|---|---|
| Tesis o documento de 6 meses | **Claude Projects** |
| Iterar ensayo 5-10 veces | **ChatGPT Canvas** |
| Knowledge base personal vivo | **Notion AI** |
| Documento que entregarás mañana | Canvas o Projects |
| Reportes corporativos | Copilot Word + Projects |

## Workflow combinado

Para tesis seria, combina:
```
1. NotebookLM para procesar fuentes (U5).
2. Perplexity para encontrar nuevas fuentes (U5).
3. Claude Project para redacción y mantenimiento de drafts.
4. ChatGPT Canvas para iteración quirúrgica de capítulos.
5. Notion para apuntes y to-do.
```

## Limitaciones

- **Privacidad.** Todos los servicios suben tu trabajo a la nube. Para tesis sensible, usa Llama 4 local con Ollama.
- **Costo acumulado.** Si pagas Claude Pro + ChatGPT Plus + Notion AI = $50/mes. Decide cuál es esencial.
- **Vendor lock-in.** Si dejas Anthropic, perderías Projects. Exporta drafts regularmente.
