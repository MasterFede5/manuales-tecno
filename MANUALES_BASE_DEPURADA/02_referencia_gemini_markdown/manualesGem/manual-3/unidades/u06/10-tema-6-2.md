# 6.2 Hojas: GPT for Sheets, Copilot Excel, Bardeen

Las hojas de cálculo dejaron de requerir años de Excel para ser útiles. Con IA puedes:
- Clasificar 1000 filas en segundos con lenguaje natural.
- Generar fórmulas complejas describiéndolas en español.
- Crear pivots y gráficos sin saber sintaxis.
- Hacer análisis estadístico sin programar.

::concepto{titulo="IA en hojas de cálculo"}
Tres approaches:
- **GPT for Sheets** (extensión de Google Sheets) — funciones `=GPT()`, `=GPT_TABLE()`, etc. Pago por API key.
- **Microsoft Copilot Excel** ($30/u/mes M365 Copilot) — integrado nativamente.
- **Bardeen** o **Sourcetable** — alternativas con AI nativo.

Capacidades:
1. Clasificación masiva.
2. Extracción de info de texto largo.
3. Generación de fórmulas desde lenguaje natural.
4. Análisis estadístico conversacional.
5. Visualización automática.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Si llevas hojas con datos (calificaciones, presupuesto, tareas, lista de invitados), IA te ahorra horas. Y te abre análisis que antes no podías hacer porque "no sabías Excel". Estudiantes que dominan IA en hojas terminan tareas de estadística básica en minutos en lugar de horas.
::/interioriza

## GPT for Sheets

**Acceso:** extensión gratuita en Google Workspace Marketplace. Requiere API key de OpenAI (~$5 USD/mes uso casual).

**Funciones más usadas:**

```
=GPT("clasifica sentimiento como positivo/negativo/neutral", A2)
→ "positivo"

=GPT("traduce a inglés", B2)
→ "I love this product"

=GPT("extrae el email de este texto", C2)
→ "user@email.com"

=GPT_TABLE("genera tabla con 10 países y su capital", "país,capital")
→ tabla 10x2

=GPT("resume en 1 oración", D2)
→ resumen
```

**Caso real escolar.** Tienes lista de 50 alumnos en Sheets con sus comentarios sobre la materia. Quieres clasificar sentimiento. En B2: `=GPT("Clasifica el sentimiento como POSITIVO, NEUTRAL, NEGATIVO", A2)`. Arrastras hacia abajo. Listo.

## Copilot en Excel

**Acceso:** $30/u/mes con M365 Copilot.

**Funciones:**

### Lenguaje natural en celda

Botón Copilot lateral. Escribes: "*Genera tabla pivote con ventas por mes y producto*" → se crea automáticamente.

### Análisis automático

"*Analiza esta tabla y dame insights*" → genera resumen de patrones, outliers, tendencias.

### Generar fórmula

"*Quiero contar las celdas donde la columna B sea 'pendiente' y la C sea > 100*" → `=COUNTIFS(B:B,"pendiente",C:C,">100")`.

### Crear gráfico

"*Genera gráfico que muestre tendencia mensual de gastos*" → genera con título y formato.

## Casos de uso para estudiante

### Caso 1 — Clasificar 100 fuentes para tesis

```
Columna A: títulos de papers.
Columna B: =GPT("Clasifica como TEÓRICO, EMPÍRICO o MIXTO", A2)
Columna C: =GPT("¿Es relevante para mi tesis sobre IA en educación bachillerato? Responde SÍ/NO/PARCIAL", A2)

→ Filtras los SÍ. Lees solo esos.
```

### Caso 2 — Análisis de calificaciones

```
Tabla con calificaciones de 3 semestres.
Copilot en Excel: "Genera análisis con: promedio por materia, tendencia,
materias en riesgo, recomendación de plan de estudio."

→ Reporte completo en 30 segundos.
```

### Caso 3 — Limpieza de datos

```
Lista de 200 nombres con errores tipográficos.
=GPT("normaliza este nombre, corrige tipos, capitaliza correctamente", A2)
=GPT("extrae el primer nombre", A2)
=GPT("extrae el apellido", A2)

→ Datos limpios automáticamente.
```

## Costos y trade-offs

| Plataforma | Costo | Cuándo elegir |
|---|---|---|
| **GPT for Sheets** | API ~$5/mes uso casual | Vives en Google Sheets, casos puntuales |
| **Copilot Excel** | $30/u/mes M365 | Vives en Excel, análisis intensivo |
| **Bardeen / Sourcetable** | $20-50/mes | Quieres workflow específico no-code |

**Tip de costo.** GPT for Sheets cobra por token. Si haces 500 filas con prompts cortos, gastas ~$1. Si haces 10 000 filas con prompts largos, $50+. **Estima antes de procesar masivamente**.

## Limitaciones

- **Velocidad.** GPT for Sheets procesa 1 fila a la vez (~3 s c/u). Para 500 filas tomas ~25 min.
- **Errores en clasificación.** A veces el modelo se equivoca; verifica al menos 5% de los resultados.
- **Privacidad.** Los datos van a OpenAI/Microsoft. NO uses con info confidencial real (datos personales, financieros sensibles).
