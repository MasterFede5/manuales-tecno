# Unidad 02

## Fuerzas, Newton y Gravitación

> Si la cinemática responde *cómo* se mueven las cosas, las Leyes de Newton responden *por qué*. Una sola idea —que todo cambio de movimiento requiere una fuerza— es la base de la mecánica clásica, la ingeniería moderna y todo lo que vuela, rueda o se mantiene en pie.
