# 10.1 Comparativo de planes (ChatGPT, Claude, Gemini, Copilot)

Tu institución no debería pagar 3 plataformas premium para 50 personas. Pero sí debería elegir bien la principal. Esta lección te da el comparativo honesto 2025.

::concepto{titulo="Planes empresariales de IA"}
Los **planes empresariales de IA** son las suscripciones que ofrecen los principales proveedores con features adicionales, soporte y compliance. Diferencian de planes individuales en:

- **Multi-usuario** con admin centralizado.
- **SSO** corporativo (SAML, OIDC).
- **Audit logs** para cumplimiento.
- **Data privacy**: tus datos NO entrenan el modelo (a diferencia de planes free).
- **Soporte prioritario.**
- **Compliance** (SOC 2, ISO 27001, HIPAA con planes específicos).

4 jugadores principales: **ChatGPT** (OpenAI), **Claude** (Anthropic), **Gemini** (Google), **Copilot M365** (Microsoft). Cada uno con 2-3 niveles.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
50 staff × 3 plataformas × $30/mes = $4 500/mes = $54 000/año. La decisión de proveedor mal hecha cuesta cientos de miles. La bien hecha mantiene presupuesto y maximiza valor.
::/interioriza

## Comparativo (2025)

### ChatGPT (OpenAI)

| Plan | Precio/mes | Features clave |
|---|---|---|
| Plus | $20/usuario | GPT-5, image gen, code interpreter, Custom GPTs, Canvas |
| Team | $25-30/usuario (≥2) | Plus + workspace compartido + admin |
| Enterprise | desde $60/usuario | + SSO, audit logs, unlimited GPT-5, priority compute |

**Fortalezas:** modelo más versátil, mainstream, ecosistema GPT Store.

**Debilidades:** datos pueden quedar en logs (excepto Enterprise), Operator solo en Pro.

### Claude (Anthropic)

| Plan | Precio/mes | Features clave |
|---|---|---|
| Pro | $20/usuario | Opus 4.7, Projects, Artifacts |
| Team | $30/usuario | + workspace + admin centralizado |
| Enterprise | personalizado | + SSO, BYOK, compliance, SOC 2 |

**Fortalezas:** mejor razonamiento de la categoría, Artifacts excelentes, MCP nativo.

**Debilidades:** sin image gen, ecosistema menor que OpenAI.

### Gemini (Google)

| Plan | Precio/mes | Features clave |
|---|---|---|
| Advanced (Google One AI Premium) | $20/usuario | Gemini 2 Pro/Ultra, Deep Research, NotebookLM Plus |
| Workspace Business + AI | +$20/usuario sobre Workspace | Gemini en Gmail, Docs, Meet, Slides |
| Workspace Enterprise + AI | personalizado | + audit, compliance, BYO key |

**Fortalezas:** integración Google Workspace nativa, español académico excelente, audio Deep Research.

**Debilidades:** Gemini standalone menos pulido que ChatGPT/Claude.

### Microsoft 365 Copilot

| Plan | Precio/mes |
|---|---|
| Copilot Pro (individual) | $20/usuario |
| Copilot for Microsoft 365 (E3+) | $30/usuario sobre M365 |
| Copilot Enterprise | personalizado |

**Fortalezas:** integración Office (Word, Excel, Outlook, Teams) inigualable, compliance enterprise.

**Debilidades:** modelo subyacente (GPT-4 personalizado) menos potente que GPT-5 directo, lock-in M365.

## Combinación adecuada para institución educativa mexicana

Caso típico: 50 staff, presupuesto $30 000-50 000 USD/año.

**Opción A** — Stack OpenAI mainstream:
- 50 × ChatGPT Team $30 = $1 500/mes = $18 000/año.
- Tools: GPT-5, Custom GPTs, Canvas, Code Interpreter.

**Opción B** — Stack mixto premium:
- 30 × Gemini Workspace + AI $20 = $600/mes (para staff que vive en Drive).
- 5 × Claude Pro $20 = $100/mes (para producción de contenido especializado).
- Total: $700/mes = $8 400/año.

**Opción C** — Stack Microsoft completo:
- 50 × M365 + Copilot $30 sobre M365 = $1 500/mes = $18 000/año.
- + ChatGPT Team Adicional 5 staff técnicos = $150/mes.

**Opción D** — Stack soberano híbrido:
- 5 × Claude Pro = $100/mes (staff técnico).
- Servidor local Ollama (ya pagado, $50/mes operación) para masivo.
- Open WebUI multi-user gratis para los 50 staff.
- Total: $150/mes = $1 800/año + costo de hardware amortizado.

La Opción D suele ser ganadora si tu institución tiene capacidad TI. Sin esa capacidad, B es razonable.

## Criterios de decisión

1. **¿Dónde vive ya tu staff?** (M365 → Copilot; Google → Gemini; sin preferencia → libre).
2. **¿Privacidad estricta?** (Enterprise plans + Local con Ollama).
3. **¿Volumen alto?** (Local soberano gana en TCO arriba de cierto volumen).
4. **¿Capacidad TI?** (sin → cloud all-in; con → local soberano fuerte).
5. **¿Compliance específica?** (HIPAA → cloud Enterprise certificado).

## Anti-patrones

- **Pagar 3 plataformas para todos.** $4 500/mes innecesarios.
- **No segmentar por rol.** Coordinadora ≠ docente ≠ técnico. Distintos planes según uso.
- **Ignorar el plan free para casos masivos.** Open WebUI + Ollama = $0/usuario.
- **Lock-in temprano.** Plan Enterprise 3 años puede ser jaula. Negocia 1 año primero.
- **Sin auditoría de uso.** Pagas planes que nadie usa.
