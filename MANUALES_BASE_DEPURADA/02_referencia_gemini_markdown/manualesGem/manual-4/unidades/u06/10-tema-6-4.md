# 6.4 Plataformas: ChatGPT Agents, Claude Computer Use, Manus, Devin, Lindy

El espacio de plataformas de agentes explotó en 2024-2025. Esta lección te da el mapa de las 6 más relevantes y cuándo elegir cada una para tu Asistente Institucional.

::concepto{titulo="Plataformas de agentes (2025)"}
A diferencia de los workflows (donde Make/Zapier/n8n cubren ya todo), las **plataformas de agentes** están en plena evolución. Cada una hace una apuesta diferente:

- **ChatGPT Agents (OpenAI)** — agentes integrados al producto ChatGPT, enfocados en conocimiento general + acciones via Zapier.
- **Claude Computer Use** — Claude controla un sistema operativo (mouse, teclado, navegador) como un humano.
- **Manus** — agente generalista de propósito amplio (China-origin, popular en 2025).
- **Devin (Cognition)** — agente desarrollador de software autónomo.
- **Lindy** — plataforma para construir agentes de negocio sin código.
- **AutoGen / CrewAI / LangGraph** — frameworks open source para construir agentes propios.

No hay "el mejor". Hay el adecuado a tu caso, riesgo y presupuesto.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Elegir una plataforma de agentes es como elegir vehículo: una motoneta para entregas urbanas, una pickup para construcción, un sedan ejecutivo para reuniones, un autobús para el equipo. Mismo "transporte", problemas distintos. Saber el catálogo te ahorra elegir el equivocado.
::/interioriza

## ChatGPT Agents (OpenAI)

**Acceso:** ChatGPT Plus/Team/Enterprise, opción "Tasks" o "Operator" según versión.

**Cómo funciona:**
- Tasks de ChatGPT: agendar acciones recurrentes ("revísame el calendario cada mañana y resume").
- Operator: agente que controla un navegador en la nube, hace tareas web por ti (reservar, buscar, comprar).

**Fortalezas:**
- Integración nativa con ecosistema ChatGPT.
- Tools listos: web browsing, code interpreter, image generation, image vision.
- Operator es genuino agente de navegación.

**Limitaciones:**
- Menos personalizable que frameworks open source.
- Operator solo en navegador cloud (no en tu PC).
- Costos según plan; Operator solo en Pro.

**Caso ideal del Asistente:** investigación web rutinaria, tareas mainstream con apps populares.

## Claude Computer Use (Anthropic)

**Acceso:** API Anthropic, modelos Claude 4.7 Opus / Sonnet con feature `computer-use`.

**Cómo funciona:**
- El modelo recibe **screenshots** de tu pantalla.
- Decide acciones: mover mouse a (x, y), click, type, scroll, capturar otra screenshot.
- Itera hasta cumplir el objetivo.

**Fortalezas:**
- Funciona en **cualquier app del sistema** (no solo navegador).
- Muy potente para automatizar UIs no API-friendly.
- Open: corres tú el loop con la API.

**Limitaciones:**
- Latencia (cada screenshot + decisión).
- Costoso (modelo grande llamado muchas veces).
- Riesgos de seguridad si no se ejecuta en sandbox.

**Caso ideal del Asistente:** automatizar interfaces antiguas (sistema de calificaciones legacy sin API), workflow donde solo hay UI.

## Manus

**Acceso:** manus.im (suscripción), versión cloud.

**Cómo funciona:**
- Agente generalista que descompone tareas complejas en sub-pasos automáticamente.
- Acceso a navegador, código, archivos, gen imágenes.
- Plan-and-execute con verificación.

**Fortalezas:**
- Muy autónomo; puede correr 30 minutos sin intervención.
- Ecosistema integrado (no necesitas configurar tools).
- Bueno en research y producción de outputs largos.

**Limitaciones:**
- Origen China, depending on tu policy de proveedores.
- Privacidad: tus tareas pasan por su infra.
- No customizable como AutoGen.

**Caso ideal del Asistente:** investigación de mercado profunda, generación de reportes largos one-shot.

## Devin (Cognition Labs)

**Acceso:** suscripción enterprise.

**Cómo funciona:**
- Agente especializado en desarrollo de software.
- Recibe issue de GitHub o brief, escribe código, lo prueba, hace PR.

**Fortalezas:**
- En su nicho (coding) es de los mejores.
- Integración nativa con GitHub, Slack, Jira.

**Limitaciones:**
- Solo desarrollo de software (no aplica al Asistente Institucional típico).
- Costoso.

**Caso ideal del Asistente:** si tu institución tiene pequeño equipo de TI y quiere acelerar PRs internos, no para el Asistente en sí.

## Lindy

**Acceso:** lindy.ai (planes desde $50/mes).

**Cómo funciona:**
- Plataforma no-code para construir "Lindys" (agentes de negocio personalizados).
- UI guiada: defines objetivo, conectas apps, configuras instrucciones.
- Tipos: agente para email, calendar, ventas, soporte, contratación, RH.

**Fortalezas:**
- No-code, accesible para no-developers.
- Plantillas pre-configuradas.
- Soporte 200+ integraciones (similar a Zapier).

**Limitaciones:**
- Cloud only.
- Personalización moderada (vs frameworks open).

**Caso ideal del Asistente:** primer agente del Asistente sin construirlo desde cero. Buen punto de entrada.

## Frameworks open source (AutoGen, CrewAI, LangGraph)

**Acceso:** instalación local (Python/Node), gratis.

**Cómo funcionan:**
- Bibliotecas para construir agentes con código.
- Soporte para multi-agente (AutoGen, CrewAI), grafos de estados (LangGraph).

**Fortalezas:**
- Control total.
- Self-hosted, privacidad máxima.
- Gratis.

**Limitaciones:**
- Requiere developer.
- Mantenimiento tuyo.

**Caso ideal del Asistente:** cuando creciste, quieres soberanía y tienes un developer.

## Recomendación de stack para tu Asistente Institucional

**Fase inicial (mes 1-3):** Lindy o n8n AI Agent — no-code, rápido, suficiente.

**Fase media (mes 4-9):** AutoGen o LangGraph self-hosted — cuando quieras control y privacidad. Agente reactivo o deliberativo.

**Fase avanzada (mes 12+):** combinación — Lindy para flujos rutinarios, AutoGen para casos especializados, Computer Use solo si una UI legacy lo justifica.

## Anti-patrones

- **Empezar con AutoGen sin developer.** Sobre-tu-cabeza.
- **Computer Use sin sandbox.** El modelo puede borrar archivos por error. **Siempre VM aislada.**
- **Agente generalista para tareas críticas.** Para acciones con consecuencia, agente especializado con guardrails.
- **No comparar costos.** ChatGPT Operator $200/mes, AutoGen self-hosted $0 (pero developer time). Calcula.
- **Lock-in temprano.** Diseña tools y prompts portables; la plataforma puede cambiar.
