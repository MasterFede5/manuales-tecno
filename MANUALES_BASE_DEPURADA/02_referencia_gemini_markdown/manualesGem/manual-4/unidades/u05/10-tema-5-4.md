# 5.4 Power Automate con Copilot

Si tu institución vive en Microsoft 365 (Outlook, SharePoint, Teams, Excel Online), Power Automate ya está esperándote sin costo extra. Y desde 2024, Copilot te lo escribe en lenguaje natural.

::concepto{titulo="Power Automate y Copilot"}
**Power Automate** (antes Microsoft Flow) es la plataforma de automatización de Microsoft, integrada a fondo con Microsoft 365, Dynamics, Azure y aplicaciones de terceros. Más de 1 000 conectores nativos.

**Copilot in Power Automate** es la capa de IA conversacional: describes en lenguaje natural lo que quieres automatizar y Copilot **construye el flujo** completo, lo nombra, lo configura y te deja ajustarlo. Reduce setup de 30 min a 3 min para flujos típicos.

Su mayor ventaja: **integración profunda** con todo Microsoft 365. Su mayor limitación: **fuera del ecosistema Microsoft**, otras plataformas son superiores.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Si tu institución usa SharePoint para documentos, Outlook para correo, Teams para mensajería y Excel Online para datos, Power Automate accede a todo eso **sin permisos extra y sin costo adicional** (incluido en muchos planes M365). Para una institución 100 % Microsoft, es la opción nativa.
::/interioriza

## La fortaleza única: integración nativa M365

| App | Acciones típicas |
|---|---|
| **Outlook** | Trigger en correo recibido; enviar; mover a carpeta; flag |
| **SharePoint** | Crear/actualizar items en lista; subir docs; check-out/check-in |
| **Teams** | Postear en canal; crear meeting; mensaje directo |
| **Excel Online** | Agregar fila; leer; actualizar celda |
| **OneDrive** | Subir archivo; mover; convertir a PDF |
| **Forms** | Trigger en respuesta nueva |
| **Planner** | Crear tarea; asignar; completar |
| **Power BI** | Refresh dataset; export reporte |

Y todo con autenticación corporativa: Azure AD, MFA, condicionales por grupo.

## Workflow típico con Copilot

Tú escribes:
> *"Cuando llegue un correo a admisiones@miescuela.edu con asunto que contenga 'inscripción', extrae nombre y email, agrega fila a SharePoint List 'Solicitudes 2026', y envía respuesta automática con folio."*

Copilot construye:
1. Trigger: "When a new email arrives to admisiones@..."
2. Filter: "Subject contains 'inscripción'"
3. AI Builder: extract entities (nombre, email).
4. SharePoint: "Add item to list Solicitudes 2026".
5. Send reply: plantilla con folio generado.

Tiempo total: 5 minutos incluyendo prueba.

## Cuándo Power Automate es la mejor opción

- **Institución 100 % Microsoft 365.** Ya está pagado.
- **Datos sensibles dentro de M365.** No tienen que salir a SaaS externo.
- **Compliance estricto.** Microsoft cumple ISO 27001, SOC 2, FedRAMP, GDPR.
- **Integración con Dynamics 365 / Power Apps / Power BI** (todo el Power Platform).
- **Flujos largos** con muchas approvals (Teams approvals nativo es excelente).

## Cuándo NO

- **Stack Google Workspace.** No tienen integraciones nativas; Workspace + Apps Script o Zapier.
- **Apps de terceros mainstream** (Notion, Airtable, Slack, ClickUp). Tienen conectores pero menos pulidos.
- **AI moderna avanzada.** Copilot es bueno; los nodos de IA son menos sofisticados que n8n con LangChain.
- **Self-hosting.** No existe; solo cloud Microsoft.

## AI Builder vs nodos AI de n8n

Power Automate tiene "AI Builder" para tareas comunes:
- Extraer información de documentos (formularios, recibos).
- Clasificar texto en categorías predefinidas.
- Detección de entidades (nombre, email, fecha).
- Predicción simple (en preview).

Para tareas más sofisticadas (RAG, agentes, generación creativa), conectas a Azure OpenAI Service o llamas a APIs externas con HTTP request.

## Pricing en M365 (2025)

- **Plan Microsoft 365 Business Standard** ($12.50/usuario/mes) — incluye Power Automate básico.
- **Power Automate Premium** ($15/usuario/mes) — agrega RPA, AI Builder, conectores premium.
- **Power Automate Process** (RPA per process) — desde $150/mes.

Para una institución de 50 staff con M365 Business: Power Automate básico está incluido. Si necesitas AI Builder intensivo, sumar Premium.

## Patrón Asistente + Power Automate

```
Asistente Institucional (Custom GPT o Claude Project)
     ↓ (vía Zapier AI Actions o webhook directo)
Power Automate workflow
     ↓ (acciones M365)
SharePoint, Outlook, Teams, Excel
```

El Asistente "conversa"; Power Automate "ejecuta dentro de Microsoft". Encaje natural en instituciones M365.

## Anti-patrones

- **Construir flujos sin Copilot.** Para flujos típicos, Copilot ahorra 80 % del tiempo.
- **AI Builder para todo.** Para tareas complejas, llama a Azure OpenAI o APIs externas.
- **Sin testeo manual.** Power Automate tiene "Test" robusto. Úsalo antes de activar.
- **Mezclar contextos M365.** Si el flujo toca datos personales (RR. HH.), respeta políticas de retención y permissions.
