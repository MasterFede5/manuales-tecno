# 1.7 Nomenclatura inorgánica básica

¿Cómo se llama un compuesto que tiene calcio y oxígeno? ¿Es "óxido de calcio", "monóxido de calcio", "óxido cálcico", o todas las anteriores? La nomenclatura química es el **lenguaje internacional** que evita ambigüedad. Aprenderla es como aprender alfabeto: parece tedioso al inicio, pero después puedes leer cualquier etiqueta de reactivo del mundo.

::concepto{titulo="Nomenclatura química inorgánica"}
La **nomenclatura química** es el conjunto de reglas para nombrar compuestos a partir de su fórmula y, recíprocamente, deducir la fórmula a partir del nombre. Existen tres sistemas oficiales reconocidos por la IUPAC: la **nomenclatura sistemática** (con prefijos griegos), la **Stock** (con números romanos) y la **tradicional** (con sufijos -oso/-ico). Los tres son válidos; el contexto y el nivel de detalle deciden cuál conviene usar.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Si tu mamá te pide que vayas a la farmacia por "el medicamento blanco" y la receta del doctor dice "ácido acetilsalicílico", la diferencia entre ambas frases puede ser entre llevar aspirina o paracetamol. La nomenclatura química resuelve eso para todas las sustancias del planeta. Cuando entiendas las reglas, leerás una etiqueta de reactivo y, sin tener que buscar en internet, sabrás qué átomos contiene, en qué proporción, y para qué se usa.
::/interioriza

## Estados de oxidación — la base del nombre

El **estado de oxidación (EO)** es la carga aparente que tendría un átomo dentro de un compuesto si todos los enlaces fueran iónicos. Reglas rápidas:

| Regla | Ejemplo |
|---|---|
| Elementos en estado libre: EO = 0 | Fe(s), O₂, H₂ |
| Iones simples: EO = carga del ion | Na⁺ (+1), Cl⁻ (−1), Ca²⁺ (+2) |
| H en compuestos: +1 (excepto hidruros metálicos: −1) | HCl: H = +1; NaH: H = −1 |
| O en compuestos: −2 (excepto peróxidos: −1; con F: +2) | H₂O: O = −2; H₂O₂: O = −1 |
| Suma de EO en un compuesto neutro: 0 | NaCl: +1 + (−1) = 0 |
| Suma de EO en un ion poliatómico: carga del ion | SO₄²⁻: S(+6) + 4·O(−2) = −2 |

Conociendo el EO de los átomos comunes, puedes deducir el de cualquier otro elemento por suma algebraica.

## Compuestos binarios — dos elementos

### Óxidos básicos (metal + O)

Fórmula: **Metal + O**. Nombre: **óxido de [metal]** (Stock con número romano si el metal tiene varios EO).

| Fórmula | Nombre Stock | Nombre tradicional |
|---|---|---|
| Na₂O | óxido de sodio | óxido sódico |
| CaO | óxido de calcio | óxido cálcico |
| FeO | óxido de hierro(II) | óxido ferroso |
| Fe₂O₃ | óxido de hierro(III) | óxido férrico |
| CuO | óxido de cobre(II) | óxido cúprico |

### Óxidos ácidos / anhídridos (no metal + O)

Fórmula: **No metal + O**. Nombre sistemático con prefijos di-, tri-, tetra-:

| Fórmula | Nombre sistemático | Nombre tradicional |
|---|---|---|
| CO | monóxido de carbono | óxido carbonoso |
| CO₂ | dióxido de carbono | anhídrido carbónico |
| SO₂ | dióxido de azufre | anhídrido sulfuroso |
| SO₃ | trióxido de azufre | anhídrido sulfúrico |
| N₂O₅ | pentóxido de dinitrógeno | anhídrido nítrico |

### Hidruros

- **Metálicos:** hidruro de [metal]. Ej. NaH = hidruro de sodio.
- **No metálicos:** se nombran como ácidos cuando se disuelven en agua (sección de hidrácidos).

### Sales binarias (metal + no metal)

Fórmula y nombre: el no metal con sufijo **-uro** + "de" + metal.

| Fórmula | Nombre |
|---|---|
| NaCl | cloruro de sodio |
| KBr | bromuro de potasio |
| CaF₂ | fluoruro de calcio |
| FeS | sulfuro de hierro(II) |
| Al₂S₃ | sulfuro de aluminio |

### Ácidos hidrácidos

Cuando un hidruro de no metal del grupo 16 o 17 se disuelve en agua, se comporta como ácido. Se nombran con el prefijo **ácido** + raíz + sufijo **-hídrico**:

| Fórmula | Nombre del compuesto puro | Nombre como ácido |
|---|---|---|
| HCl(g) | cloruro de hidrógeno | ácido clorhídrico (HCl ac) |
| HBr | bromuro de hidrógeno | ácido bromhídrico |
| H₂S | sulfuro de hidrógeno | ácido sulfhídrico |
| HF | fluoruro de hidrógeno | ácido fluorhídrico |

## Compuestos ternarios — tres elementos

### Hidróxidos / bases (metal + OH)

Fórmula: **Metal + (OH)ₙ**. Nombre: **hidróxido de [metal]**.

| Fórmula | Nombre |
|---|---|
| NaOH | hidróxido de sodio |
| Ca(OH)₂ | hidróxido de calcio |
| Al(OH)₃ | hidróxido de aluminio |
| Fe(OH)₃ | hidróxido de hierro(III) |

### Oxoácidos (H + no metal + O)

Cuando un anhídrido se hidrata, forma un oxoácido. Reglas tradicionales por sufijo:

- EO bajo del no metal: prefijo **hipo-** + sufijo **-oso** (ej. ácido hipocloroso HClO).
- EO medio-bajo: sufijo **-oso** (ej. ácido cloroso HClO₂).
- EO medio-alto: sufijo **-ico** (ej. ácido clórico HClO₃).
- EO máximo: prefijo **per-** + sufijo **-ico** (ej. ácido perclórico HClO₄).

| Fórmula | Nombre tradicional | EO del no metal |
|---|---|:---:|
| HClO | ácido hipocloroso | +1 |
| HClO₂ | ácido cloroso | +3 |
| HClO₃ | ácido clórico | +5 |
| HClO₄ | ácido perclórico | +7 |
| H₂SO₃ | ácido sulfuroso | +4 |
| H₂SO₄ | ácido sulfúrico | +6 |
| H₂CO₃ | ácido carbónico | +4 |
| HNO₂ | ácido nitroso | +3 |
| HNO₃ | ácido nítrico | +5 |
| H₃PO₄ | ácido fosfórico | +5 |

### Oxisales (metal + no metal + O)

Cuando un oxoácido se neutraliza con un hidróxido, los H del ácido se reemplazan por el catión metálico. El nombre cambia el sufijo del ácido así:

| Sufijo del ácido | Sufijo de la sal |
|---|---|
| hipo-…-oso | hipo-…-ito |
| -oso | -ito |
| -ico | -ato |
| per-…-ico | per-…-ato |

| Fórmula | Nombre |
|---|---|
| NaClO | hipoclorito de sodio (cloro casero) |
| NaClO₃ | clorato de sodio |
| Na₂SO₃ | sulfito de sodio |
| Na₂SO₄ | sulfato de sodio |
| CaCO₃ | carbonato de calcio (mármol, sarro) |
| KNO₃ | nitrato de potasio (salitre) |
| Ca₃(PO₄)₂ | fosfato de calcio (huesos) |

## Aplicación al bebedero

Con estas reglas, ya puedes traducir la etiqueta del laboratorio:

- "Cloro residual" como **HOCl** = ácido hipocloroso (oxoácido), o como **NaClO** = hipoclorito de sodio (oxisal, "cloro" comercial).
- "Dureza" causada por **CaCO₃** = carbonato de calcio (oxisal) y **Ca(HCO₃)₂** = hidrogencarbonato (bicarbonato) de calcio.
- "Sodio" como ion **Na⁺** que viene del NaCl (cloruro de sodio).
