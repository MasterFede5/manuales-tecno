::act-mcq{titulo="Estequiometría, energía y equilibrio"}
1. La reacción $\ce{N2 + 3 H2 -> 2 NH3}$ requiere para 5 mol de N₂…
   - [ ] 5 mol de H₂
   - [x] 15 mol de H₂
   - [ ] 1 mol de H₂
   - [ ] 10 mol de H₂

2. Una reacción es espontánea cuando…
   - [ ] ΔH < 0 siempre
   - [ ] ΔS > 0 siempre
   - [x] ΔG < 0
   - [ ] Kc = 1

3. Un catalizador modifica…
   - [ ] el ΔH de la reacción
   - [ ] la espontaneidad
   - [x] la energía de activación
   - [ ] la cantidad de producto teórica

4. Si en $\ce{N2 + 3 H2 <=> 2 NH3}$ exotérmica aumentas la temperatura…
   - [ ] el equilibrio se desplaza a productos
   - [x] el equilibrio se desplaza a reactivos
   - [ ] no cambia
   - [ ] aumenta solo la velocidad sin desplazar

5. La economía atómica del 80 % significa que…
   - [ ] 80 % del reactivo se usa
   - [x] 80 % de los átomos terminan en el producto deseado
   - [ ] el rendimiento es 80 %
   - [ ] la pureza es 80 %
::/act-mcq

::act-match{titulo="Conceptos termodinámicos"}
| Concepto | Definición |
|---|---|
| 1. Entalpía (ΔH) | a) Constante a una temperatura dada; predice rendimiento |
| 2. Entropía (ΔS) | b) Calor intercambiado a presión constante |
| 3. Energía libre de Gibbs (ΔG) | c) Medida del desorden del sistema |
| 4. Constante de equilibrio (Kc) | d) Predice espontaneidad |
| 5. Energía de activación (Ea) | e) Barrera mínima para que ocurra la reacción |
::/act-match

::albatros{titulo="Diseña una reacción 'verde' para producir oxígeno en casa" tipo="experimento" tiempo="60 min"}
**Pregunta detonadora.** ¿Cómo producir O₂ en casa usando reactivos seguros y minimizando residuos? El método clásico (descomposición térmica de KClO₃) es peligroso. Diseña una alternativa verde.

**Lo que harás.**
1. Investiga 3 métodos de producción de O₂ a pequeña escala (bachillerato).
2. Para cada uno, evalúa: economía atómica, E-factor estimado, peligrosidad, costo.
3. Elige el más verde y justifica.
4. Si tienes acceso a materiales (peróxido de hidrógeno + dióxido de manganeso es buena alternativa), realiza el experimento documentando la reacción $\ce{2 H2O2 -> 2 H2O + O2}$ con MnO₂ como catalizador.
5. Mide cuántitativamente el O₂ producido (volumen) y calcula rendimiento.

**Materiales.** H₂O₂ farmacia (3 %) · MnO₂ del interior de pila gastada (con guantes) · matraz · vela para verificar combustión avivada · cuaderno.

**Entregable.** Reporte de 1 página con tabla comparativa de los 3 métodos, justificación de la elección y resultados experimentales con foto.

**Rúbrica corta.**
| Criterio | 1 | 3 | 5 |
|---|---|---|---|
| Comparación | un método solo | tres métodos sin métricas | tres métodos con métricas verde cuantificadas |
| Experimento | no se hace | se hace pero sin datos | datos numéricos + foto + cálculo de rendimiento |
| Justificación verde | "es seguro" | aplica 2 principios | aplica 4+ principios con argumentos |
::/albatros

::act-order{titulo="Pasos para aplicar Hess y calcular ΔH global"}
[ ] Selecciona reacciones intermedias con ΔH conocido.
[ ] Multiplica cada reacción por el coeficiente que cuadre la global.
[ ] Invierte las que sea necesario y cambia el signo del ΔH.

::albatros{titulo="Reto Albatros — calcula el costo del tratamiento de tu casa" tipo="reto" tiempo="50 min"}
**Pregunta detonadora.** ¿Cuánto cuesta —en pesos, kWh y kg CO₂— purificar el agua de tu casa por un mes? ¿Es más barato que comprar garrafones?

**Lo que harás.**
1. Estima el consumo mensual de agua potable de tu familia (litros/día × 30).
2. Diseña 3 esquemas: a) cloración (como el dossier), b) filtro de carbón activado + UV, c) compra de garrafones.
3. Para cada uno calcula: costo inicial, costo operativo mensual, energía mensual (kWh), CO₂eq mensual.
4. Compara con compra de garrafones (≈ \$45 por garrafón de 20 L).
5. Identifica el "punto de equilibrio": ¿en cuántos meses se paga el sistema casero?

**Materiales.** Calculadora · recibos de luz · precios locales (Mercado Libre, Home Depot).

**Entregable.** Hoja A4 con tabla comparativa, punto de equilibrio y recomendación argumentada.

**Rúbrica corta.**
| Criterio | 1 | 3 | 5 |
|---|---|---|---|
| Datos | estimaciones a ojo | estimaciones razonables | datos verificables con recibo o link |
| Cálculo CO₂ | sin huella | huella mensual | huella mensual + anual + comparación con promedio MX |
| Decisión | "es muy caro" | recomendación | recomendación con punto de equilibrio en meses |
::/albatros
