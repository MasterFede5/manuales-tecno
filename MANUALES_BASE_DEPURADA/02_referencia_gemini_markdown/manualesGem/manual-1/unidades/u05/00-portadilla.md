# Unidad 05

## Energía y Reacciones Químicas

> Toda reacción cuesta o libera energía. Aprender estequiometría es aprender a contar átomos; aprender termodinámica es aprender a contar energía. Juntas, estas dos contabilidades te permiten diseñar cualquier proceso químico — incluyendo el tratamiento del agua escolar.
