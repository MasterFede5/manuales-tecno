### Subtema 3.3 — REDOX y balanceo

::act-table{titulo="Balanceo de ecuaciones — completa"}
| Reacción sin balancear | Balanceada | Tipo |
|---|---|---|
| Mg + O₂ → MgO |  | combustión / redox |
| C₃H₈ + O₂ → CO₂ + H₂O |  |  |
| Fe + HCl → FeCl₂ + H₂ |  |  |
| KClO₃ → KCl + O₂ |  |  |
| Cu + AgNO₃ → Cu(NO₃)₂ + Ag |  |  |
::/act-table

::act-order{titulo="Pasos para balancear una redox por ion-electrón"}
[ ] Identifica los números de oxidación de cada átomo.
[ ] Separa la reacción en dos semirreacciones.
[ ] Balancea átomos distintos a O y H.
[ ] Balancea O con H₂O y H con H⁺ (medio ácido).
[ ] Iguala electrones intercambiados.
[ ] Suma las semirreacciones y simplifica.
::/act-order

### Subtema 3.6 — Lluvia ácida y atmósfera global

## Clave de respuestas

**Subtema 3.1 — Huecos.** 1) nitrógeno (N₂) · 2) oxígeno (O₂) · 3) 1 % · 4) 420 ppm (≈ 420) · 5) metano (CH₄) o vapor de agua (acepta ambas) · 6) 1.2.

**Subtema 3.1 — MCQ.** 1-b · 2-c · 3-b.

**Subtema 3.3 — Tabla.** 1) 2 Mg + O₂ → 2 MgO (redox) · 2) C₃H₈ + 5 O₂ → 3 CO₂ + 4 H₂O (combustión) · 3) Fe + 2 HCl → FeCl₂ + H₂ (sustitución redox) · 4) 2 KClO₃ → 2 KCl + 3 O₂ (descomposición) · 5) Cu + 2 AgNO₃ → Cu(NO₃)₂ + 2 Ag (sustitución).

**Subtema 3.3 — Orden.** Asigna números de oxidación → separa semirreacciones → balancea átomos distintos a O/H → balancea O con H₂O y H con H⁺ → iguala electrones → suma y simplifica.

**Subtema 3.3 — Cálculos.** a) $n(C_8H_{18}) = 1000/114.23 = 8.76$ mol; $n(CO_2) = 8 \times 8.76 = 70.1$ mol; m = 3083 g ≈ **3.08 kg**. b) $V(O_2) = (25/2) \times 8.76 \times 24 \approx 2628$ L. c) $n(CO_2) = 50000/44.01 = 1136$ mol; $n(C_8H_{18}) = 1136/8 = 142$ mol; m ≈ **16.2 kg**. d) 50 kg/h × 1000/(44.01 × 3600) ≈ **0.32 mol/s**.

**Subtema 3.5 — Match.** 1-c · 2-a · 3-d · 4-b · 5-e · 6-f.

**Subtema 3.5 — V/F.** 1-F (es muy estable por el triple enlace) · 2-V · 3-F (5.6 es el límite natural) · 4-V · 5-V.

**Subtema 3.6 — Cálculos.** a) pH = 4.30, lluvia ácida moderada. b) 30 % de 90/46.01 × 10⁻⁶ = 5.87 × 10⁻⁷ mol; en 1 mL = 5.87 × 10⁻⁴ M; pH ≈ 3.23. c) [H⁺] = 10⁻³ M ⇒ 5 × 10⁻⁴ mol Zn ⇒ 32.7 mg Zn por litro. d) Sí supera el límite NOM-127 (5 mg/L); riesgo de exposición.
