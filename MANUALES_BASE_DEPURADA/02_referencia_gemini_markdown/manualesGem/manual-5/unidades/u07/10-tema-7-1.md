# 7.1 Neurona artificial e intuición

Una neurona artificial es **un poco más que una multiplicación-suma con un switch al final**. Apilando muchas, capturas patrones complejos.

::concepto{titulo="Neurona artificial"}
Una **neurona artificial** (perceptrón) toma entradas, las multiplica por **pesos**, las suma con un **bias**, y aplica una **función de activación** para producir salida.

```
salida = activación(w₁·x₁ + w₂·x₂ + ... + wₙ·xₙ + b)
```

- **xᵢ**: entradas (features).
- **wᵢ**: pesos (lo que la red aprende).
- **b**: bias (sesgo, offset).
- **activación**: función no lineal que decide la salida (ReLU, sigmoid, tanh).

El "aprendizaje" consiste en ajustar w y b para que la salida coincida con lo deseado.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
La neurona artificial inspira en biología pero es **mucho más simple**: solo aritmética. Lo poderoso es la **composición**: capas de neuronas pueden representar funciones arbitrariamente complejas (teorema de aproximación universal).
::/interioriza

## Ejemplo concreto

Una neurona predictora de "aprueba" según horas, asistencia, cal_anterior:

```
Entradas: x₁=horas=5, x₂=asis=0.9, x₃=cal_ant=8
Pesos:    w₁=0.1, w₂=2.0, w₃=0.5
Bias:     b=-3

Suma ponderada: 0.1·5 + 2.0·0.9 + 0.5·8 + (-3) = 0.5 + 1.8 + 4.0 - 3 = 3.3

Activación sigmoid: σ(3.3) = 0.965

Output: probabilidad de aprobar = 0.965 → predicción "aprueba".
```

Los pesos no fueron mágicos: la red los **aprendió** durante entrenamiento.

## Función de activación

Sin activación, una capa es solo combinación lineal. La activación introduce **no-linealidad**, permitiendo aprender patrones complejos:

| Activación | Salida | Cuándo usar |
|---|---|---|
| **ReLU** | max(0, x) | capas ocultas (default moderno) |
| **Sigmoid** | 1/(1+e⁻ˣ) entre 0-1 | salida clasificación binaria |
| **Tanh** | entre -1 y 1 | obsoleto pero útil |
| **Softmax** | probabilidades sumando 1 | salida multi-clase |

## De una neurona a una red

Apilando neuronas en **capas**:

- **Capa de entrada**: una neurona por feature.
- **Capas ocultas**: cualquier número, cualquier neuronas.
- **Capa de salida**: una para regresión, K para clasificación K clases.

Total número de pesos para MLP de 4-16-1: 4×16 + 16 + 16×1 + 1 = 97 pesos. La red ajusta estos 97 números durante entrenamiento.

## Anti-patrones

- **Pensar que la neurona "piensa"** como cerebro biológico. Es aritmética simple.
- **Una sola neurona para problema complejo**. Limitada a línea recta. Necesitas capas.
- **Sin activación no-lineal**. Solo regresión lineal disfrazada.
