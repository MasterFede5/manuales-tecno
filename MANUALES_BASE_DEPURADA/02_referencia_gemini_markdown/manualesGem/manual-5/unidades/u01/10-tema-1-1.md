# 1.1 Configuración: Anaconda, VS Code, Colab

Antes de escribir tu primera línea de Python, necesitas decidir **dónde** la vas a escribir. Tres opciones razonables. Esta lección te ayuda a elegir y a tener el entorno listo en 30 minutos.

::concepto{titulo="Entorno de desarrollo Python"}
Un **entorno de desarrollo** es el conjunto de herramientas (intérprete Python + editor + librerías) donde escribes y ejecutas tu código. Para principiantes en IA hay **3 caminos** principales:

- **Anaconda + Jupyter Notebook** — instalación todo-en-uno con cientos de librerías de ciencia de datos pre-instaladas. Ideal para principiantes en data science.
- **VS Code + Python** — editor profesional con extensión Python. Ideal si quieres flexibilidad y vas a programar más allá de IA.
- **Google Colab** — Jupyter Notebook en la nube, gratis, sin instalación. Ideal si tu computadora es modesta o si quieres GPU gratis para redes neuronales.

Para este manual: **cualquiera de los tres funciona**. Te recomendamos Colab para empezar, Anaconda para tu propia computadora, VS Code cuando ya seas avanzado.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Igual que un cocinero elige sus utensilios antes de cocinar, tú eliges tu entorno antes de programar. Mal elegido: pasarás horas peleando con la herramienta. Bien elegido: te enfocas en aprender Python y datos, no en configurar.
::/interioriza

## Camino 1 — Google Colab (recomendado para empezar)

**Acceso:** colab.research.google.com (necesitas cuenta Google).

**Pasos:**
1. Ir al sitio.
2. Click en "Nuevo notebook".
3. Ya tienes Python listo con cientos de librerías y GPU gratis (limitada).
4. Subes archivos al notebook con icono de carpeta.

**Pros:**
- Cero instalación.
- GPU gratuita para redes neuronales (U7).
- Compartible como Google Doc.
- Funciona en cualquier laptop, incluso modestas.

**Contras:**
- Necesitas internet.
- Sesiones expiran después de 12h inactividad.
- No es ideal para proyectos grandes a largo plazo.

## Camino 2 — Anaconda en tu computadora

**Descarga:** anaconda.com/download.

**Pasos:**
1. Instalador para tu OS (Windows, Mac, Linux).
2. Sigue wizard (acepta defaults).
3. Abre "Anaconda Navigator".
4. Lanza "JupyterLab" o "Jupyter Notebook".
5. Crea notebook nuevo Python 3.

**Pros:**
- Independiente de internet.
- Control total.
- Performance buena en tu hardware.

**Contras:**
- Instalación 10-20 minutos.
- Ocupa ~5 GB de disco.
- Requiere ~8 GB RAM mínimo.

## Camino 3 — VS Code

**Descarga:** code.visualstudio.com (Visual Studio Code).

**Pasos:**
1. Instalar VS Code.
2. Instalar Python desde python.org (Python 3.11+).
3. Abrir VS Code, instalar extensión "Python" de Microsoft.
4. Instalar extensión "Jupyter".
5. Crear archivo `.ipynb` o `.py` y empezar.

**Pros:**
- Editor de nivel profesional.
- Extensible para múltiples lenguajes.
- Integración Git nativa.

**Contras:**
- Curva de aprendizaje más alta.
- Hay que configurar pip, virtual environments (no en U1).

## Verificación: tu primer "Hola, IA"

En cualquiera de los 3 caminos, abre un notebook (`.ipynb`) y escribe en una celda:

```python
print("¡Hola, mundo de IA!")
print("2 + 2 =", 2 + 2)
```

Presiona Shift+Enter. Si ves:
```
¡Hola, mundo de IA!
2 + 2 = 4
```

**Felicidades, tu entorno funciona.**

## Librerías que vas a usar (instalación si necesario)

En Colab, la mayoría ya están. En Anaconda también. En VS Code, instalas con:

```bash
pip install numpy pandas matplotlib seaborn scikit-learn
```

Las 5 cubren U2 a U6. Para U7 (PyTorch) y U8 (APIs IA) instalas más cuando llegues.

## Anti-patrones

- **Instalar todo desde cero con `pip install`.** En Anaconda ya viene; ahorra tiempo usando lo pre-instalado.
- **Mezclar Anaconda con Python "puro" instalado por separado.** Conflictos. Quédate con Anaconda o Python puro, no mezcles.
- **Empezar con VS Code sin experiencia.** Para U1, abruma. Empieza con Colab.
- **Saltarte verificación.** Asume que funciona y no funcionar más tarde es peor.
