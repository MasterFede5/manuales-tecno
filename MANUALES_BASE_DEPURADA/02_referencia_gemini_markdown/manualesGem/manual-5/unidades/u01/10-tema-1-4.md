# 1.4 Funciones y argumentos

Si necesitas clasificar 480 calificaciones, no escribes 480 veces el `if/elif/else`. Lo escribes UNA vez como **función** y la llamas 480 veces. Así se construye software.

::concepto{titulo="Función y argumentos"}
Una **función** es un bloque de código con nombre que recibe **argumentos** (entradas), ejecuta lógica y opcionalmente devuelve un **resultado** (salida con `return`).

```python
def nombre_funcion(parametro1, parametro2):
    # cuerpo indentado
    resultado = ...
    return resultado
```

**Por qué importan:**
- **Reusabilidad** — escribes una vez, usas N veces.
- **Claridad** — el nombre de la función documenta intención.
- **Testabilidad** — pruebas pequeñas piezas en aislamiento.
- **Composición** — combinas funciones para construir programas grandes.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Cuando construyes una receta de cocina, le pones nombre ("salsa boloñesa"), defines ingredientes ("carne, tomate, especies") y pasos. Cuando la necesitas, la haces — sin reinventar. Las funciones son lo mismo: nombre + entradas + pasos + resultado.
::/interioriza

## Anatomía

```python
def clasificar_calificacion(cal):
    if cal >= 9:
        return "Excelente"
    elif cal >= 8:
        return "Muy bien"
    elif cal >= 6:
        return "Aprobó"
    else:
        return "Reprobó"

# Uso
resultado = clasificar_calificacion(7.5)
print(resultado)  # "Aprobó"

# En bucle
calificaciones = [8.5, 6.0, 9.5, 5.5]
for cal in calificaciones:
    print(cal, "→", clasificar_calificacion(cal))
```

## Parámetros y argumentos

Los **parámetros** son las variables en la definición; los **argumentos** son los valores que pasas al llamar:

```python
def calcular_promedio(notas, pesos):
    suma = 0
    for i in range(len(notas)):
        suma += notas[i] * pesos[i]
    return suma / sum(pesos)

# Llamada
notas = [8.5, 7.0, 9.0]
pesos = [0.3, 0.4, 0.3]
prom = calcular_promedio(notas, pesos)
print(prom)
```

## Argumentos por defecto

```python
def calcular_indice(horas_estudio, asistencia=1.0):
    return horas_estudio * asistencia

# Si no pasas asistencia, usa 1.0
print(calcular_indice(5))         # 5.0
print(calcular_indice(5, 0.8))    # 4.0
```

## Argumentos por nombre (keyword arguments)

```python
def reportar(nombre, calificacion, materia="General"):
    print(f"{nombre} obtuvo {calificacion} en {materia}")

reportar("Andrea", 8.5, "Mate")
reportar(calificacion=9.0, nombre="Carlos", materia="Física")
```

## Funciones que devuelven varios valores

```python
def estadisticas(notas):
    promedio = sum(notas) / len(notas)
    minimo = min(notas)
    maximo = max(notas)
    return promedio, minimo, maximo

prom, mn, mx = estadisticas([8.5, 6.0, 9.5, 5.5])
print("Promedio:", prom)
print("Min:", mn)
print("Max:", mx)
```

Python devuelve tupla, lo "unpackeas" en variables.

## Funciones puras vs con efecto

- **Pura**: misma entrada → misma salida. No modifica nada externo. **Preferible**.
- **Con efecto**: modifica variables globales, archivos, BD. **Cuidado**.

```python
# Pura
def doblar(x):
    return x * 2

# Con efecto (evitar en general)
total = 0
def sumar(x):
    global total
    total += x
```

## Anti-patrones

- **Función gigante de 100 líneas.** Si no cabe en pantalla, parte en sub-funciones.
- **Sin `return`.** Sin return devuelve `None`. Asegúrate de retornar.
- **Argumentos posicionales confusos.** `calcular(a, b, c, d, e)` es opaco. Usa keyword arguments.
- **Nombres opacos.** `def f(x):` no dice nada. Usa `def calcular_promedio_ponderado(notas, pesos):`.
- **Modificar argumentos input.** Si recibes una lista y la modificas, sorprendes al usuario de la función.

## Caso aplicado

```python
def evaluar_alumno(notas, asistencia, horas_estudio):
    """Evalúa al alumno y devuelve diagnóstico."""
    prom = sum(notas) / len(notas)
    
    if asistencia < 0.7:
        return "En riesgo: baja asistencia"
    if prom < 6:
        return "Reprobado"
    if horas_estudio < 3:
        return "Aprueba pero podría mejorar con más estudio"
    if prom >= 9:
        return "Excelente"
    return "Aprueba"

# Uso
print(evaluar_alumno([8.0, 7.5, 9.0], 0.92, 5.5))
# Output: "Aprueba"
```
