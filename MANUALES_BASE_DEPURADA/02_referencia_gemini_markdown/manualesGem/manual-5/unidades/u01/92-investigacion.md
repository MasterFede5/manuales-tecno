::investiga{titulo="¿Cuál es la sintaxis de Python para 5 tareas comunes en otros lenguajes? Comparativo cross-language." tiempo="2 h"}
**Pregunta de investigación.** Si has visto JavaScript, Java, C# u otro lenguaje, esto te ayuda a mapear conocimiento. Investiga **5 tareas comunes** y compara cómo las hacen Python vs **otros 2 lenguajes** que conozcas o te interesen (JavaScript, Java, C#, R, Go, etc.).

**Lo que debes encontrar.**
- 5 tareas: declarar variable, condicional if/else, bucle for sobre lista, definir función, leer archivo de texto.
- Sintaxis de cada tarea en los 3 lenguajes.
- Cuál es más conciso, cuál más explícito, cuál tiene mejor manejo de errores.
- Qué le hace **único a Python** que sus competidores no tienen (o sí tienen).
- Aplicabilidad: ¿cuándo migrarías de Python a otro? ¿Cuándo no?

**Cómo presentarlo.** Documento de 2 páginas:
- Página 1: tabla comparativa de las 5 tareas en 3 lenguajes.
- Página 2: tu análisis y recomendación contextual.

**Lugares para arrancar.**
- learnxinyminutes.com (cheatsheet por lenguaje).
- StackOverflow para idiomatismos.
- Documentación oficial de cada lenguaje.

**Criterios de evaluación.**
- Precisión técnica de la sintaxis (40%).
- Profundidad del análisis comparativo (30%).
- Originalidad de las observaciones (15%).
- Claridad de la recomendación (15%).
::/investiga
