# 1.2 Variables, tipos, operadores

Una calificación es 8.5. Un nombre es "Andrea". Tu primer Python es **darle nombres a estas cosas** para que tu programa pueda usarlas. Esto se llama variables.

::concepto{titulo="Variable, tipo, operador"}
Una **variable** es un nombre que asocias a un valor. En Python:

```python
calificacion = 8.5
nombre = "Andrea"
aprobado = True
```

Un **tipo** es la categoría del valor: número entero (`int`), decimal (`float`), texto (`str`), verdadero/falso (`bool`). Python detecta el tipo automáticamente al asignar.

Un **operador** es un símbolo que opera sobre valores: aritméticos (`+`, `-`, `*`, `/`, `%`, `**`), de comparación (`==`, `!=`, `<`, `>`, `<=`, `>=`), lógicos (`and`, `or`, `not`).
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Igual que en una hoja de cálculo le pones nombre a una columna ("Calificaciones") para luego hacer fórmulas con ella, en programación le pones nombre a los valores. Sin variables, tendrías que escribir el número exacto cada vez. Con variables, escribes el nombre.
::/interioriza

## Los 4 tipos básicos

```python
# Entero (int) — sin decimales
edad = 17
horas_estudio = 6

# Decimal (float) — con decimales
calificacion = 8.5
porcentaje_asistencia = 92.3

# Texto (str) — entre comillas
nombre = "Andrea"
materia = 'Matemáticas'  # comillas simples también valen

# Verdadero/Falso (bool)
aprobado = True
faltó_examen = False
```

Para verificar tipo: `print(type(calificacion))`. Devuelve `<class 'float'>`.

## Operadores aritméticos

```python
# Suma, resta, multiplicación, división
total = 8.5 + 9.0  # 17.5
diferencia = 9.0 - 7.5  # 1.5
producto = 4 * 25  # 100
promedio = (8.5 + 9.0 + 7.5) / 3  # 8.333...

# División entera y módulo
cociente = 17 // 3  # 5 (sin decimales)
resto = 17 % 3  # 2 (sobrante)

# Potencia
cuadrado = 5 ** 2  # 25
```

## Operadores de comparación

Devuelven `True` o `False`:

```python
calificacion = 7.5

calificacion >= 6     # True (aprueba)
calificacion == 10    # False (no es 10)
calificacion != 0     # True (no es cero)
```

## Operadores lógicos

```python
asistencia = 0.85
calificacion = 7.0

aprueba = (asistencia >= 0.80) and (calificacion >= 6.0)  # True
en_riesgo = (asistencia < 0.70) or (calificacion < 6.0)   # False

reprobado = not aprueba  # False
```

## Cast (conversión entre tipos)

A veces necesitas convertir:

```python
edad_str = "17"
edad_num = int(edad_str)  # ahora es entero 17

cal_str = "8.5"
cal_num = float(cal_str)  # ahora es float 8.5

cal_num_str = str(cal_num)  # convierte de float a "8.5"
```

## Reglas de nombres de variables

- **Solo letras, números, guion bajo**: `cal_final`, `horas2024`, `_temp` (válidos).
- **No empezar con número**: `2024datos` (inválido).
- **Sin espacios**: `cal final` no funciona; usa `cal_final`.
- **Case-sensitive**: `Edad` y `edad` son distintas.
- **Reservadas no permitidas**: `if`, `for`, `True`, `class`, etc.
- **Convención (PEP 8)**: `snake_case` para variables (`mi_variable`).

## Anti-patrones

- **Nombres opacos**: `x = 8.5`. ¿Qué es x? Mejor `calificacion = 8.5`.
- **Nombres en inglés sin razón**: `grade = 8.5` cuando trabajas con coordinación mexicana, mejor en español.
- **Sobrescribir tipos sin querer**: `edad = 17` y luego `edad = "diecisiete"`. Tu programa rompe.
- **Comparar `is` con `==` mezclados**: `==` para valores, `is` para identidad de objeto. Para principiante usa `==` siempre.

```python
# Datos de un alumno
id_alumno = 124
horas_estudio_promedio = 5.5
asistencia = 0.92
calificacion_anterior = 7.8

# Cálculo simple de "índice de esfuerzo"

print("ID:", id_alumno, "Índice:", indice_esfuerzo)
# Output: ID: 124 Índice: 5.06
```
