# 1.3 Estructuras de control: if, for, while

Tu programa necesita **decidir** ("si la calificación es >= 6, aprueba") y **repetir** ("revisa cada uno de los 480 estudiantes"). Sin estas estructuras, Python solo ejecutaría línea por línea sin lógica. Esta lección te las enseña.

::concepto{titulo="Estructuras de control"}
Las **estructuras de control** dirigen el flujo de ejecución del programa. Tres familias:

- **Condicionales (`if`/`elif`/`else`)** — ejecutan código según condición.
- **Bucles for** — repiten código por cada elemento de una secuencia.
- **Bucles while** — repiten código mientras una condición sea verdadera.

En Python, los bloques se delimitan por **indentación** (4 espacios o 1 tab). No hay llaves `{}` como en otros lenguajes.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Sin condicionales, no podrías clasificar 480 estudiantes en aprobados/reprobados. Sin bucles, tendrías que escribir 480 veces el mismo código. La inteligencia de tu programa vive en estas estructuras.
::/interioriza

## Condicional `if`/`elif`/`else`

```python
calificacion = 7.5

if calificacion >= 9:
    print("Excelente")
elif calificacion >= 8:
    print("Muy bien")
elif calificacion >= 6:
    print("Aprobó")
else:
    print("Reprobó")
```

**Reglas:**
- `if` obligatorio.
- `elif` opcional, puedes tener varios.
- `else` opcional, máximo uno.
- Indentación de 4 espacios marca el bloque.

## Bucle `for`

```python
calificaciones = [8.5, 6.0, 9.5, 5.5, 7.8]

for cal in calificaciones:
    if cal >= 6:
        print(cal, "→ aprueba")
    else:
        print(cal, "→ reprueba")
```

**Output:**
```
8.5 → aprueba
6.0 → aprueba
9.5 → aprueba
5.5 → reprueba
7.8 → aprueba
```

Iteración con `range()` cuando necesitas índice:

```python
for i in range(5):
    print("Iteración", i)  # 0, 1, 2, 3, 4

for i in range(1, 11):
    print(i)  # 1 a 10
```

## Bucle `while`

```python
contador = 0
calificaciones_aprobadas = 0
calificaciones = [8.5, 6.0, 9.5, 5.5, 7.8]

while contador < len(calificaciones):
    if calificaciones[contador] >= 6:
        calificaciones_aprobadas += 1
    contador += 1

print("Aprobaron:", calificaciones_aprobadas, "de", len(calificaciones))
# Output: Aprobaron: 4 de 5
```

`while` es útil cuando NO sabes cuántas iteraciones necesitas (ej.: leer hasta encontrar fin de archivo).

## Patrones útiles

### Contar elementos que cumplen condición

```python
aprobados = 0
for cal in calificaciones:
    if cal >= 6:
        aprobados += 1
print("Aprobaron:", aprobados)
```

### Sumar / acumular

```python
suma = 0
for cal in calificaciones:
    suma += cal
promedio = suma / len(calificaciones)
print("Promedio:", promedio)
```

### `break` y `continue`

```python
for cal in calificaciones:
    if cal == 0:
        continue  # salta este, pasa al siguiente
    if cal == 10:
        break     # rompe el bucle entero
    print(cal)
```

## Anti-patrones

- **Olvidar indentación.** Python da `IndentationError`. Tu IDE muestra cuadrículas para ver indentación.
- **`if cal = 6`** (asignación) vs `if cal == 6` (comparación). Confusión común.
- **Modificar lista mientras la iteras.** Crea bugs. Si necesitas, copia primero.
- **`while` sin condición que cambia.** Loop infinito. Asegúrate que hay incremento o cambio.
- **Variables temporales mal nombradas.** `i, j, k` está bien para índices; `temp` no.

## Caso aplicado: clasificar 480 estudiantes

```python
# Lista simulada de calificaciones
calificaciones = [8.5, 6.0, 9.5, 5.5, 7.8, 4.2, 9.0]

categorias = {
    "excelente": 0,
    "muy_bien": 0,
    "aprobo": 0,
    "reprobo": 0
}

for cal in calificaciones:
    if cal >= 9:
        categorias["excelente"] += 1
    elif cal >= 8:
        categorias["muy_bien"] += 1
    elif cal >= 6:
        categorias["aprobo"] += 1
    else:
        categorias["reprobo"] += 1

print(categorias)
# Output: {'excelente': 2, 'muy_bien': 1, 'aprobo': 2, 'reprobo': 2}
```
