# 4.1 Media, mediana, varianza, desviación estándar

Estos 4 números resumen una variable. Reportarlos correctamente es la base de toda estadística descriptiva.

::concepto{titulo="Estadísticos descriptivos básicos"}
- **Media** (μ): suma dividida entre n. La "promedio".
- **Mediana**: valor central cuando ordenas. 50 % arriba, 50 % abajo.
- **Varianza** (σ²): promedio de desviaciones al cuadrado. Mide dispersión.
- **Desviación estándar** (σ): raíz cuadrada de la varianza. En las mismas unidades que los datos.

```python
df["cal_final"].mean()      # 7.42
df["cal_final"].median()    # 7.50
df["cal_final"].std()       # 1.49
df["cal_final"].var()       # 2.22
```
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Decir "el promedio fue 7" sin decir "y la desviación estándar fue 0.5" es ocultar información. Dos grupos con mismo promedio pueden tener historias muy distintas: uno homogéneo, otro polarizado.
::/interioriza

## Media vs mediana

```python
notas = [6, 7, 7, 8, 9]
mean = 7.4
median = 7

# Con outlier
notas_outlier = [6, 7, 7, 8, 99]
mean_o = 25.4   # ¡distorsionada!
median_o = 7    # robusta
```

Regla: si tus datos tienen outliers o son asimétricos, **reporta mediana**. Si son simétricos, media está bien.

## Cuándo cada medida

| Caso | Medida ideal |
|---|---|
| Calificaciones simétricas | Media |
| Salarios (asimétrico fuerte) | Mediana |
| Edades en escuela | Media |
| Tiempos de respuesta | Mediana |
| Conteo de errores | Mediana o moda |

## Desviación estándar: qué significa

Si media = 7 y std = 1, la mayoría de los datos cae entre 6 y 8 (≈68 % en distribución normal).

```python
notas = pd.Series([7, 7, 7, 7, 7])
notas.std()    # 0 (todos iguales)

notas = pd.Series([5, 6, 7, 8, 9])
notas.std()    # 1.58 (dispersos pero simétricos)
```

## Coeficiente de variación (CV)

CV = std / mean × 100 %. Útil para comparar dispersión entre grupos con medias distintas:

```python
cv_mate = df[df["materia"]=="Mate"]["cal_final"].std() / df[df["materia"]=="Mate"]["cal_final"].mean() * 100
# Ej: 22% — disperso

cv_quim = ... # 15% — más concentrado
```

## Reportar correctamente

❌ "Promedio fue 7.42."
✅ "Promedio 7.42 ± 1.49 (std), mediana 7.50, n=470 estudiantes."

## Anti-patrones

- **Reportar solo media** sin desviación.
- **Media para datos asimétricos**. Mejor mediana.
- **Mediana para datos simétricos sin razón** (pierde precisión).
- **No verificar n**. Una media de 5 datos no es comparable a una de 500.
