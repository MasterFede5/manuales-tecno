# 4.4 Correlación (Pearson) — sin matemáticas duras

"¿Correlacionan horas de estudio y calificación?". El coeficiente de correlación de Pearson lo responde con un número entre -1 y +1. Esta lección te enseña a calcularlo, interpretarlo y comunicarlo.

::concepto{titulo="Correlación de Pearson"}
El **coeficiente de correlación de Pearson** (r) mide la **fuerza y dirección** de relación lineal entre dos variables numéricas:

- **r = +1**: relación lineal positiva perfecta (cuando una sube, la otra sube exactamente).
- **r = 0**: sin relación lineal.
- **r = -1**: relación lineal negativa perfecta.

Reglas prácticas de interpretación:

| \|r\| | Fuerza |
|---|---|
| 0.0 – 0.1 | nula / despreciable |
| 0.1 – 0.3 | débil |
| 0.3 – 0.5 | moderada |
| 0.5 – 0.7 | fuerte |
| 0.7 – 0.9 | muy fuerte |
| 0.9 – 1.0 | casi perfecta |

```python
df["horas_estudio"].corr(df["cal_final"])  # 0.42 (moderada)
```
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Pearson es el primer paso de cualquier análisis predictivo. Si tu variable predictora correlaciona 0.05 con el target, no la uses; si correlaciona 0.7, es candidata fuerte. Sin esto, modelos de ML son ciegos.
::/interioriza

## Cómo calcular

```python
# Una variable contra otra
df["horas_estudio"].corr(df["cal_final"])  # 0.42

# Matriz de correlaciones (todas vs todas)
df[["horas_estudio", "asistencia", "cal_anterior", "cal_final"]].corr()
```

Output:
```
                horas_estudio  asistencia  cal_anterior  cal_final
horas_estudio        1.00           0.20         0.35       0.42
asistencia           0.20           1.00         0.45       0.55
cal_anterior         0.35           0.45         1.00       0.72
cal_final            0.42           0.55         0.72       1.00
```

Lectura: cal_anterior es el predictor más fuerte de cal_final (0.72).

## Limitaciones de Pearson

- **Solo lineal**. Si la relación es curva, Pearson da bajo aunque haya relación.
- **Sensible a outliers**. Un outlier extremo distorsiona r.
- **Solo numéricas**. Para categóricas usa otras (chi-cuadrado, no en este manual).

Para relaciones no lineales: **Spearman** (basado en rangos) o gráficos.

```python
df["x"].corr(df["y"], method="spearman")
```

## Interpretación con cuidado

Correlación de 0.42 NO significa "42 % causalidad". Significa: cuando una sube, la otra **tiende** a subir. La relación puede ser:
- Causal directa: A causa B.
- Causal inversa: B causa A.
- Confundida: C causa ambas.
- Coincidencia.

El próximo subtema (4.5) profundiza esto.

## R² (coeficiente de determinación)

`r²` mide qué porcentaje de varianza de Y se explica por X.

- r = 0.7 → r² = 0.49 → 49 % de la varianza explicada.
- r = 0.42 → r² = 0.18 → solo 18 % explicada.

Es la lectura honesta: una correlación 0.42 no es "casi mitad", es "menos de un quinto" de explicación.

## Anti-patrones

- **Reportar r sin contexto**. r=0.5 es fuerte en psicología, débil en física.
- **Asumir causalidad** desde correlación.
- **Ignorar Pearson cuando relación es curva**. Visualiza primero, calcula después.
- **R² alto sin pensar**. 0.95 puede ser sobreajuste o variable trivial.
