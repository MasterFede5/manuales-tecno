# 4.3 Percentiles y cuartiles

"Tu hijo está en el percentil 90 de su grupo" comunica más que "tu hijo sacó 8.5". Los percentiles ubican un valor en el ranking del grupo, **independientemente de la forma de la distribución**.

::concepto{titulo="Percentiles y cuartiles"}
El **percentil P** es el valor por debajo del cual se encuentra el P % de los datos:

- **P25** (Q1): 25 % de datos abajo. Cuartil 1.
- **P50** (Q2): mediana.
- **P75** (Q3): 75 % de datos abajo. Cuartil 3.
- **P90, P95, P99**: extremos altos.

**IQR** (Inter-Quartile Range) = Q3 − Q1: rango del 50 % central. Robusto a outliers.

```python
df["cal_final"].quantile(0.25)  # 6.5 (Q1)
df["cal_final"].quantile(0.5)   # 7.5 (mediana)
df["cal_final"].quantile(0.75)  # 8.5 (Q3)
df["cal_final"].quantile([0.1, 0.25, 0.5, 0.75, 0.9])  # vector
```
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Los percentiles son **el lenguaje del ranking**. Comunican posición sin requerir que entiendas la escala. "Top 10 %" es más intuitivo que "8.5 sobre 10" para alguien que no conoce la materia.
::/interioriza

## Lectura de un boxplot (ahora con números)

```
   |------|═══════[ ▌ ]═══════|------|
   min   Q1    mediana    Q3     max
        |←────IQR────→|

Bigotes: hasta 1.5 × IQR
Outliers: puntos más allá de bigotes
```

## Aplicaciones

```python
# Top 10% de alumnos
top10 = df[df["cal_final"] >= df["cal_final"].quantile(0.9)]

# En riesgo (bottom 25%)
en_riesgo = df[df["cal_final"] <= df["cal_final"].quantile(0.25)]

# Percentil de cada alumno
df["percentil"] = df["cal_final"].rank(pct=True) * 100
```

## Comparación entre grupos

```python
df.groupby("materia")["cal_final"].quantile([0.25, 0.5, 0.75]).unstack()
```

Resultado:
```
        0.25  0.5  0.75
materia
Mate     6.0  6.5   7.5
Física   7.0  7.5   8.0
Química  7.5  8.0   8.5
```

Mate tiene Q3 (7.5) inferior al Q2 (8.0) de Química. Toda la mitad superior de Mate cae por debajo de la mediana de Química.

## Cuándo usar percentiles vs media+std

| Caso | Mejor |
|---|---|
| Ranking individual | Percentil |
| Distribución asimétrica | Percentiles |
| Distribución simétrica normal | Media + std |
| Outliers presentes | Percentiles (IQR) |
| Comunicación a no-expertos | Percentiles ("top X%") |
| Cálculo de probabilidad | Media + std (si normal) |

## Anti-patrones

- **Confundir percentil con porcentaje de calificación**. P90 ≠ 90 % de aciertos.
- **Reportar percentil sin n**. P90 de 10 alumnos no significa lo mismo que P90 de 1000.
- **Usar percentiles en datos uniformes**. No aporta sobre rango directo.
