# 3.3 Seaborn para gráficos rápidos y bonitos

Si matplotlib es la base, **Seaborn** es la capa estética. Construido sobre matplotlib, ofrece gráficos estadísticos de calidad publicación en una línea.

::concepto{titulo="Seaborn"}
**Seaborn** (sns) es la librería de visualización estadística de alto nivel para Python. Sus 6 gráficos más usados:

- `sns.histplot` — histograma con KDE.
- `sns.scatterplot` — dispersión con colores por categoría.
- `sns.boxplot` — boxplot por grupos.
- `sns.violinplot` — combina boxplot + densidad.
- `sns.heatmap` — matriz de calor (correlaciones).
- `sns.pairplot` — todas las parejas de variables.

Usa **DataFrame de pandas directamente**: `sns.histplot(data=df, x="cal_final")`.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
La diferencia entre presentar gráficos matplotlib "feos" o seaborn "pulidos" en una junta cambia tu credibilidad. Seaborn no es solo cosmética: las elecciones de paleta, espaciado y tipografía vienen pensadas por estadísticos.
::/interioriza

## Histograma con seaborn

```python
import seaborn as sns

sns.histplot(data=df, x="cal_final", bins=20, kde=True)
```

Una línea. Histograma con curva de densidad superpuesta.

## Histograma por grupos

```python
sns.histplot(data=df, x="cal_final", hue="materia", bins=20, kde=True)
```

`hue="materia"` colorea por materia automáticamente con leyenda.

## Scatter por grupos

```python
sns.scatterplot(data=df, x="horas_estudio", y="cal_final", hue="materia", size="asistencia")
```

`size="asistencia"` hace los puntos más grandes según asistencia. 4 dimensiones en un solo gráfico.

## Boxplot

```python
sns.boxplot(data=df, x="materia", y="cal_final")
```

Una línea, comparación lista.

## Violinplot (boxplot + densidad)

```python
sns.violinplot(data=df, x="materia", y="cal_final")
```

Muestra forma de la distribución dentro de cada caja. Más información que boxplot.

## Heatmap (matriz de correlación)

```python
correlaciones = df[["horas_estudio", "asistencia", "cal_anterior", "cal_final"]].corr()
sns.heatmap(correlaciones, annot=True, cmap="coolwarm", center=0)
```

Visualiza qué variables correlacionan. `annot=True` muestra los valores numéricos.

## Pairplot (todas las parejas)

```python
sns.pairplot(df[["horas_estudio", "asistencia", "cal_final"]], hue="materia")
```

Genera matriz de scatter plots: cada par de variables en un gráfico, con histogramas en la diagonal. **Increíblemente útil** al inicio de un análisis para ver todo.

## Cuándo usar matplotlib vs seaborn

| Caso | Mejor opción |
|---|---|
| Gráfico estadístico rápido | Seaborn |
| Comparar grupos con `hue` | Seaborn |
| Customización extrema | Matplotlib |
| Subplots complejos | Matplotlib + ax |
| Producción exploratoria rápida | Seaborn |
| Publicación científica con guidelines | Matplotlib |

En la práctica: **usas ambos**. Seaborn por default; matplotlib para customizar lo que seaborn no permite.

## Anti-patrones

- **Reinventar gráficos seaborn en matplotlib**. Seaborn tiene casi todo. Busca primero.
- **Usar paleta default sin pensar**. Algunas paletas no son daltónicos-friendly. `viridis` es buena opción universal.
- **Pairplot con 20 variables**. Crece N². Limita a 5-7.
- **`heatmap` sin `annot`** cuando son pocos valores. Mejor con números.
