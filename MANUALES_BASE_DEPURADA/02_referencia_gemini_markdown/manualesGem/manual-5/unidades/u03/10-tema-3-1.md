# 3.1 Matplotlib esencial: línea, barras, dispersión

Matplotlib es **la base** de visualización en Python. Casi todas las otras librerías (Seaborn, Plotly, etc.) construyen sobre ella. Aprender los 3 gráficos básicos abre todo lo demás.

::concepto{titulo="Matplotlib"}
**Matplotlib** es la librería de visualización 2D más usada en Python. Su API más común es **`pyplot`** (importada como `plt`). Tres gráficos cubren 80 % de los casos:

- **Línea** (`plt.plot`) — tendencias en el tiempo.
- **Barras** (`plt.bar`) — comparar categorías.
- **Dispersión** (`plt.scatter`) — relación entre dos variables numéricas.

```python
import matplotlib.pyplot as plt
plt.plot([1, 2, 3], [4, 5, 6])
plt.show()
```

Hay dos estilos: **pyplot funcional** (más simple) y **objeto-orientado fig/ax** (más control). Para principiante: empieza con pyplot.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Una tabla con 100 filas no entra en una junta. Una gráfica sí. Saber traducir tabla → gráfica con un mensaje claro es la diferencia entre "te leí el reporte" y "ahora entiendo el problema".
::/interioriza

## Gráfico de línea (tendencia)

```python
import matplotlib.pyplot as plt

semanas = [1, 2, 3, 4, 5, 6]
promedio = [7.2, 7.5, 7.8, 7.6, 8.0, 8.2]

plt.plot(semanas, promedio, marker="o", color="blue")
plt.title("Promedio del grupo por semana")
plt.xlabel("Semana")
plt.ylabel("Promedio")
plt.grid(True)
plt.show()
```

Cuándo: tendencias en el tiempo, evolución, series temporales.

## Gráfico de barras (comparar categorías)

```python
materias = ["Mate", "Física", "Química"]
promedios = [6.95, 7.42, 7.65]

plt.bar(materias, promedios, color=["#ff6b6b", "#4ecdc4", "#ffe66d"])
plt.title("Promedio por materia")
plt.ylabel("Promedio")
plt.ylim(0, 10)
plt.show()
```

Cuándo: comparar valores entre categorías discretas.

## Gráfico de dispersión (relación entre 2 variables)

```python
import pandas as pd
df = pd.read_csv("dataset_escolar.csv")

plt.scatter(df["horas_estudio"], df["cal_final"], alpha=0.5)
plt.title("Horas de estudio vs calificación final")
plt.xlabel("Horas de estudio semanal")
plt.ylabel("Calificación final")
plt.show()
```

Cuándo: explorar correlación entre dos variables numéricas.

## Anatomía: figure y axes

Para más control:

```python
fig, ax = plt.subplots(figsize=(10, 6))
ax.bar(materias, promedios)
ax.set_title("Promedio por materia")
ax.set_xlabel("Materia")
ax.set_ylabel("Promedio")
ax.set_ylim(0, 10)
plt.savefig("promedios.png", dpi=150)
plt.show()
```

`fig` es el lienzo, `ax` es el área del gráfico. Útil para múltiples subplots.

## Subplots (varios gráficos en uno)

```python
fig, axes = plt.subplots(1, 3, figsize=(15, 5))

axes[0].bar(materias, promedios)
axes[0].set_title("Promedio")

axes[1].scatter(df["horas_estudio"], df["cal_final"], alpha=0.3)
axes[1].set_title("Horas vs Cal")

axes[2].plot(semanas, promedio, marker="o")
axes[2].set_title("Tendencia")

plt.tight_layout()
plt.show()
```

## Guardar gráfico

```python
plt.savefig("grafico.png", dpi=150, bbox_inches="tight")
plt.savefig("grafico.pdf")  # vectorial para imprimir
```

`dpi=150` para web, 300+ para impresión. `bbox_inches="tight"` recorta espacios extra.

## Anti-patrones

- **No poner títulos ni etiquetas.** Tu gráfica no se entiende sin contexto.
- **Demasiados colores.** 3-5 max. Más confunde.
- **Eje y no empieza en cero en gráfico de barras.** Distorsiona percepción.
- **Líneas para datos categóricos.** "Línea" implica tendencia continua.
- **Olvidar `plt.show()`** en algunos contextos.
