# 8.2 Tu primer chatbot en 30 líneas

Un chatbot mantiene conversación: recuerda lo dicho, responde con coherencia. La clave: **historial de mensajes**.

::concepto{titulo="Chatbot con historial"}
Un **chatbot conversacional** mantiene contexto enviando todo el historial de mensajes en cada call. Estructura:

```python
messages = [
    {"role": "system", "content": "Instrucciones generales..."},
    {"role": "user", "content": "Mensaje 1"},
    {"role": "assistant", "content": "Respuesta 1"},
    {"role": "user", "content": "Mensaje 2"},
    # ...
]
```

En cada turno, agregas el nuevo mensaje del usuario, mandas TODO al modelo, y agregas su respuesta al historial.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Sin historial, cada pregunta es independiente. Con historial, el bot recuerda contexto: "¿Y por qué?" se entiende solo si recuerda la pregunta anterior.
::/interioriza

## Chatbot básico (Claude)

```python
import os
from anthropic import Anthropic

client = Anthropic(api_key=os.environ.get("ANTHROPIC_API_KEY"))

messages = []
system_prompt = "Eres un asistente educativo conciso y amable."

print("Chatbot listo. Escribe 'salir' para terminar.\n")

while True:
    user_msg = input("Tú: ")
    if user_msg.lower() == "salir":
        break
    
    messages.append({"role": "user", "content": user_msg})
    
    response = client.messages.create(
        model="claude-haiku-4-5",
        system=system_prompt,
        max_tokens=500,
        messages=messages
    )
    
    assistant_msg = response.content[0].text
    messages.append({"role": "assistant", "content": assistant_msg})
    
    print(f"\nClaude: {assistant_msg}\n")
```

30 líneas, chatbot funcional con memoria.

## System prompt: el rol del asistente

```python
system_prompt = """Tu rol específico aquí.

Reglas:
- Sé conciso.
- Cita datos específicos cuando sea posible.
- No inventes información que no tengas.
- Si no sabes, dilo."""
```

System prompt persistente influye TODOS los turnos.

## Limitaciones del historial

Cada call manda TODO el historial → costo crece linealmente. Para sesiones largas:

```python
# Limitar a últimos 20 mensajes
messages = messages[-20:]

# O resumir cada N turnos
if len(messages) > 30:
    summary = client.messages.create(
        model="claude-haiku-4-5",
        max_tokens=200,
        messages=[{"role": "user", "content": f"Resume esta conversación: {messages}"}]
    )
    messages = [{"role": "user", "content": f"Contexto previo: {summary}"}]
```

## Anti-patrones

- **No usar system prompt**. El bot pierde rol.
- **Historial infinito sin límite**. Costo descontrolado.
- **No manejar errores de API**. Una falla y bot muere.
