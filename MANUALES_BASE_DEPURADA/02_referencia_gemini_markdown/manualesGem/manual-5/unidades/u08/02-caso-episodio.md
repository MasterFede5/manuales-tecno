**Caso Albatros — Episodio 8: Tu predictor explica resultados con IA generativa.**

Tu predictor de U5/U7 dice "alumno X tendrá 5.5 final → riesgo". El coordinador pregunta lo único importante: *"¿Por qué? ¿Qué hago para evitarlo?"*. Tu modelo no responde — es solo un número. **GPT/Claude sí**.

El Episodio 8 cierra el manual integrando ML clásico con IA generativa. Cubrirás API setup, primer chatbot, streaming, tool use, embeddings, mini-RAG sobre PDFs institucionales, despliegue Streamlit, y buenas prácticas (costo, errores, caching). Al final, **app Streamlit funcional** que toma datos de un alumno, predice rendimiento, y explica con IA por qué + qué hacer.

> *La pregunta gancho:* ¿cuántas líneas de Python necesitas para conversar con Claude? La respuesta: 5. Empezamos.
