# 8.1 Configurar API de Claude / OpenAI

Para usar IA generativa programáticamente necesitas API key + SDK. Esta lección te lleva del setup al primer "Hola" en 15 minutos.

::concepto{titulo="API de modelo generativo"}
Las **APIs de IA generativa** te permiten usar Claude, GPT, Gemini desde código en lugar del chat web. Ventajas:
- Automatización (procesar 1000 textos sin pegar manualmente).
- Integración (en tu app, web, workflow).
- Control fino (temperature, max_tokens, system prompt).

Setup canónico:
1. Crear cuenta en proveedor.
2. Generar API key.
3. Guardar key como variable de entorno (NUNCA en código).
4. Instalar SDK.
5. Primer call.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Sin API: dependes de chat manual. Con API: ChatGPT/Claude se vuelve **componente de tu software**. Ahí está el potencial real de IA programática.
::/interioriza

## Anthropic Claude

```bash
pip install anthropic
```

```python
import os
from anthropic import Anthropic

client = Anthropic(api_key=os.environ.get("ANTHROPIC_API_KEY"))

response = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "Hola, explícame en 2 frases qué es ML."}
    ]
)
print(response.content[0].text)
```

## OpenAI

```bash
pip install openai
```

```python
import os
from openai import OpenAI

client = OpenAI(api_key=os.environ.get("OPENAI_API_KEY"))

response = client.chat.completions.create(
    model="gpt-5",
    messages=[
        {"role": "user", "content": "Hola, explícame en 2 frases qué es ML."}
    ]
)
print(response.choices[0].message.content)
```

## Variables de entorno (seguridad)

**NUNCA** pegues API keys en código. Tres formas correctas:

### En Colab
```python
from google.colab import userdata
api_key = userdata.get('ANTHROPIC_API_KEY')
```

### En script local con .env
```bash
# .env (no subir a Git, agregar a .gitignore)
ANTHROPIC_API_KEY=sk-ant-xxx
```

```python
from dotenv import load_dotenv
load_dotenv()
```

### En terminal antes
```bash
export ANTHROPIC_API_KEY="sk-ant-xxx"
python script.py
```

## Anti-patrones

- **API key en código fuente**. La filtras en GitHub. Bots la usan, te llega factura $$$.
- **Sin límite de tokens / max_tokens alto**. Costo descontrolado.
- **Probar con Opus para "Hola mundo"**. Usa Haiku/mini para tests.
