# 8.6 Despliegue con Streamlit

Tu código en notebook funciona. Para que la coordinadora lo use sin abrir Colab, lo conviertes en **app web** con Streamlit. Sin HTML, sin CSS, sin servidor.

::concepto{titulo="Streamlit"}
**Streamlit** es framework Python para construir apps web de data science con código Python puro. Sin necesidad de saber JavaScript, HTML o frontend.

```python
import streamlit as st

st.title("Mi App")
nombre = st.text_input("Tu nombre")
if st.button("Saludar"):
    st.write(f"Hola, {nombre}")
```

Corres con `streamlit run app.py` y se abre en navegador. Despliegue: Streamlit Community Cloud (gratis), Hugging Face Spaces, Render, Heroku.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Sin Streamlit: tu modelo vive en notebook que solo tú entiendes. Con Streamlit: cualquiera lo usa desde su navegador. Es la pieza que cierra el ciclo "investigación → producto usable".
::/interioriza

## App básica

```python
# app.py
import streamlit as st
import joblib
import pandas as pd
import os
from anthropic import Anthropic

# Cargar modelo
model = joblib.load("modelo_predictor_v1.pkl")

claude = Anthropic(api_key=st.secrets["ANTHROPIC_API_KEY"])

st.title("🎓 Predictor de Rendimiento Albatros")
st.markdown("Predicción + explicación con IA")

# Inputs
col1, col2, col3 = st.columns(3)
with col1:
    horas = st.slider("Horas de estudio/semana", 0.0, 10.0, 4.0)
with col2:
    asistencia = st.slider("Asistencia", 0.0, 1.0, 0.85)
with col3:
    cal_ant = st.slider("Calificación anterior", 0.0, 10.0, 7.0)

if st.button("Predecir y explicar"):
    # Predecir
    X = pd.DataFrame([[horas, asistencia, cal_ant]], 
                     columns=["horas_estudio", "asistencia", "cal_anterior"])
    pred = model.predict(X)[0]
    
    # Mostrar
    st.metric("Calificación predicha", f"{pred:.2f}")
    
    # Explicar con IA
    riesgo = "ALTO" if pred < 6 else ("MEDIO" if pred < 7 else "BAJO")
    
    prompt = f"""Eres asesor académico. Un alumno con:
- Horas estudio: {horas}/semana
- Asistencia: {asistencia*100:.0f}%
- Calificación anterior: {cal_ant}

Tiene predicción de calificación final: {pred:.2f}. Riesgo: {riesgo}.

Explica en 3 partes:
1. Por qué la predicción (factores principales).
2. Qué cambiar para mejorar.
3. Acción concreta sugerida al coordinador."""
    
    with st.spinner("Generando explicación..."):
        response = claude.messages.create(
            model="claude-sonnet-4-6",
            max_tokens=600,
            messages=[{"role": "user", "content": prompt}]
        )
        explicacion = response.content[0].text
    
    st.markdown("### Análisis")
    st.write(explicacion)
```

Run: `streamlit run app.py` → abre `localhost:8501`.

## Manejo de secrets

```python
# .streamlit/secrets.toml (NO subir a Git)
ANTHROPIC_API_KEY = "sk-xxx"
```

```python
# En código
api_key = st.secrets["ANTHROPIC_API_KEY"]
```

## Anti-patrones

- **API keys en código fuente o subidas a Git**.
- **App sin `@st.cache_resource`** para modelo: recarga cada interacción.
- **Sin manejo de errores**: si API cae, app explota.
- **Diseño "default ChatGPT clone"**: aprovecha layout de columns y tabs.
