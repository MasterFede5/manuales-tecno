# 2.5 Filtrado y selección

Tu DataFrame tiene 480 estudiantes. Quieres "los aprobados de bachillerato técnico con promedio >8". Esto es **filtrado**, una de las habilidades más usadas en datos.

::concepto{titulo="Filtrado y selección en pandas"}
Pandas ofrece **4 técnicas** principales para extraer subconjuntos de un DataFrame:

1. **Boolean indexing** — el más común, usando expresiones lógicas.
2. **`.loc` / `.iloc`** — selección por etiqueta o posición.
3. **`.query()`** — sintaxis tipo SQL en string.
4. **`.isin()`** — pertenencia a lista de valores.

Cada uno tiene su caso ideal. Aprender los 4 te da fluidez completa.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Cuando una hoja de Excel filtra con el embudo, hace lo mismo. La diferencia: pandas es **reproducible** (queda en código), **encadenable** (filtras → agrupas → calculas en una línea), y **escalable** (millones de filas).
::/interioriza

## Boolean indexing (el básico)

```python
# Filtros simples
aprobados = df[df["calificacion"] >= 6]
top = df[df["calificacion"] > 9]

# Múltiples condiciones (AND, OR)
exitosos = df[(df["calificacion"] >= 8) & (df["asistencia"] >= 0.85)]
en_riesgo = df[(df["calificacion"] < 6) | (df["asistencia"] < 0.7)]

# Negación
no_aprobados = df[~(df["calificacion"] >= 6)]
```

**Crucial:** usar `&`, `|`, `~` (no `and`, `or`, `not`) y **paréntesis** alrededor de cada condición.

## `.loc` y `.iloc`

```python
# .loc: selección por etiqueta
df.loc[df["calificacion"] >= 6, "nombre"]                  # nombres de aprobados
df.loc[0:5, ["nombre", "calificacion"]]                    # filas 0-5, columnas elegidas
df.loc[df["calificacion"] >= 9, ["nombre", "calificacion"]]  # nombres+calif top

# .iloc: por posición
df.iloc[0:5, 0:3]   # primeras 5 filas, primeras 3 columnas
df.iloc[-1]         # última fila
```

Asignación con `.loc`:
```python
df.loc[df["calificacion"] >= 6, "estado"] = "Aprobado"
df.loc[df["calificacion"] < 6, "estado"] = "Reprobado"
```

## `.query()` (sintaxis tipo SQL)

```python
# Equivalente a boolean indexing pero más legible
aprobados = df.query("calificacion >= 6")
exitosos = df.query("calificacion >= 8 and asistencia >= 0.85")

# Variables externas con @
umbral = 7
top = df.query("calificacion >= @umbral")
```

`.query` es excelente para condiciones complejas con muchos `and`/`or`.

## `.isin()` (pertenencia)

```python
# Filtra alumnos cuyo grupo está en lista
grupos_objetivo = ["A1", "B2", "C3"]
seleccion = df[df["grupo"].isin(grupos_objetivo)]

# Negación
otros = df[~df["grupo"].isin(grupos_objetivo)]
```

## Selección de columnas

```python
# Una columna
df["nombre"]               # devuelve Series
df[["nombre"]]             # devuelve DataFrame

# Varias
df[["nombre", "calificacion"]]

# Por tipo
df.select_dtypes(include="number")   # solo numéricas
df.select_dtypes(include="object")   # solo texto

# Por patrón en nombre
df.filter(like="cal")               # columnas que contienen "cal"
df.filter(regex="^cal_")             # columnas que empiezan con "cal_"
```

## Combinaciones encadenadas (idiom pandas)

```python
# Top 5 alumnos con cal>8 y asistencia>0.9, mostrando nombre y promedio
resultado = (df
    .query("calificacion > 8 and asistencia > 0.9")
    .nlargest(5, "calificacion")
    [["nombre", "calificacion", "asistencia"]]
)
```

Pandas idiomático = encadenamiento. Cada línea es una transformación.

## Patrones útiles

### Top N

```python
df.nlargest(5, "calificacion")     # top 5 por calificación
df.nsmallest(3, "calificacion")    # bottom 3
```

### Sample aleatorio

```python
df.sample(10)                      # 10 filas al azar
df.sample(frac=0.1)                # 10% de las filas
```

### Selección con string contains

```python
df[df["materia"].str.contains("Matemática")]
df[df["nombre"].str.startswith("A")]
```

## Anti-patrones

- **`and` en lugar de `&`**. Da error o resultado incorrecto. Usa siempre `&` y paréntesis.
- **Filtros encadenados en líneas separadas no atómicas**. `df = df[cond1]; df = df[cond2]` modifica `df`. Mejor `df_filtrado = df[cond1 & cond2]`.
- **Olvidar `.copy()` cuando vas a modificar**. Si filtras y modificas, puede dar SettingWithCopyWarning. `df_sub = df[cond].copy()`.
- **`.query` con caracteres especiales en columnas**. `query("cal final >= 6")` falla. Renombra primero o usa boolean indexing.
