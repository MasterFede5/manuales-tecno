# 2.4 Limpieza: nulos, duplicados, tipos

Datos del mundo real **siempre** vienen sucios. Limpieza es 60-70 % del trabajo de un data scientist. Esta lección te enseña los 4 problemas más comunes y cómo resolverlos.

::concepto{titulo="Limpieza de datos"}
Los **4 problemas canónicos** que pandas resuelve:

1. **Valores faltantes (NaN)** — celdas vacías o "N/A".
2. **Duplicados** — filas repetidas por error.
3. **Tipos incorrectos** — números guardados como strings, fechas mal parseadas.
4. **Valores fuera de rango** — outliers, errores de captura.

Cada uno tiene métodos pandas específicos para detectar y corregir.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Modelo entrenado con datos sucios = modelo malo. La calidad de tu predicción depende **directamente** de la calidad de tu limpieza. La frase "garbage in, garbage out" es ley.
::/interioriza

## Detectar valores faltantes (NaN)

```python
df.isna()                     # DataFrame de booleans
df.isna().sum()               # cuántos NaN por columna
df.isna().sum().sum()         # total NaN

# Visualizar dónde están
df[df["calificacion"].isna()]  # filas con calificacion NaN
```

## Manejar valores faltantes

Tres opciones:

```python
# 1. Eliminar filas con NaN
df_limpio = df.dropna()                     # cualquier NaN
df_limpio = df.dropna(subset=["calificacion"])  # solo si NaN en esa columna

# 2. Rellenar con valor
df["calificacion"] = df["calificacion"].fillna(0)
df["asistencia"] = df["asistencia"].fillna(df["asistencia"].mean())  # con media

# 3. Rellenar con valor anterior (forward fill, para series temporales)
df["temperatura"] = df["temperatura"].fillna(method="ffill")
```

**Decisión de qué hacer**: depende del caso.
- Si los NaN son raros (<5 %) y la fila no aporta nada → eliminar.
- Si son sistemáticos (toda una columna) → investigar la fuente.
- Si son importantes pero pocos → imputar con media/mediana.
- Si son numéricos y la cero es válido → fillna(0).

## Eliminar duplicados

```python
# Detectar
df.duplicated()                 # Series boolean
df.duplicated().sum()           # cuántos
df[df.duplicated(keep=False)]   # ver todas las filas duplicadas

# Eliminar
df_limpio = df.drop_duplicates()
df_limpio = df.drop_duplicates(subset=["id"], keep="first")  # por columna específica
```

## Convertir tipos

```python
# Ver tipos actuales
df.dtypes

# Convertir
df["edad"] = df["edad"].astype(int)
df["calificacion"] = df["calificacion"].astype(float)
df["fecha"] = pd.to_datetime(df["fecha"])
df["categoria"] = df["categoria"].astype("category")  # eficiente para texto repetido

# Cuando astype falla por valores raros
df["edad"] = pd.to_numeric(df["edad"], errors="coerce")  # convierte raros a NaN
```

## Manejo de strings

```python
df["nombre"] = df["nombre"].str.strip()          # quitar espacios
df["nombre"] = df["nombre"].str.lower()          # minúsculas
df["nombre"] = df["nombre"].str.replace("  ", " ")  # doble espacio → simple

# Reemplazar valores sentinela
df["calificacion"] = df["calificacion"].replace({"N/A": None, "?": None})
```

## Outliers (valores fuera de rango)

```python
# Detectar valores raros con .describe()
df["calificacion"].describe()
# si min=-5 o max=100, hay errores (calificación debería ser 0-10)

# Validar rangos
df_limpio = df[(df["calificacion"] >= 0) & (df["calificacion"] <= 10)]

# Clip (acotar a rango)
df["calificacion"] = df["calificacion"].clip(0, 10)
```

## Pipeline completo de limpieza

```python
def limpiar_dataset(df):
    # 1. Quitar duplicados por id
    df = df.drop_duplicates(subset=["id"], keep="first")
    
    # 2. Validar rangos
    df = df[(df["calificacion"] >= 0) & (df["calificacion"] <= 10)]
    df = df[(df["asistencia"] >= 0) & (df["asistencia"] <= 1)]
    
    # 3. Manejar NaN
    df = df.dropna(subset=["calificacion"])  # eliminar si calif es NaN
    df["asistencia"] = df["asistencia"].fillna(df["asistencia"].mean())
    
    # 4. Tipos
    df["id"] = df["id"].astype(int)
    df["calificacion"] = df["calificacion"].astype(float)
    
    # 5. Reset index
    df = df.reset_index(drop=True)
    
    return df

df_limpio = limpiar_dataset(df)
print(f"Antes: {len(df)} → Después: {len(df_limpio)}")
```

## Documentación de limpieza

**Siempre** registra qué eliminaste:

```python
print(f"Filas iniciales: {len(df)}")
df_dup = df.drop_duplicates(subset=["id"])
print(f"Después de duplicados: {len(df_dup)} (eliminados {len(df)-len(df_dup)})")
df_nan = df_dup.dropna(subset=["calificacion"])
print(f"Después de NaN: {len(df_nan)} (eliminados {len(df_dup)-len(df_nan)})")
```

Si más adelante alguien pregunta "¿de dónde sacaste estos números?", tienes traza.

## Anti-patrones

- **Borrar todo lo que tenga NaN sin pensar.** Si la columna que tiene NaN es opcional, eliminas datos buenos.
- **Imputar con media sin justificar.** Cambias la distribución. Documenta la decisión.
- **`drop_duplicates()` sin `subset`.** Si el DataFrame tiene noise (timestamps), nada se borra. Especifica columnas clave.
- **Olvidar `reset_index`.** Tras filtrar, índice queda con saltos. Importante para concatenaciones después.
- **Limpiar in-place sin guardar.** Si lo haces mal, perdiste el original. Trabaja sobre copia.
