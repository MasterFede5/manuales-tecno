# 2.1 NumPy: arrays y operaciones básicas

NumPy (*Numerical Python*) es la **base** sobre la que se construye Pandas, scikit-learn, PyTorch, todo el stack de datos en Python. Aprender NumPy bien es invertir en entender todo lo demás.

::concepto{titulo="NumPy y arrays"}
**NumPy** es la librería fundamental de cómputo numérico en Python. Su pieza central es el **`ndarray`** (n-dimensional array): una estructura de datos similar a una lista de Python pero **homogénea** (todos los elementos del mismo tipo) y **vectorizada** (operaciones se aplican a todos los elementos a la vez sin loop explícito).

```python
import numpy as np

calificaciones = np.array([8.5, 6.0, 9.5, 5.5, 7.8])
# Operación vectorizada
porcentaje = calificaciones * 10  # array([85., 60., 95., 55., 78.])
```

Ventajas:
- **10-100× más rápido** que listas para operaciones numéricas.
- **Sintaxis natural** (operadores funcionan elemento a elemento).
- **Funciones matemáticas** masivas: media, std, min, max, percentiles.
- **Multidimensional**: matrices 2D, tensores 3D para imágenes y redes neuronales.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Si en U1 procesar 480 calificaciones tomó loop de 5 líneas, con NumPy es 1. Y si procesas 4.8 millones (un dataset real), la diferencia entre tu programa que tarda 1 segundo vs 100 segundos es justamente esta vectorización.
::/interioriza

## Crear arrays

```python
import numpy as np

# Desde lista
a = np.array([1, 2, 3, 4, 5])

# Rangos
b = np.arange(0, 10)         # [0,1,2,...,9]
c = np.linspace(0, 1, 5)     # [0, 0.25, 0.5, 0.75, 1.0]

# Con valor inicial
zeros = np.zeros(5)          # [0., 0., 0., 0., 0.]
ones = np.ones(5)            # [1., 1., 1., 1., 1.]
fives = np.full(5, 5.0)      # [5., 5., 5., 5., 5.]

# 2D (matriz)
matriz = np.array([[1, 2, 3], [4, 5, 6]])
print(matriz.shape)          # (2, 3) — 2 filas, 3 columnas
```

## Operaciones vectorizadas

```python
calificaciones = np.array([8.5, 6.0, 9.5, 5.5, 7.8])

# Aritméticas
calificaciones + 1            # array([9.5, 7. , 10.5, 6.5, 8.8])
calificaciones * 0.5
calificaciones ** 2

# Estadísticas
calificaciones.mean()         # 7.46
calificaciones.std()          # 1.49
calificaciones.min()          # 5.5
calificaciones.max()          # 9.5
calificaciones.sum()          # 37.3
np.percentile(calificaciones, 50)  # 7.8 (mediana)
```

## Indexación y slicing

```python
a = np.array([10, 20, 30, 40, 50])

a[0]           # 10
a[-1]          # 50
a[1:4]         # array([20, 30, 40])

# Boolean indexing (filtrado)
mask = a > 25
a[mask]        # array([30, 40, 50])

# Múltiples condiciones
a[(a > 15) & (a < 45)]  # array([20, 30, 40])
```

## Operaciones entre arrays

```python
horas = np.array([5, 6, 4, 7, 5])
asistencia = np.array([0.9, 0.85, 0.95, 0.7, 0.92])

# Operaciones elemento a elemento
indice_esfuerzo = horas * asistencia
# array([4.5, 5.1, 3.8, 4.9, 4.6])

# Comparación elemento a elemento
sobresalientes = (horas >= 5) & (asistencia >= 0.85)
# array([True, True, False, False, True])
```

## Agregaciones por eje (en 2D)

```python
notas = np.array([
    [8, 7, 9],   # alumno 1: mate, fis, qui
    [6, 5, 7],   # alumno 2
    [9, 9, 10]   # alumno 3
])

# Promedio por alumno (filas, axis=1)
notas.mean(axis=1)   # array([8., 6., 9.33])

# Promedio por materia (columnas, axis=0)
notas.mean(axis=0)   # array([7.67, 7., 8.67])
```

## Reshape y transposición

```python
a = np.arange(12)              # 1D, 12 elementos
a.reshape(3, 4)                # 2D, 3x4
a.reshape(2, 2, 3)             # 3D
a.reshape(-1, 4)               # -1 = "calcula tú", da 3x4

# Transpuesta
m = np.array([[1, 2], [3, 4]])
m.T                            # array([[1, 3], [2, 4]])
```

## NaN (Not a Number) — datos faltantes

```python
notas = np.array([8.5, np.nan, 7.0, np.nan, 9.0])

notas.mean()              # nan (un nan contamina)
np.nanmean(notas)         # 8.17 (ignora nans)
np.isnan(notas)           # array([False, True, False, True, False])
```

## Anti-patrones

- **Loop manual sobre array**. `for i in range(len(arr)): arr[i] = ...` desperdicia NumPy. Vectoriza.
- **Mezclar tipos**. NumPy promueve al tipo más permisivo. `np.array([1, "hola"])` da array de strings, no error.
- **`.mean()` con NaN sin pensar**. Da `nan`. Usa `nanmean`, `nansum`, etc.
- **No verificar `.shape`**. Bug típico: forma incorrecta. Imprime shape en debug.
- **Reinventar funciones**. NumPy tiene `min`, `max`, `sort`, `unique`, `diff`, `cumsum`, etc. Búscalas antes de programar.
