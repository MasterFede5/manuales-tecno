# 2.6 Agrupación (groupby) y agregaciones

"¿Cuál es el promedio de calificación **por materia**?". "¿Cuántos estudiantes **por nivel** aprobaron?". Estas preguntas las responde `groupby` en una línea.

::concepto{titulo="GroupBy y agregaciones"}
**GroupBy** es la operación que **agrupa** filas por uno o más valores y luego **aplica** una función agregada (suma, promedio, conteo, max, min) a cada grupo, devolviendo un DataFrame con los resultados.

Sigue el patrón **split-apply-combine**:
1. **Split**: divide el DataFrame en grupos según valores de columna.
2. **Apply**: aplica función agregada a cada grupo.
3. **Combine**: combina los resultados en un nuevo DataFrame.

```python
df.groupby("materia")["calificacion"].mean()
# Series con promedio por materia
```

Es **la operación más poderosa** de pandas para análisis exploratorio.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
En Excel, hacer "promedio por materia" requiere tabla dinámica con configuración. En pandas: una línea, reproducible, escalable. Si dominas groupby, dominas el 80 % del análisis exploratorio.
::/interioriza

## Sintaxis básica

```python
# Promedio por categoría
df.groupby("materia")["calificacion"].mean()

# Múltiples agregaciones
df.groupby("materia")["calificacion"].agg(["mean", "min", "max", "count"])

# Por múltiples columnas
df.groupby(["nivel", "materia"])["calificacion"].mean()

# Múltiples columnas a agregar
df.groupby("materia")[["calificacion", "asistencia"]].mean()

# Agregaciones diferentes por columna
df.groupby("materia").agg({
    "calificacion": "mean",
    "asistencia": "mean",
    "horas_estudio": "sum"
})
```

## Funciones de agregación más usadas

| Función | Qué calcula |
|---|---|
| `mean()` | promedio |
| `median()` | mediana |
| `sum()` | suma |
| `count()` | conteo (no incluye NaN) |
| `size()` | tamaño (incluye NaN) |
| `min()`, `max()` | mínimo, máximo |
| `std()`, `var()` | desviación, varianza |
| `nunique()` | valores únicos |
| `first()`, `last()` | primero, último |
| `quantile(0.5)` | mediana / percentil |

## Función personalizada con apply

```python
# Tasa de aprobación por materia
def tasa_aprobacion(grupo):
    return (grupo["calificacion"] >= 6).mean()

df.groupby("materia").apply(tasa_aprobacion)
```

## Transformaciones (no agregadas)

`transform` devuelve algo del mismo tamaño que el grupo, útil para crear columnas comparativas:

```python
# Diferencia con promedio del grupo
df["promedio_materia"] = df.groupby("materia")["calificacion"].transform("mean")
df["diff_promedio"] = df["calificacion"] - df["promedio_materia"]

# Ranking dentro del grupo
df["rank_en_materia"] = df.groupby("materia")["calificacion"].rank(ascending=False)
```

## Aplicado al caso escolar

```python
# Estadísticas completas por materia
resumen = df.groupby("materia").agg(
    n_alumnos=("id", "count"),
    cal_promedio=("calificacion", "mean"),
    cal_mediana=("calificacion", "median"),
    cal_std=("calificacion", "std"),
    asistencia_promedio=("asistencia", "mean"),
    tasa_aprobacion=("calificacion", lambda x: (x >= 6).mean())
).round(2)

print(resumen)
#            n_alumnos  cal_promedio  cal_mediana  cal_std  asistencia_promedio  tasa_aprobacion
# materia
# Física           160          7.42          7.5     1.49                 0.85             0.81
# Matemáticas      160          6.95          7.0     1.62                 0.83             0.74
# Química          160          7.65          8.0     1.41                 0.87             0.85
```

Una sola línea. Resultado idéntico a 30 minutos de tabla dinámica de Excel.

## Multi-nivel (group por varias columnas)

```python
df.groupby(["nivel", "materia"])["calificacion"].mean()
```

Resultado: índice multi-nivel. Para "aplanar":

```python
df.groupby(["nivel", "materia"])["calificacion"].mean().reset_index()
```

## Anti-patrones

- **`groupby().apply(lambda x: x.mean())`** cuando puedes `groupby().mean()`. Más lento.
- **Olvidar `.reset_index()`**. Tu resultado tiene índice multi-nivel raro al guardar a CSV.
- **Agregación sin nombrar**. `df.groupby("x").agg({"y": "mean", "y": "sum"})` colisiona. Usa nombres: `agg(media=("y", "mean"), suma=("y", "sum"))`.
- **GroupBy con NaN sin pensar**. Si tu columna de agrupación tiene NaN, esas filas se descartan por default.
