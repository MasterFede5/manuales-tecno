### Subtema 5.1 — Estequiometría mol-mol, masa-masa

::act-calc{titulo="Estequiometría básica" renglones=12}
- Para cada reacción, balancea la ecuación.
- Identifica la relación estequiométrica.
- Resuelve paso a paso.

a) $2\,Mg + O_2 \to 2\,MgO$. ¿Cuántos g de MgO obtienes a partir de 24 g de Mg? (M_Mg = 24, M_MgO = 40.3)

b) $C_3H_8 + 5\,O_2 \to 3\,CO_2 + 4\,H_2O$. ¿Cuántos litros de CO₂ a CNTP produces al quemar 22 g de propano? (M = 44.10)

c) $\ce{2 NaOH + H2SO4 -> Na2SO4 + 2 H2O}$. ¿Cuántos gramos de NaOH neutralizan 9.8 g de H₂SO₄? (M_NaOH = 40, M_H₂SO₄ = 98)

d) Para tratar 5 m³ de agua del bebedero con dosis 0.8 mg/L de Cl libre, ¿cuántos g de NaOCl al 5 % en peso necesitas? (M_NaOCl = 74.44, M_Cl = 35.45)
::/act-calc

### Subtema 5.6 — Cinética química

::interioriza
La cinética química es como el tráfico: un catalizador es un carril exprés que reduce el tiempo de viaje (energía de activación) sin cambiar tu origen ni tu destino.
::/interioriza

::act-order{titulo="Pasos para determinar la ley de velocidad experimental"}
[ ] Mide la velocidad inicial al variar [A] manteniendo [B] constante.
[ ] Repite variando [B] manteniendo [A] constante.
[ ] Calcula los órdenes parciales m y n.
[ ] Determina la constante k.
[ ] Verifica con un experimento adicional.
[ ] Reporta la ley: v = k [A]^m [B]^n.
::/act-order

## Clave de respuestas

**5.1.** 
- a) n_Mg = 24/24 = 1 mol → n_MgO = 1 mol → m = 40.3 g. 
- b) n_C₃H₈ = 22/44.10 = 0.499 mol → n_CO₂ = 1.50 mol → V = 33.5 L. 
- c) n_H₂SO₄ = 0.10 mol → n_NaOH = 0.20 mol → m = 8.0 g. 
- d) Cl total = 0.8 mg/L × 5000 L = 4.0 g; n = 4.0/35.45 = 0.113 mol; m_NaOCl = 0.113×74.44 = 8.4 g; al 5 % → 168 g de solución comercial (≈ 157 mL).

**5.2.** a) n_N₂ = 0.5 mol; n_H₂ = 3.0 mol; relación necesaria 1:3. Tienes exactamente lo justo, pero verifica: 3 mol H₂ alcanza para 1 mol N₂; tienes 0.5 mol N₂ → necesitas 1.5 mol H₂. Tienes 3.0 → **N₂ es limitante**. b) n_NH₃ = 2 × 0.5 = 1.0 mol → m = 17.0 g. c) % rend = 7.5/17.0 × 100 = 44 %. d) 0.5 × M_producto teórico (es decir, 50 % del producto esperado por reactivo limitante).

**5.3 — MCQ.** 1-b · 2-b · 3-c · 4-a.

**5.3 — Cálculos.** a) V = nRT/P = 0.5×0.0821×298/1 = 12.2 L. b) n = PV/RT = 2×5/(0.0821×300) = 0.406 mol. c) n = 2/4 = 0.5 mol; P = nRT/V = 0.5×0.0821×300/10 = 1.23 atm. d) V₂/V₁ = T₂/T₁ = 373/298 = 1.25 ⇒ aumenta 25 %.

**5.5 — Huecos.** 1) entalpía · 2) entropía · 3) energía · 4) menor · 5) espontánea · 6) ley.

**5.5 — V/F.** 1-F (depende también de ΔS y T) · 2-V · 3-F (no afecta H) · 4-V · 5-V.

**5.6 — Orden.** Variar [A] → variar [B] → calcular m, n → determinar k → verificar → reportar.

**5.6 — MCQ.** 1-a · 2-c · 3-c.

**5.7 — Le Chatelier.** N₂↑ → derecha (consume); H₂↓ → izquierda; P↑ → derecha (menos moles gas); T↑ → izquierda (favorece endo); catalizador → ningún desplazamiento (solo acelera ambas direcciones); retirar NH₃ → derecha (repone).
