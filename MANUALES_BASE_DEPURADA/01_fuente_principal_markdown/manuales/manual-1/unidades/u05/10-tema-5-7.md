# 5.7 Equilibrio químico y Principio de Le Chatelier

¿Por qué la cisterna necesita cloro constante si "no se gasta"? 
¿Por qué tu pH sanguíneo no cambia al comer alimentos ácidos?
La respuesta es el **equilibrio químico**.
Muchas reacciones nunca terminan, sino que oscilan infinitamente entre reactivos y productos.

::concepto{titulo="Equilibrio químico"}
El **equilibrio químico** ocurre cuando:
- Las velocidades de reacción directa e inversa son **iguales**.
- Las concentraciones de reactivos y productos se vuelven **constantes**.
- Se representa con doble flecha: $$\ce{A + B <=> C + D}$$

La **constante de equilibrio (Kc)** mide esta proporción:
$$K_c = \frac{[C][D]}{[A][B]}$$
- **Kc >> 1:** Favorece productos.
- **Kc << 1:** Favorece reactivos.
::/concepto

::interioriza{titulo="Analogía: La caminadora del gimnasio"}
Imagina correr en una caminadora a 10 km/h mientras la cinta se mueve hacia atrás a 10 km/h.
Tu posición respecto a la habitación es constante (equilibrio macroscópico).
Pero sigues moviendo las piernas a toda velocidad (dinamismo molecular).
¡Eso es el equilibrio químico!
::/interioriza

::interioriza{titulo="¿Por qué te importa?"}
Controlar el equilibrio químico es dominar la química:
- **En tu cuerpo:** El sistema bicarbonato mantiene tu pH constante.
- **En la industria:** El proceso Haber-Bosch manipula equilibrios para crear amoniaco.
- **En la purificación:** El cloro del bebedero es un equilibrio activo de iones.
::/interioriza

## Constante de equilibrio Kc

Para una reacción genérica $a A + b B \rightleftharpoons c C + d D$, la constante es:
$$K_c = \frac{[C]^c [D]^d}{[A]^a [B]^b}$$

**Reglas de la Kc:**
- Sólidos y líquidos puros **se ignoran** (su concentración no cambia).
- Solo aplica a reactivos/productos en la misma fase (gases o acuosos).
- Cambia **únicamente** con la temperatura.

**Ejemplo 1: Vinagre (ácido acético)**
$$\ce{CH3COOH <=> CH3COO^- + H^+}, \quad K_a = 1.8 \times 10^{-5}$$
- Kc ínfima = La reacción apenas avanza.
- Conclusión: Es un ácido **débil**.

**Ejemplo 2: Ácido Clorhídrico (HCl)**
$$\ce{HCl -> H^+ + Cl^-}, \quad K \approx 10^7$$
- Kc gigante = La reacción se completa hacia la derecha.
- Conclusión: Es un ácido **fuerte**.

## Principio de Le Chatelier

Establecido en 1884, este principio dicta cómo manipular reacciones:
> *"Si un sistema en equilibrio sufre una perturbación, este se desplazará para **contrarrestar** dicho cambio."*

::interioriza{titulo="Analogía: El resorte molesto"}
Imagina un resorte en reposo. 
Si lo empujas (agregas presión), te empuja de vuelta. 
Si lo estiras (quitas presión), intenta encogerse.
Los sistemas químicos actúan igual: siempre llevan la contraria.
::/interioriza

**1. Cambios de concentración:**
- **Agregas reactivos:** El sistema genera productos.
- **Retiras productos:** El sistema genera más productos (para reponer).

**2. Cambios de temperatura:**
- **Reacción exotérmica (libera calor):** Subir la temperatura frena los productos. (El calor actúa como un "producto").
- **Reacción endotérmica:** Subir la temperatura favorece los productos.

**3. Cambios de presión (en gases):**
- **Aumentar presión:** El sistema huye hacia el lado con **menos moles** de gas.
- **Disminuir presión:** El sistema va al lado con **más moles**.

**4. Uso de catalizador:**
- No altera el balance final.
- Solo hace que el equilibrio se alcance **más rápido**.

## Caso de Estudio 1: El Proceso Haber-Bosch

Síntesis de amoniaco para fertilizantes, vital para la humanidad:
$$\ce{N2(g) + 3 H2(g) <=> 2 NH3(g)}, \quad \Delta H = -92\,\text{kJ/mol}$$

**La receta según Le Chatelier:**
- **Temperatura:** Es exotérmica, así que exige **baja T**.
- **Presión:** Pasa de 4 moles a 2 moles, así que exige **alta P**.

**El choque con la realidad:**
- A temperatura muy baja, la reacción es eterna.
- **Solución industrial:** 400 °C (equilibrio vs rapidez) y presión brutal de 200 atmósferas.
- Se añade catalizador de hierro para acelerar todo.

## Caso de Estudio 2: Tu sangre (Sistema Bicarbonato)

Tu cuerpo lucha constantemente por mantener un pH de 7.4.
Lo hace mediante este delicado equilibrio:
$$\ce{CO2 + H2O <=> H2CO3 <=> HCO3^- + H^+}$$

**¿Qué pasa si haces mucho ejercicio (produces ácido)?**
- Sube el nivel de $H^+$.
- El sistema huye a la izquierda para consumirlo.
- Respiras más rápido para expulsar $\ce{CO2}$.

**¿Qué pasa si comes mucha base?**
- Caen los niveles de $H^+$.
- El sistema corre a la derecha para generar más.

## Aplicación al Bebero (Case Study Central)

¿Por qué el cloro residual no desaparece del bebedero hasta que llega una bacteria?

**El equilibrio oculto:**
$$\ce{HOCl <=> OCl^- + H^+}$$
- El HOCl (ácido hipocloroso) es la molécula asesina de bacterias.
- A pH 7, ambas moléculas coexisten (70% HOCl, 30% OCl⁻).

**Cuando ataca la bacteria:**
- El HOCl ataca y se destruye (se retira producto).
- Automáticamente, el OCl⁻ inactivo se convierte en más HOCl.
- ¡El equilibrio "repone" a los soldados caídos!

Por esto, mantener 0.2 - 1.5 mg/L de cloro residual es vital. Es tu ejército de reserva químico.
