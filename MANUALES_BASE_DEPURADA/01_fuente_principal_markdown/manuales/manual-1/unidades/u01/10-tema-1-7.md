# 1.7 Nomenclatura inorgánica básica

- ¿Cómo se llama un compuesto con calcio y oxígeno?
- ¿Es "óxido de calcio", "monóxido de calcio" o "óxido cálcico"?
- La nomenclatura química es el **lenguaje internacional** sin ambigüedades.

::concepto{titulo="Nomenclatura química inorgánica"}
- Es el conjunto de reglas para nombrar compuestos a partir de su fórmula.
- Sirve también para deducir la fórmula a partir del nombre.
- Hay tres sistemas IUPAC oficiales: **sistemática** (prefijos griegos), **Stock** (números romanos) y **tradicional** (sufijos -oso/-ico).
::/concepto

::interioriza{titulo="El idioma de las etiquetas"}
- Imagina que un compañero te pide "ese líquido transparente" en el laboratorio.
- Podría ser agua, pero también ácido clorhídrico letal.
- La nomenclatura es como el código de barras de cada sustancia: una vez que lo lees, sabes exactamente qué contiene el frasco y evitas accidentes.
::/interioriza

## Estados de oxidación — la base del nombre

- El **estado de oxidación (EO)** es la carga "aparente" de un átomo.
- Se asume que todos los enlaces son puramente iónicos.
- Con las reglas básicas, deduces el EO de cualquier elemento.

| Regla | Ejemplo |
|---|---|
| Elementos en estado libre: EO = 0 | Fe(s), O₂, H₂ |
| Iones simples: EO = carga del ion | Na⁺ (+1), Cl⁻ (−1), Ca²⁺ (+2) |
| H en compuestos: +1 (excepto hidruros metálicos: −1) | HCl: H = +1; NaH: H = −1 |
| O en compuestos: −2 (excepto peróxidos: −1; con F: +2) | H₂O: O = −2; H₂O₂: O = −1 |
| Suma de EO en compuesto neutro: 0 | NaCl: +1 + (−1) = 0 |
| Suma de EO en ion poliatómico: carga del ion | SO₄²⁻: S(+6) + 4·O(−2) = −2 |

::interioriza{titulo="La contabilidad de electrones"}
- Piensa en el EO como el saldo bancario de un átomo.
- Un compuesto neutro siempre debe tener sus "cuentas" cuadradas a cero.
- Si un átomo gana electrones, su saldo es negativo; si los pierde, es positivo.
::/interioriza

## Compuestos binarios — dos elementos

### Óxidos básicos (metal + O)

- **Fórmula:** Metal + O
- **Nombre:** óxido de [metal]
- Usa Stock (números romanos) si el metal tiene varios EO.

| Fórmula | Nombre Stock | Nombre tradicional |
|---|---|---|
| Na₂O | óxido de sodio | óxido sódico |
| CaO | óxido de calcio | óxido cálcico |
| FeO | óxido de hierro(II) | óxido ferroso |
| Fe₂O₃ | óxido de hierro(III) | óxido férrico |
| CuO | óxido de cobre(II) | óxido cúprico |

### Óxidos ácidos / anhídridos (no metal + O)

- **Fórmula:** No metal + O
- Usa prefijos di-, tri-, tetra- para el nombre sistemático.

| Fórmula | Nombre sistemático | Nombre tradicional |
|---|---|---|
| CO | monóxido de carbono | óxido carbonoso |
| CO₂ | dióxido de carbono | anhídrido carbónico |
| SO₂ | dióxido de azufre | anhídrido sulfuroso |
| SO₃ | trióxido de azufre | anhídrido sulfúrico |
| N₂O₅ | pentóxido de dinitrógeno | anhídrido nítrico |

### Hidruros

- **Metálicos:** hidruro de [metal].
- Ejemplo: NaH = hidruro de sodio.
- **No metálicos:** nombrados como ácidos al disolverse (ver hidrácidos).

### Sales binarias (metal + no metal)

- **Fórmula:** no metal con sufijo **-uro** + "de" + metal.

| Fórmula | Nombre |
|---|---|
| NaCl | cloruro de sodio |
| KBr | bromuro de potasio |
| CaF₂ | fluoruro de calcio |
| FeS | sulfuro de hierro(II) |
| Al₂S₃ | sulfuro de aluminio |

### Ácidos hidrácidos

- Un hidruro de no metal (grupo 16 o 17) disuelto en agua es un ácido.
- **Nombre:** prefijo **ácido** + raíz + sufijo **-hídrico**.

| Fórmula | Nombre (puro) | Nombre (ácido ac) |
|---|---|---|
| HCl(g) | cloruro de hidrógeno | ácido clorhídrico |
| HBr | bromuro de hidrógeno | ácido bromhídrico |
| H₂S | sulfuro de hidrógeno | ácido sulfhídrico |
| HF | fluoruro de hidrógeno | ácido fluorhídrico |

## Compuestos ternarios — tres elementos

### Hidróxidos / bases (metal + OH)

- **Fórmula:** Metal + (OH)ₙ
- **Nombre:** hidróxido de [metal]

| Fórmula | Nombre |
|---|---|
| NaOH | hidróxido de sodio |
| Ca(OH)₂ | hidróxido de calcio |
| Al(OH)₃ | hidróxido de aluminio |
| Fe(OH)₃ | hidróxido de hierro(III) |

### Oxoácidos (H + no metal + O)

- Se forman al hidratar un anhídrido.
- Se nombran con prefijos/sufijos según el EO del no metal.
- **Hipo- ... -oso** (EO bajo), **-oso** (EO medio-bajo).
- **-ico** (EO medio-alto), **per- ... -ico** (EO máximo).

| Fórmula | Nombre tradicional | EO del no metal |
|---|---|:---:|
| HClO | ácido hipocloroso | +1 |
| HClO₂ | ácido cloroso | +3 |
| HClO₃ | ácido clórico | +5 |
| HClO₄ | ácido perclórico | +7 |
| H₂SO₃ | ácido sulfuroso | +4 |
| H₂SO₄ | ácido sulfúrico | +6 |
| H₂CO₃ | ácido carbónico | +4 |
| HNO₂ | ácido nitroso | +3 |
| HNO₃ | ácido nítrico | +5 |
| H₃PO₄ | ácido fosfórico | +5 |

::interioriza{titulo="La regla del oso y el plato"}
- Una regla mnemotécnica para recordar oxisales:
- "El **oso** toca el p**ito**, el per**ico** toca el pl**ato**".
- Cambia -oso por -ito, y -ico por -ato al formar sales.
::/interioriza

### Oxisales (metal + no metal + O)

- Se forman al neutralizar un oxoácido con un hidróxido.
- Los protones (H) del ácido son reemplazados por cationes metálicos.
- Aplica la mnemotecnia (oso $\rightarrow$ ito, ico $\rightarrow$ ato).

| Fórmula | Nombre |
|---|---|
| NaClO | hipoclorito de sodio (cloro casero) |
| NaClO₃ | clorato de sodio |
| Na₂SO₃ | sulfito de sodio |
| Na₂SO₄ | sulfato de sodio |
| CaCO₃ | carbonato de calcio (mármol, sarro) |
| KNO₃ | nitrato de potasio (salitre) |
| Ca₃(PO₄)₂ | fosfato de calcio (huesos) |

## Aplicación al bebedero

- Ya puedes traducir las etiquetas del laboratorio:
- **HOCl:** ácido hipocloroso (oxoácido, cloro residual).
- **NaClO:** hipoclorito de sodio (oxisal, cloro comercial).
- **CaCO₃:** carbonato de calcio (oxisal, causante de dureza).
