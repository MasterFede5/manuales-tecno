# U6 - Seguridad industrial

## Historia conductora

Antes de aplicar cualquier tratamiento, la escuela debe saber como manejar reactivos, residuos y comunicacion de riesgos. Una solucion quimica sin seguridad puede crear un problema mayor que el original.

## Preguntas introductorias

- Que informacion debe tener una etiqueta quimica?
- Por que un pictograma puede prevenir accidentes?
- Que se hace con un residuo despues de la practica?
- Como se comunica un riesgo sin causar panico?

## Teoria esencial - 40%

- Sistema Globalmente Armonizado y pictogramas.
- Fichas de Datos de Seguridad.
- Equipo de proteccion personal.
- Manejo, almacenamiento y disposicion de residuos.
- Normas mexicanas relacionadas con seguridad y residuos.
- Comunicacion clara de riesgos.

## Especificacion de la actividad - 60%

**Actividad:** Crear un protocolo seguro para el tratamiento del bebedero.

**Objetivo:** Definir pasos, riesgos, controles y evidencias para realizar una practica quimica escolar de forma segura.

**Materiales:** ejemplo de etiqueta, ficha de seguridad, lista de reactivos simulados y formato de protocolo.

**Instrucciones:** identificar riesgos, elegir controles, ordenar pasos y definir disposicion final.

**Producto:** protocolo seguro con checklist previo y posterior.

## Practica guiada

1. Lee una ficha de seguridad simplificada.
2. Identifica peligro principal.
3. Selecciona equipo de proteccion.
4. Define zona de trabajo.
5. Escribe pasos seguros.
6. Indica que hacer con residuos.
7. Redacta aviso breve para la comunidad escolar.

## Practica integradora

El estudiante entrega un protocolo final para diagnosticar y tratar el bebedero escolar. Debe incluir riesgos, controles, responsables, residuos y comunicacion.

## Evidencia y criterios

- Interpreta etiquetas y pictogramas.
- Propone controles antes de actuar.
- Incluye manejo de residuos.
- Comunica riesgos de forma clara y responsable.
