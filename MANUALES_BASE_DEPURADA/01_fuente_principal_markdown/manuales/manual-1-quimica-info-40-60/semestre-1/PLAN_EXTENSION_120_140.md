## Estructura obligatoria de cada unidad

1. Historia conductora conectada con el bebedero escolar.
2. Preguntas introductorias.
3. Analogias para reducir la curva de aprendizaje.
4. Formulas con explicacion de cada elemento.
5. Unidades correctas de cada variable.
6. Ejemplos resueltos antes de ejercicios autonomos.
7. Banco de 10 a 20 ejercicios con datos variados.
8. Especificacion de actividad.
9. Practica guiada.
10. Practica integradora con evidencia.

## Reglas para ejercicios

- Variar datos, unidades y contexto.
- Mezclar calculo, interpretacion, clasificacion y justificacion.
- Incluir espacio para respuesta en HTML imprimible.
- No usar opcion multiple como verificacion principal.
- Cada banco debe poder resolverse con la teoria inmediata de la unidad.
