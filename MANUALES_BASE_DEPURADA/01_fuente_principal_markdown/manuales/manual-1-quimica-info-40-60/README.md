# Manual 1 Quimica Info 40/60

Version de trabajo solo informativa para construir el manual de Quimica uno por uno.

## Regla central

Cada unidad mantiene esta distribucion:

- 40% teoria: historia conductora, preguntas introductorias, conceptos, explicacion breve y criterios de comprension.
- 60% practica: actividad guiada, practica aplicada, evidencia, producto y rubrica breve.

## Estructura fija por unidad

1. Historia conductora.
2. Preguntas introductorias.
3. Teoria esencial.
4. Especificacion de la actividad.
5. Practica guiada.
6. Practica integradora.
7. Evidencia y criterios de logro.

No incluir imagenes, HTML, CSS ni codigo obligatorio en esta version.
