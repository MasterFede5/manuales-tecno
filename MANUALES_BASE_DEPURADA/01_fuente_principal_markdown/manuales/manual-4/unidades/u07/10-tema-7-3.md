# 7.3 Open WebUI (interfaz local tipo ChatGPT)

Tu Ollama corre perfectamente y la línea de comandos responde.
Sin embargo, tu coordinadora académica no abrirá una terminal jamás.
Necesita la experiencia de ChatGPT, pero corriendo localmente.

::concepto{titulo="Open WebUI"}
**Open WebUI** es una interfaz web open source que se conecta a Ollama.
Ofrece una experiencia idéntica a ChatGPT directamente en localhost.
Características principales:

- **UI tipo ChatGPT:** Conversaciones, historial y edición.
- **Multi-modelo:** Alterna entre Llama, Mistral y Qwen en una sesión.
- **Multi-usuario:** Autenticación, roles y permisos.
- **RAG sin código:** Base de conocimientos integrada.
- **Herramientas:** Funciones personalizadas (Functions).
- **Multi-modalidad:** Soporte nativo para imágenes y Whisper para audio.
- **Despliegue rápido:** Self-hosted vía Docker en minutos.
::/concepto

::interioriza{titulo="La Carrocería del Motor"}
Imagina que Ollama es un potente motor V8 de última generación.
Por sí solo, el motor funciona, pero nadie puede conducirlo cómodamente sentado en él.
Open WebUI es la carrocería, el volante y el tablero digital de ese coche.
Es lo que permite a todo el colegio "conducir" la IA sin mancharse de grasa.
::/interioriza

## Instalación con Docker

Asumiendo que Ollama ya corre:

```bash
docker run -d \
  -p 3000:8080 \
  --add-host=host.docker.internal:host-gateway \
  -v open-webui:/app/backend/data \
  --name open-webui \
  --restart always \
  ghcr.io/open-webui/open-webui:main
```

Abres `http://localhost:3000`. Crear cuenta admin. Detecta automáticamente Ollama en `host.docker.internal:11434`.

## Multi-usuario y base de conocimientos (RAG)

Open WebUI no es solo para ti, está pensado para toda la institución:

- **Roles:** Administradores (configuran) y Usuarios (solo chatean).
- **Autenticación:** Soporta SSO (Google, GitHub) o usuarios locales.
- **Knowledge Base (RAG):** Sube PDFs institucionales al "Workspace".
- Open WebUI procesa y vectoriza los textos automáticamente.
- Menciona tu base con `#nombre_kb` en el chat para obtener respuestas basadas en tus datos.

## Costos reales y Anti-patrones

El costo de implementación es bajo comparado con opciones comerciales:

- **Hardware:** $1500-3000 USD por un servidor con buena GPU.
- **Software:** $0 USD, tanto Ollama como Open WebUI son gratuitos.
- **ROI:** Se recupera en meses frente a licencias de ChatGPT Enterprise.

Pero ten cuidado con los siguientes errores comunes (anti-patrones):

- **Sin backups:** Perderás todo el historial si el contenedor falla.
- **Sin HTTPS:** Nunca expongas contraseñas en texto plano por la red.
- **Sin filtros:** Modelos abiertos pueden dar respuestas no deseadas a estudiantes.
