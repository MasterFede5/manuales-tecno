# 8.4 ChatGPT con conectores empresariales

OpenAI llegó tarde a MCP pero llegó con fuerza.
Mientras lo abraza progresivamente, ChatGPT (Team/Enterprise) ofrece **Connectors** propios.
Estos conectores cumplen una función similar para grandes ecosistemas empresariales.

::concepto{titulo="ChatGPT Connectors y soporte MCP"}
**Connectors** son integraciones nativas de ChatGPT para acceder a servicios corporativos.
- Cuentan con autenticación OAuth y UI guiada.
- No requieren configurar archivos JSON.
- Son *first-party* de OpenAI, preconfigurados y listos para usar.

En 2025, OpenAI añadió soporte MCP en Tools.
- Es más limitado que Claude, pero se alinea al estándar.
- Soporta servidores específicos, aunque el ecosistema es menos maduro.
::/concepto

::interioriza{titulo="¿Por qué te importa? La navaja suiza vs. El cuchillo de chef"}
Imagina que tu equipo ya usa cuchillos de chef profesionales (Google Workspace, M365). 
Los Connectors de ChatGPT son como afilar esos cuchillos: se integran sin esfuerzo a tu rutina actual.
Forzar a usar una navaja suiza nueva (Claude Desktop + MCP) requiere que todos aprendan a usarla.
Si ya pagas ChatGPT Team/Enterprise, aprovecha los Connectors para evitar fricción.
::/interioriza

## Connectors disponibles (2025)

| Connector | Auth | Caso típico |
|---|---|---|
| **Google Drive / Calendar** | OAuth Google | Leer docs, revisar eventos |
| **Microsoft 365** | OAuth Microsoft | Ecosistema MS (OneDrive, Outlook) |
| **Box / Dropbox** | OAuth | Almacenamiento de archivos |
| **Salesforce / GitHub** | OAuth | CRM y Repositorios |
| **Notion / Slack** | OAuth | Espacios y mensajes |

- Cada conector se activa en Settings → Connected Apps.
- Solo requiere login corporativo.

## Diferencias prácticas Claude Desktop vs ChatGPT

| Dimensión | Claude Desktop + MCP | ChatGPT + Connectors |
|---|---|---|
| **Filosofía** | Abierto, estándar MCP | Empaquetado, OAuth |
| **Setup** | Editar config JSON | UI guiada |
| **Ecosistema**| Total (comunidad) | Limitado, *first-party* |
| **Privacidad**| Puede ser 100% local | Siempre cloud OpenAI |

## Cuándo elegir cada uno

**Elige Connectors de ChatGPT si:**
- Tu institución está 100% comprometida con OpenAI.
- Trabajas con apps mainstream (Google, M365, Salesforce).
- Tu equipo no es técnico y necesita interfaces guiadas.

**Elige Claude Desktop + MCP si:**
- Requieres servidores custom (sistemas internos, BD).
- Buscas flexibilidad y apertura.
- Priorizas privacidad local.

## Camino híbrido y Anti-patrones

Las instituciones serias **usan ambos** clientes:
- ChatGPT para tareas administrativas estándar.
- Claude Desktop para conectarse a sistemas internos customizados.

**Anti-patrones a evitar:**
- Cancelar tus útiles Connectors solo por el "hype" de MCP.
- Perder la flexibilidad de Claude por la simpleza de ChatGPT.
- Configurar los mismos tokens en ambos clientes (riesgo de seguridad).
