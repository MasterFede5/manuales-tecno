# 3.3 ChatGPT Canvas y Gemini Canvas

Tu equipo usa Claude para algunos módulos y ChatGPT para otros. 
Tu directora prefiere Gemini porque ya paga Google Workspace. 
Si solo conoces Claude Artifacts, tendrás equipos hablando lenguajes distintos.
Este subtema te equipa para entender y dominar los tres entornos.

::concepto{titulo="ChatGPT Canvas y Gemini Canvas"}
**ChatGPT Canvas** (OpenAI) y **Gemini Canvas** (Google) son las alternativas a Claude Artifacts.
Ambos abren un panel separado del chat para alojar contenido editable.
Sus principales diferencias radican en tipos soportados, ejecución de código y modelo de publicación.

A finales de 2025, los tres convergen en concepto pero divergen en detalles de uso. 
La elección final depende de **dónde ya vive tu equipo** y qué ecosistema dominan.
::/concepto

::interioriza{titulo="¿Por qué te importa? La analogía de la oficina"}
Imagina que eliges una oficina para tu equipo. No buscas la que tenga la mejor cafetera, sino **la que le quede más cerca a la mayoría**. 
Aquí es igual: si tu institución ya paga ChatGPT Enterprise, invierte en su Canvas. 
Si usan Google Workspace, ve por Gemini Canvas. No cambies de plataforma por funciones menores.
::/interioriza

## ChatGPT Canvas en resumen

**Acceso:** Planes Plus, Team o Enterprise (Web y app desktop).

**Tipos soportados:**
- Texto largo (su fortaleza original).
- Código (sandbox para ejecutar Python).
- HTML/CSS/JS estático.
- React (soporte limitado pero en expansión).

**Diferenciadores principales:**
- **Edición tipo Word:** Selecciona texto y cambia solo esa parte.
- **Code interpreter:** Ejecuta Python y muestra gráficas en el Canvas.
- **Sugerencias proactivas:** Botones rápidos como "más formal" o "añadir emoji".
- **Compartir fácil:** Genera links públicos con un clic.

**Limitaciones:**
- Soporte para React menos pulido que Claude.
- Historial de versiones menos navegable y algo opaco.
- Sin soporte nativo interactivo para diagramas Mermaid.

**¿Cuándo brilla?**
- Documentos largos colaborativos (ensayos, PRDs).
- Análisis de datos usando Python.
- Equipos que ya están anclados al ecosistema OpenAI.

## Gemini Canvas en resumen

**Acceso:** Gemini Advanced o Workspace (Business/Enterprise).

**Tipos soportados:**
- Texto largo y código.
- HTML/CSS estático con previsualización.
- Diagramas apoyados en Google Charts o D3.

**Diferenciadores principales:**
- **Google Workspace:** Exporta directo a Docs, Slides o Sheets.
- **Multilingüe fluido:** Excelente para español académico.
- **Multimodal:** Acepta imágenes o PDFs en el contexto del Canvas.
- **Deep Research:** Crea reportes largos directamente allí.

**Limitaciones:**
- Poca potencia en React o JS interactivo.
- Compartir fuera de la organización es menos directo.
- Documentación técnica inferior a la de Anthropic.

**¿Cuándo brilla?**
- Organizaciones que viven 100% en Google Workspace.
- Reportes largos de investigación con múltiples citas.
- Contextos educativos con mucho contenido multimodal.

## Comparativo rápido para decidir

| Dimensión | Claude | ChatGPT | Gemini |
|---|---|---|---|
| **Texto largo** | Bueno | Excelente | Bueno |
| **React / JS** | Excelente | Bueno | Medio |
| **Código + Ejecución** | Medio | Excelente (Python) | Bueno |
| **Mermaid / Diagramas** | Excelente | Medio | Bueno |
| **Edición selectiva** | Bueno | Excelente | Bueno |
| **Integración Google** | Nula | Nula | Excelente |

## Patrón "multi-canvas" estratégico

Los equipos profesionales **no eligen uno solo**, combinan según la tarea:

- **Claude Artifacts:** Para dashboards React con sliders o diagramas Mermaid.
- **ChatGPT Canvas:** Para código Python con Code Interpreter o edición de texto selectiva.
- **Gemini Canvas:** Para investigación (Deep Research) o exportar directo a Docs.

## Adaptando tu primer prompt

El primer prompt debe cambiar según el proveedor elegido:
- **Claude:** Usa etiquetas XML y descripciones muy precisas.
- **ChatGPT:** Prefiere tono conversacional y referencias de estilo.
- **Gemini:** Destaca cuando le das contexto multimodal (PDFs, imágenes).

## Anti-patrones que debes evitar
- **Claude para Python intenso:** No tiene intérprete; solo escupe texto.
- **ChatGPT para Mermaid:** Renderizado menos pulido y estático.
- **Gemini para React:** Demasiado inestable en interactividad pura.
- **Saltar de proveedor:** Pierdes el historial y cuesta migrar contexto.
