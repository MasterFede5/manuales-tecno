Variables, tipos y operadores

Una variable es una etiqueta para guardar un valor. En el predictor escolar, una variable puede guardar horas de estudio, porcentaje de asistencia o calificacion previa.

horas_estudio = 6.5
asistencia = 0.92
aprobo = True

Los tipos importan porque no se suma igual un numero que un texto. Un error comun es leer "8" como texto y querer operarlo como numero.

Control de flujo

Un programa necesita decidir. Si una calificacion es mayor o igual a 6, el estudiante aprueba; si no, requiere apoyo. Esta idea se expresa con condicionales.

if calificacion >= 6:
  estado = "aprobado"
else:
  estado = "requiere apoyo"

Funciones

Una funcion guarda una tarea reutilizable. En vez de repetir la misma condicion para cada estudiante, creamos una funcion que reciba calificacion y devuelva estado.

def clasificar(calificacion):
  if calificacion >= 6:
    return "aprobado"
  return "requiere apoyo"

Listas, diccionarios y archivos

Una lista guarda varios valores. Un diccionario guarda valores con nombre. Para datos escolares, un diccionario puede representar un estudiante y una lista puede reunir muchos estudiantes.

estudiante = {"nombre": "A01", "horas": 6.5, "calificacion": 8.2}

Errores comunes

Programar tambien implica manejar errores. Un archivo puede no existir, un valor puede estar vacio o una calificacion puede venir como texto. El objetivo no es evitar todo error, sino detectarlo y responder con claridad.

try:
  calificacion = float(valor)
except ValueError:
  calificacion = None

Ejemplo resuelto

El siguiente fragmento clasifica tres calificaciones sin usar librerias externas. Es pequeno porque la unidad busca control basico antes de Pandas.

calificaciones = [8.5, 5.9, 7.2]
for nota in calificaciones:
  print(clasificar(nota))

El bucle recorre una nota a la vez. La funcion decide si cada estudiante esta aprobado o requiere apoyo. Esta salida sera la base para analizar datos mas grandes.
