# 1.5 Listas, tuplas, diccionarios

Una variable guarda **un solo valor**.
Para guardar **muchos** (480 calificaciones, perfiles de alumnos), Python ofrece 4 estructuras clave.

::concepto{titulo="Estructuras de datos básicas"}
Las **estructuras de datos** son contenedores para múltiples valores. Tienen reglas distintas:

- **Lista (`list`)**: Secuencia ordenada y **mutable** (puedes modificarla).
- **Tupla (`tuple`)**: Secuencia ordenada e **inmutable** (fija).
- **Diccionario (`dict`)**: Pares clave-valor. Mantiene orden de inserción (Python 3.7+).
- **Conjunto (`set`)**: Elementos únicos sin orden.

Cada una tiene su caso ideal. Aprenderlas te permite estructurar cualquier dato.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Piensa en el trabajo escolar diario:
- La columna de calificaciones de un grupo es una **lista** (el orden importa, hay cambios).
- Una pareja de "matrícula-calificación final" es una **tupla** (es fija, no cambia).
- Un alumno con sus datos (nombre, edad, grupo) es un **diccionario**.
- Las matrículas únicas de todos los alumnos forman un **set**.
::/interioriza

## Lista (la más común)

- Úsala para secuencias que pueden cambiar.
- Es la estructura que más utilizarás en el día a día.

```python
# Crear y acceder
calificaciones = [8.5, 6.0, 9.5]
print(calificaciones[0])   # 8.5 (índice 0)

# Modificar
calificaciones.append(8.0)        # agrega al final
calificaciones.insert(0, 10.0)    # agrega al inicio
```

## Tupla (cuando NO quieres cambios)

- Úsala para datos que deben protegerse de modificaciones (coordenadas, registros fijos).

```python
# Crear y acceder
coord_alumno = (124, 8.5)
id_a, cal_a = coord_alumno

# Inmutable: coord_alumno[0] = 200 da ERROR
```

## Diccionario (clave-valor)

- Úsalo para representar entidades con atributos (como una fila de un CSV).

```python
# Crear y acceder
alumno = {"nombre": "Andrea", "promedio": 8.33}
print(alumno["nombre"])  # "Andrea"

# Acceso seguro sin error
print(alumno.get("edad", 18))  # 18 si no existe
```

## Set (elementos únicos)

- Úsalo para eliminar duplicados o comparar conjuntos (intersección, diferencia).

```python
inscritas = {"Mate", "Física", "Química", "Mate"} 
# Duplicado se elimina automáticamente
print(inscritas)  # {"Mate", "Física", "Química"}
```

## Anidación y List Comprehension

- Puedes anidar estructuras: una **lista de diccionarios** es ideal para representar tablas de datos.
- **List comprehension** te permite crear listas filtradas en una sola línea.

```python
# Lista de diccionarios (tabla)
estudiantes = [{"nombre": "Andrea", "promedio": 8.5}, {"nombre": "Carlos", "promedio": 7.0}]

# List comprehension: extrae aprobados
aprobados = [e["nombre"] for e in estudiantes if e["promedio"] >= 6]
```

## Anti-patrones comunes

- **Usar lista para búsquedas rápidas:** Si buscas elementos únicos frecuentemente, usa set.
- **Mutar dicts al iterar:** Itera sobre `list(d.items())` si vas a modificar el diccionario.
- **Acceso ciego a dicts:** Usar `d["clave"]` sin verificar genera `KeyError`. Usa `.get()`.
