# 1.2 Variables, tipos, operadores

- Una calificación es 8.5. Un nombre es "Andrea".
- Tu primer paso en Python es **darle nombres a estas cosas**.
- Así, tu programa puede usarlas. Esto se llama variables.

::concepto{titulo="Variable, tipo, operador"}
- Una **variable** asocia un nombre a un valor. En Python:

```python
calificacion = 8.5
nombre = "Andrea"
aprobado = True
```

- Un **tipo** es la categoría: entero (`int`), decimal (`float`), texto (`str`), booleano (`bool`).
- Python detecta el tipo automáticamente.

- Un **operador** manipula valores: aritméticos (`+`, `-`), comparación (`==`, `<`), lógicos (`and`, `or`).
::/concepto

::interioriza{titulo="El archivero escolar"}
- Imagina una variable como una carpeta en un archivero.
- La etiqueta de la carpeta es el **nombre** (ej. `calificacion`).
- Los documentos dentro son el **valor** (ej. `8.5`).
- El **tipo** dictamina qué puedes guardar en esa carpeta.
::/interioriza

## Los 4 tipos básicos

```python
# Entero (int) — sin decimales
edad = 17
horas_estudio = 6

# Decimal (float) — con decimales
calificacion = 8.5
porcentaje_asistencia = 92.3

# Texto (str) — entre comillas
nombre = "Andrea"
materia = 'Matemáticas'  # comillas simples también valen

# Verdadero/Falso (bool)
aprobado = True
faltó_examen = False
```

- Para verificar el tipo, usa: `type(calificacion)`.
- Esto devuelve: `<class 'float'>`.

## Operadores aritméticos

- **Básicos**: `+`, `-`, `*`, `/`
- **Especiales**: `//` (división entera), `%` (módulo), `**` (potencia)

```python
# Suma, resta, multiplicación, división
total = 8.5 + 9.0  # 17.5
diferencia = 9.0 - 7.5  # 1.5
producto = 4 * 25  # 100
promedio = (8.5 + 9.0 + 7.5) / 3  # 8.333...

# División entera y módulo
cociente = 17 // 3  # 5 (sin decimales)
resto = 17 % 3  # 2 (sobrante)

# Potencia
cuadrado = 5 ** 2  # 25
```

## Operadores de comparación

- Evalúan condiciones y devuelven `True` o `False`.
- Usados para comparar calificaciones o asistencias.

```python
calificacion = 7.5

calificacion >= 6     # True (aprueba)
calificacion == 10    # False (no es 10)
calificacion != 0     # True (no es cero)
```

## Operadores lógicos

- Combinan múltiples condiciones.
- Útiles para reglas de negocio escolares (ej. asistencias y notas).

```python
asistencia = 0.85
calificacion = 7.0

aprueba = (asistencia >= 0.80) and (calificacion >= 6.0)  # True
en_riesgo = (asistencia < 0.70) or (calificacion < 6.0)   # False

reprobado = not aprueba  # False
```

## Cast (conversión entre tipos)

- A veces un dato llega como texto y necesitas un número.
- Usa funciones como `int()` o `float()` para forzar (castear) el tipo.

```python
edad_str = "17"
edad_num = int(edad_str)  # ahora es entero 17

cal_str = "8.5"
cal_num = float(cal_str)  # ahora es float 8.5

cal_num_str = str(cal_num)  # convierte de float a "8.5"
```

## Reglas de nombres de variables

- **Solo letras, números y guion bajo**: `cal_final`, `_temp`.
- **Prohibido**: Empezar con número o usar espacios.
- **Distinción**: Python diferencia mayúsculas de minúsculas.
- **Convención**: Usa `snake_case` (ej. `mi_variable`).






## Anti-patrones

- **Nombres opacos**: Evita variables como `x = 8.5`. Usa `calificacion`.
- **Nombres en inglés**: Si tu contexto escolar es en español, úsalo.
- **Mutar tipos**: No cambies una variable de `int` a `str` a medio camino.
- **Comparar mal**: Usa `==` para verificar si dos valores son iguales.
