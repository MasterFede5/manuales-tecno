# 1.3 Estructuras de control: if, for, while

- Tu programa necesita **decidir** ("si calificación >= 6, aprueba").
- También necesita **repetir** ("revisar 480 estudiantes").
- Sin estas estructuras, Python ejecutaría línea por línea de forma plana y sin lógica.

::concepto{titulo="Estructuras de control"}
Las **estructuras de control** dirigen el flujo de ejecución. Son tres:
- **Condicionales (`if`/`elif`/`else`)**: Ejecutan código solo si se cumple una condición.
- **Bucles `for`**: Repiten código por cada elemento de una secuencia.
- **Bucles `while`**: Repiten código mientras una condición sea verdadera.

**Nota clave:** En Python, los bloques se marcan con **indentación** (4 espacios o 1 tab), no con llaves `{}`.
::/concepto

::interioriza{titulo="Analogía: El guardia de la escuela"}
Imagina que eres el guardia en la entrada de la escuela:
- **El condicional (`if/else`)**: Revisas la credencial. *Si* es válida, pasas; *si no*, la rechazas.
- **El bucle (`for/while`)**: No haces esto una sola vez. *Repites* la revisión para los 480 estudiantes en la fila automáticamente.
::/interioriza

## Condicional `if`/`elif`/`else`

```python
calificacion = 7.5

if calificacion >= 9:
    print("Excelente")
elif calificacion >= 8:
    print("Muy bien")
elif calificacion >= 6:
    print("Aprobó")
else:
    print("Reprobó")
```

**Reglas de oro:**
- `if` es obligatorio e inicia el bloque.
- `elif` es opcional (puedes usar varios para múltiples condiciones).
- `else` es opcional (máximo uno, captura todo lo que no cumplió antes).
- La indentación de 4 espacios define qué hay dentro del bloque.

## Bucle `for`

Ideal para recorrer colecciones elemento por elemento:

```python
calificaciones = [8.5, 6.0, 9.5, 5.5, 7.8]

for cal in calificaciones:
    if cal >= 6:
        print(cal, "→ aprueba")
    else:
        print(cal, "→ reprueba")
```

- Iteración con índices usando `range()`:
```python
for i in range(5):
    print("Iteración", i)  # Imprime del 0 al 4
```

## Bucle `while`

Ideal cuando **no sabes** el número exacto de iteraciones necesarias:

```python
contador = 0
aprobados = 0
calificaciones = [8.5, 6.0, 9.5, 5.5, 7.8]

while contador < len(calificaciones):
    if calificaciones[contador] >= 6:
        aprobados += 1
    contador += 1

print(f"Aprobaron: {aprobados} de {len(calificaciones)}")
```

## Patrones útiles

- **Contar elementos que cumplen condición:**
```python
aprobados = 0
for cal in calificaciones:
    if cal >= 6: 
        aprobados += 1
```

- **Sumar y acumular:**
```python
suma = 0
for cal in calificaciones:
    suma += cal
```

- **Control de bucles (`break` y `continue`):**
  - `continue`: Salta la iteración actual y pasa a la siguiente.
  - `break`: Rompe y termina el bucle entero inmediatamente.

## Anti-patrones comunes

- **Olvidar la indentación:** Python lanzará un `IndentationError`. Apóyate en tu IDE.
- **Confundir asignación con comparación:** Usar `if cal = 6` (asignación) en lugar de `if cal == 6` (comparación).
- **Modificar una lista mientras la iteras:** Crea bugs. Si necesitas hacerlo, copia la lista primero.
- **Bucles infinitos:** Usar un `while` sin una condición que cambie y eventualmente lo detenga.

## Caso aplicado: clasificar 480 estudiantes

```python
calificaciones = [8.5, 6.0, 9.5, 5.5, 7.8, 4.2, 9.0]
categorias = {"excelente": 0, "muy_bien": 0, "aprobo": 0, "reprobo": 0}

for cal in calificaciones:
    if cal >= 9:
        categorias["excelente"] += 1
    elif cal >= 8:
        categorias["muy_bien"] += 1
    elif cal >= 6:
        categorias["aprobo"] += 1
    else:
        categorias["reprobo"] += 1

print(categorias)
# Output: {'excelente': 2, 'muy_bien': 1, 'aprobo': 2, 'reprobo': 2}
```
