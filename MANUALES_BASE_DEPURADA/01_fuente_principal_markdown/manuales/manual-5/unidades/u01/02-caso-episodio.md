::caso{episodio=1}
**Caso Albatros — Episodio 1: Cargar la lista de calificaciones desde un CSV.**

La coordinación académica de Albatros te entregó un archivo de Excel con calificaciones, asistencia y horas de estudio de **480 estudiantes** de bachillerato del año pasado. Los datos están **anonimizados** (sin nombres, solo IDs). Te dijo: *"Vamos a construir, paso a paso, un predictor de rendimiento. Empezamos por algo simple: que tu computadora **lea ese archivo** y te diga cuántas filas tiene."* No es trivial: nunca has programado.

El Episodio 1 te lleva de cero programación a un script Python funcional que carga el CSV, cuenta filas, y muestra la primera línea. Cubrirás configuración (Anaconda, VS Code o Colab — elige uno), variables y tipos, control de flujo, funciones, estructuras de datos (listas, dicts), lectura/escritura de archivos y manejo básico de errores. La meta: terminar la unidad con tu propio script Python ejecutado correctamente sobre el dataset escolar.

> *La pregunta gancho:* ¿qué herramienta abres primero? ¿Anaconda en tu computadora, VS Code para programadores, o Google Colab desde el navegador? La elección depende de tu contexto. Empezamos por ahí.
::/caso
