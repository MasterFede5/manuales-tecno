# 1.1 Configuración: Anaconda, VS Code, Colab

Antes de escribir código Python, necesitas elegir tu espacio de trabajo. 
Existen tres opciones principales y razonables para comenzar. 
Esta lección te ayudará a decidir y configurar tu entorno rápidamente.

::concepto{titulo="Entorno de desarrollo Python"}
Un **entorno de desarrollo** es el conjunto de herramientas donde escribes y ejecutas tu código (intérprete, editor, librerías).
Para los principiantes en IA, existen **3 caminos principales**:

- **Anaconda + Jupyter Notebook:** Instalación todo-en-uno con librerías de datos preinstaladas.
- **VS Code + Python:** Editor profesional y flexible para programar más allá de la IA.
- **Google Colab:** Entorno en la nube gratuito, sin instalación y con GPU disponible.

**Recomendación:** Cualquiera funciona. Usa Colab para empezar, Anaconda para instalación local, o VS Code si eres avanzado.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Imagina un taller mecánico. Si no tienes las herramientas correctas y ordenadas, reparar el motor será un caos.
Tu entorno de desarrollo es tu taller virtual. 
Si lo eliges bien, te enfocarás en crear IA; si lo eliges mal, perderás horas configurando.
::/interioriza

## Camino 1 — Google Colab (recomendado)

- **Acceso:** colab.research.google.com (requiere cuenta Google).
- **Setup:** Ve al sitio y haz click en "Nuevo notebook".
- **Ventaja:** Listo al instante. Incluye librerías y GPU gratis.
- **Ideal para:** Equipos de bajos recursos y trabajo colaborativo (como un Google Doc).
- **Limitaciones:** Depende de internet y las sesiones expiran tras 12h de inactividad.

## Camino 2 — Anaconda Local

- **Descarga:** anaconda.com/download (sigue el instalador estándar).
- **Setup:** Abre "Anaconda Navigator" y lanza "Jupyter Notebook".
- **Ventaja:** Control total y funcionamiento 100% offline.
- **Ideal para:** Quienes desean un entorno local robusto y exclusivo para datos.
- **Limitaciones:** Ocupa unos 5 GB de disco y requiere mínimo 8 GB de RAM.

## Camino 3 — VS Code

- **Descarga:** code.visualstudio.com y Python de python.org.
- **Setup:** Instala las extensiones "Python" y "Jupyter" en el editor.
- **Ventaja:** Herramienta profesional, altamente extensible y con Git integrado.
- **Ideal para:** Programadores con experiencia y proyectos que no son solo IA.
- **Limitaciones:** Curva de aprendizaje empinada; requiere configurar entornos virtuales.

## Verificación: tu primer "Hola, IA"

- Abre un notebook (`.ipynb`) en tu entorno elegido.
- Escribe el siguiente código en una celda:

```python
print("¡Hola, mundo de IA!")
print("2 + 2 =", 2 + 2)
```

- Presiona `Shift+Enter` para ejecutar. 
- Si aparece el texto y el número `4`, **¡tu entorno funciona perfectamente!**

## Librerías Principales

- En Colab y Anaconda, las librerías base ya vienen preinstaladas.
- En VS Code, deberás instalarlas manualmente en la terminal:

```bash
pip install numpy pandas matplotlib seaborn scikit-learn
```

- Estas cinco herramientas cubrirán casi todo el manual.
- Las herramientas avanzadas (PyTorch, APIs) se instalarán más adelante.

## Anti-patrones comunes

- **Reinventar la rueda:** No instales todo manualmente si Anaconda ya lo incluye.
- **Cruzar instalaciones:** Mezclar Anaconda con Python puro generará conflictos molestos.
- **Empezar en modo experto:** Iniciar con VS Code sin experiencia te abrumará rápido.
- **Volar a ciegas:** Nunca te saltes la verificación inicial; probar te ahorrará dolores de cabeza.
