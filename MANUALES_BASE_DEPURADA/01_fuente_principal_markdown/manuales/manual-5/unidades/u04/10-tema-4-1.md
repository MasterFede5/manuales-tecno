# 4.1 Media, mediana, varianza, desviación estándar

- Estos 4 números son los pilares para resumir cualquier variable.
- Su reporte correcto es la base de toda la estadística descriptiva.
- Nunca debes presentar uno sin el otro (centro vs. dispersión).

::concepto{titulo="Estadísticos descriptivos básicos"}
- **Media** (μ): Suma dividida entre n. El clásico "promedio".
- **Mediana**: El valor que queda exactamente al centro cuando ordenas los datos. Deja 50% arriba y 50% abajo.
- **Varianza** (σ²): Promedio de las desviaciones al cuadrado. Mide matemáticamente la dispersión.
- **Desviación estándar** (σ): Raíz cuadrada de la varianza. Muy útil porque está en las **mismas unidades** que los datos originales.

```python
df["cal_final"].mean()      # 7.42
df["cal_final"].median()    # 7.50
df["cal_final"].std()       # 1.49
df["cal_final"].var()       # 2.22
```
::/concepto

::interioriza{titulo="¿Por qué te importa? La historia completa"}
- Imagina dos profesores con un promedio de clase de 7.
- En la clase A, todos los alumnos sacaron 7 (un grupo súper homogéneo).
- En la clase B, la mitad sacó 4 y la otra mitad 10 (un grupo polarizado).
- El promedio oculta esta realidad. La desviación estándar revela el "resto de la historia".
::/interioriza

## Media vs mediana

```python
notas = [6, 7, 7, 8, 9]
mean = 7.4
median = 7

# Con outlier
notas_outlier = [6, 7, 7, 8, 99]
mean_o = 25.4   # ¡distorsionada por un error!
median_o = 7    # robusta y fiel a la mayoría
```

- **Regla de oro:** Si tus datos tienen valores extremos (outliers) o son asimétricos, **reporta la mediana**.
- Si tienen una distribución simétrica, la media es la opción ideal.

## Cuándo usar cada medida

| Caso | Medida ideal |
|---|---|
| Calificaciones simétricas | Media |
| Salarios (asimétrico fuerte) | Mediana |
| Edades en una escuela | Media |
| Tiempos de respuesta (servidor) | Mediana |
| Conteo de errores de software | Mediana o moda |

## Desviación estándar: ¿qué significa?

- Si tienes una media de 7 y una desviación estándar (std) de 1...
- ...significa que la mayoría de los datos (≈68 % en una distribución normal) cae entre 6 y 8.
- A mayor desviación estándar, más "ancho" es el rango de valores.

```python
notas = pd.Series([7, 7, 7, 7, 7])
notas.std()    # 0 (todos iguales, dispersión nula)

notas = pd.Series([5, 6, 7, 8, 9])
notas.std()    # 1.58 (dispersos, pero simétricos)
```

## Coeficiente de variación (CV)

- Es simplemente: `CV = std / mean × 100 %`
- Se usa para comparar la dispersión entre dos grupos que tienen **medias distintas**.
- Expresa la desviación estándar como un porcentaje de la media.

```python
cv_mate = df[df["materia"]=="Mate"]["cal_final"].std() / df[df["materia"]=="Mate"]["cal_final"].mean() * 100
# Ej: 22% — grupo con notas muy dispersas
```

## Reportar correctamente

- ❌ "El promedio fue 7.42." (Oculta la dispersión y el tamaño de muestra).
- ✅ "Promedio 7.42 ± 1.49 (std), mediana 7.50, n=470 estudiantes." (Transparente y completo).

## Anti-patrones que debes evitar

- **Reportar solo la media:** Siempre acompáñala de la desviación estándar.
- **Usar media para datos asimétricos:** Opta por la mediana, evitarás distorsiones.
- **Usar mediana en datos simétricos sin razón:** La media aprovecha todos los valores (es más precisa ahí).
- **No verificar n (tamaño de muestra):** Una media de 5 datos no tiene el peso de una de 500.
