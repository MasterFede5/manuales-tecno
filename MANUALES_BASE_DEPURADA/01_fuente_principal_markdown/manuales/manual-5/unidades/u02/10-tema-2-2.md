# 2.2 Pandas: Series y DataFrame

- Pandas es **la** librería esencial para manejar datos tabulares en Python.
- Si apuntas a ser *data scientist*, ocupará el 50 % de tu día a día.
- Permite procesar masivos volúmenes de datos con la familiaridad de una tabla.

::concepto{titulo="Pandas: Series y DataFrame"}
**Pandas** es la librería estándar para datos estructurados (tablas) en Python. Construida sobre NumPy, expone dos estructuras principales:

- **Series**: array 1D con **índice** (similar a columna de Excel con etiquetas de fila).
- **DataFrame**: tabla 2D con filas (índice) y columnas (cada una una Series).

```python
import pandas as pd

# Series
calificaciones = pd.Series([8.5, 6.0, 9.5], index=["Andrea", "Carlos", "María"])

# DataFrame
df = pd.DataFrame({
    "nombre": ["Andrea", "Carlos", "María"],
    "calificacion": [8.5, 6.0, 9.5],
    "horas": [5, 4, 6]
})
```

- Es **el lenguaje común** entre Excel, SQL y Python.
- Si dominas pandas, dominas la transición de hojas de cálculo a programación.
::/concepto

::interioriza{titulo="Analogía: El libro del profesor"}
Imagina que una **Series** es la lista de calificaciones de un examen pegada en el corcho del salón. Un **DataFrame** es el libro completo del profesor, donde cada columna (cada examen) es una Series, alineadas por el nombre del alumno (el índice). Pandas es tu asistente robótico que calcula promedios en milisegundos.
::/interioriza

## Crear DataFrames

- Puedes inicializar DataFrames de múltiples formas según tus datos de origen.
- Desde un diccionario de listas, lista de diccionarios, o arrays de NumPy.

```python
import pandas as pd

# Desde dict
df = pd.DataFrame({
    "id": [124, 125, 126],
    "nombre": ["Andrea", "Carlos", "María"],
    "calificacion": [8.5, 6.0, 9.5]
})

# Desde lista de dicts
df = pd.DataFrame([
    {"id": 124, "nombre": "Andrea", "calificacion": 8.5},
    {"id": 125, "nombre": "Carlos", "calificacion": 6.0}
])

# Desde array NumPy
import numpy as np
arr = np.random.rand(5, 3)
df = pd.DataFrame(arr, columns=["A", "B", "C"])
```

## Inspección rápida

- Al recibir un dataset nuevo, tu primer instinto debe ser inspeccionarlo.
- `df.info()` y `df.describe()` son tus comandos de cabecera iniciales.

```python
df.head()         # primeras 5 filas
df.tail()         # últimas 5 filas
df.shape          # (filas, columnas)
df.columns        # lista de columnas
df.dtypes         # tipos de cada columna
df.info()         # resumen completo
df.describe()     # estadísticas por columna numérica
```

## Acceso a columnas

- Puedes extraer una columna completa (Series) usando corchetes.
- Usa notación de punto (`df.columna`) solo si el nombre no tiene espacios.
- Para múltiples columnas, pasa una lista de nombres dentro de los corchetes.

```python
# Una columna (Series)
df["calificacion"]
df.calificacion         # equivalente, solo si nombre es identificador válido

# Varias columnas
df[["nombre", "calificacion"]]
```

## Acceso a filas

- Pandas ofrece dos métodos distintos para extraer filas específicas.
- `.iloc` utiliza posiciones basadas en números enteros (como listas en Python).
- `.loc` utiliza etiquetas explícitas (el índice o los nombres de las columnas).

```python
# Por posición (.iloc)
df.iloc[0]              # primera fila
df.iloc[0:3]            # primeras 3
df.iloc[0, 1]           # fila 0, columna 1

# Por etiqueta (.loc)
df.loc[0, "nombre"]     # fila índice 0, columna "nombre"
df.loc[0:2, "nombre":"calificacion"]
```

## Operaciones vectorizadas (heredadas de NumPy)

- Las operaciones se aplican a toda la columna simultáneamente.
- Se basan en NumPy, siendo increíblemente rápidas y sin usar bucles.
- Facilitan crear o transformar columnas con sintaxis matemática natural.

```python
df["calificacion"] * 10                    # vector entero
df["promedio_ponderado"] = (df["cal_anterior"] + df["cal_final"]) / 2  # nueva columna
df["aprobado"] = df["calificacion"] >= 6   # bool nueva
```

## Agregación y filtrado básicos

- Calcula estadísticas descriptivas de manera directa sobre las columnas.
- Usa condiciones lógicas para filtrar filas, creando subconjuntos de datos.

```python
# Estadísticas
df["calificacion"].mean()
df["calificacion"].std()
df["calificacion"].quantile(0.5)  # mediana

# Filtrado boolean
aprobados = df[df["calificacion"] >= 6]
top_alumnos = df[df["calificacion"] > 9]
```

## Métodos clave

- Pandas incluye decenas de métodos preconstruidos para limpieza y análisis.
- Los siguientes son los que usarás en el 90 % de tus proyectos:

| Método | Qué hace |
|---|---|
| `df.head(n)` | primeras n filas |
| `df.tail(n)` | últimas n filas |
| `df.sort_values("col")` | ordenar |
| `df.drop_duplicates()` | quitar duplicados |
| `df.dropna()` | quitar filas con NaN |
| `df.fillna(0)` | rellenar NaN con valor |
| `df["col"].astype(int)` | cambiar tipo |
| `df.rename(columns={"a":"b"})` | renombrar |
| `df.drop(columns=["c"])` | eliminar columna |
| `df["col"].apply(funcion)` | aplicar función a cada valor |
| `df["col"].value_counts()` | conteo por valor único |

## Crear / actualizar columnas

- Añade columnas nuevas asignando a un nombre que no exista.
- Transforma columnas existentes reasignando la misma variable.
- Puedes encadenar operaciones como `.clip()` o métodos `.str`.

```python
# Crear nueva
df["en_riesgo"] = (df["asistencia"] < 0.7) | (df["calificacion"] < 6)

# Modificar existente
df["calificacion"] = df["calificacion"].clip(0, 10)  # acota entre 0 y 10
df["nombre"] = df["nombre"].str.upper()
```

## Anti-patrones

- **Iterar fila por fila con `for`.** Destruye el rendimiento. Vectoriza siempre que puedas.
- **Mezclar `.loc` con `.iloc`.** Genera confusión. Decide uno y sé consistente.
- **Modificar copy (chained assignment).** `df[df["x"]>5]["y"] = 0` es ambiguo. Usa `.loc` en su lugar.
- **`.apply` cuando hay vectorización.** `.apply` es lento; vectoriza si puedes.
- **`describe()` a ciegas.** No todas las columnas (como IDs o categóricas) tienen sentido al describirse.
