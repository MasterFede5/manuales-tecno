# 2.4 Limpieza: nulos, duplicados, tipos

- Los datos del mundo real **siempre** vienen sucios y con errores.
- La limpieza ocupa el 60-70 % del trabajo de un *data scientist*.
- Veremos los 4 problemas más comunes y cómo solucionarlos.

::concepto{titulo="Limpieza de datos"}
Los **4 problemas canónicos** que Pandas resuelve:
1. **Valores faltantes (NaN)**: celdas vacías o "N/A".
2. **Duplicados**: filas repetidas por error.
3. **Tipos incorrectos**: números guardados como *strings*, fechas mal parseadas.
4. **Valores fuera de rango**: *outliers* o errores de captura.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Imagina un filtro de agua averiado: si entra agua turbia, tu café sabrá a barro. 
- En *Data Science*, el modelo es la cafetera.
- Un modelo entrenado con datos sucios da predicciones pésimas.
- La regla de oro es: *"Garbage in, garbage out"*.
::/interioriza

## Detectar y manejar valores faltantes (NaN)

```python
# Detectar
df.isna().sum()                # Total NaN por columna

# Eliminar o rellenar
df_limpio = df.dropna()                        # Elimina filas con cualquier NaN
df["calificacion"] = df["calificacion"].fillna(0) # Rellena NaN con 0
```
- **Eliminar**: Útil si los NaN son muy raros (<5 %).
- **Imputar**: Útil si son pocos y puedes usar media/mediana.

## Eliminar duplicados

```python
df.duplicated().sum()           # Cuántas filas repetidas hay
df_limpio = df.drop_duplicates(subset=["id"], keep="first")
```
- Siempre especifica `subset` con una clave única como el `id`.

## Convertir tipos y texto

```python
# Convertir tipos
df["edad"] = df["edad"].astype(int)
df["edad"] = pd.to_numeric(df["edad"], errors="coerce")  # Vuelve NaN los errores

# Limpiar strings
df["nombre"] = df["nombre"].str.strip().str.lower()
```

## Outliers (Valores fuera de rango)

```python
# Detectar con estadísticas
df["calificacion"].describe()

# Filtrar o acotar
df_limpio = df[(df["calificacion"] >= 0) & (df["calificacion"] <= 10)]
df["calificacion"] = df["calificacion"].clip(0, 10)  # Fuerza al rango
```

## Pipeline de limpieza completo

```python
def limpiar_dataset(df):
    df = df.drop_duplicates(subset=["id"], keep="first")
    df = df[(df["calificacion"] >= 0) & (df["calificacion"] <= 10)]
    df = df.dropna(subset=["calificacion"])
    df["id"] = df["id"].astype(int)
    return df.reset_index(drop=True)

df_limpio = limpiar_dataset(df)
```
- **Tip vital**: Usa `reset_index(drop=True)` al final para evitar saltos en el índice.
- Siempre imprime la cantidad de filas iniciales y finales para documentar cambios.

## Anti-patrones

- **Borrar a ciegas**: Si la columna es opcional, eliminar filas por un NaN te hace perder datos buenos.
- **Imputar sin pensar**: Reemplazar con la media altera la distribución original.
- **No guardar copias**: Si modificas `in-place` por error, pierdes los datos originales.
