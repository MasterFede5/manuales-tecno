# 2.5 Filtrado y selección

Tu DataFrame tiene 480 estudiantes.
Quieres "los aprobados de bachillerato técnico con promedio >8".
Esto es **filtrado**, una de las habilidades más usadas en datos.

::concepto{titulo="Filtrado y selección en pandas"}
Pandas ofrece **4 técnicas** principales para extraer subconjuntos:

- **Boolean indexing**: El más común, usando expresiones lógicas.
- **`.loc` / `.iloc`**: Selección por etiqueta o posición.
- **`.query()`**: Sintaxis tipo SQL en string.
- **`.isin()`**: Pertenencia a lista de valores.

Cada uno tiene su caso ideal. Aprender los 4 te da fluidez completa.
::/concepto

::interioriza{titulo="¿Por qué te importa? El colador de datos"}
Piensa en el filtrado como un colador de pasta inteligente.
Tiras los 480 estudiantes y los agujeros del colador son tus condiciones.
En pandas, diseñas la forma y tamaño de esos "agujeros" mediante código.
Esto es reproducible, se puede encadenar y escala a millones de filas.
::/interioriza

## Boolean indexing (el básico)

- **Filtros simples**: `df[df["calificacion"] >= 6]`
- **Múltiples condiciones**: Usa `&` (AND), `|` (OR).
- **Negación**: Usa `~` (NOT).

```python
exitosos = df[(df["calificacion"] >= 8) & (df["asistencia"] >= 0.85)]
no_aprobados = df[~(df["calificacion"] >= 6)]
```

- **Crucial**: Usa siempre paréntesis alrededor de cada condición.
- **No uses**: `and`, `or`, `not` estándar de Python.

## `.loc` y `.iloc`

Sirven para seleccionar partes específicas del DataFrame.
- **`.loc`**: Por etiquetas (nombres de columnas/índices).
- **`.iloc`**: Por posición numérica (0, 1, 2...).

```python
df.loc[df["calificacion"] >= 6, "nombre"] # Nombres de aprobados
df.iloc[0:5, 0:3]                         # Primeras 5 filas, primeras 3 columnas
```

- **Asignación rápida**: Ideales para modificar un subconjunto filtrado.
  `df.loc[df["calificacion"] < 6, "estado"] = "Reprobado"`

## `.query()` (sintaxis tipo SQL)

Permite escribir tus condiciones como texto, limpiando la sintaxis.
- **Legibilidad alta**: Perfecto si tienes muchos `AND`/`OR`.
- **Variables externas**: Usa el símbolo `@` antes de tu variable.

```python
umbral = 7
exitosos = df.query("calificacion >= @umbral and asistencia >= 0.85")
```

## `.isin()` (pertenencia)

- **Propósito**: Filtra filas si una columna está dentro de una lista.
- **Ventaja**: Evita tener que escribir múltiples condiciones `OR`.

```python
grupos = ["A1", "B2", "C3"]
seleccion = df[df["grupo"].isin(grupos)]
```

## Selección de columnas

- **Una sola columna**: `df["nombre"]` (Series).
- **Múltiples**: `df[["nombre", "calificacion"]]` (DataFrame).
- **Por tipo de dato**: `df.select_dtypes(include="number")`.
- **Por coincidencia**: `df.filter(like="cal")`.

## Combinaciones encadenadas

El estilo "Pro" de pandas consiste en encadenar métodos.
Cada línea procesa y transforma el resultado de la anterior.

```python
resultado = (df
    .query("calificacion > 8 and asistencia > 0.9")
    .nlargest(5, "calificacion")
    [["nombre", "calificacion", "asistencia"]]
)
```

## Patrones útiles y Anti-patrones

- **Extremos**: `df.nlargest(5, "cal")` y `df.nsmallest(3, "cal")`.
- **Muestra al azar**: `df.sample(10)` o `df.sample(frac=0.1)`.
- **Filtro de texto**: `df[df["materia"].str.contains("Mate")]`.

- **Evita**: Ignorar el `.copy()` si vas a modificar los datos tras filtrar.
- **Evita**: Caracteres especiales en nombres de columnas si vas a usar `.query()`.
