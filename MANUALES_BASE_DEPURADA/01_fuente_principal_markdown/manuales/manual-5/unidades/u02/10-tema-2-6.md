# 2.6 Agrupación (groupby) y agregaciones

¿Cuál es el promedio de calificación **por materia**?
¿Cuántos estudiantes **por nivel** aprobaron?
Estas preguntas las responde `groupby` en una sola línea.

::concepto{titulo="GroupBy y agregaciones"}
**GroupBy** es la operación que agrupa filas y les aplica una función agregada.
- **Agrupa:** por uno o más valores.
- **Aplica:** funciones como suma, promedio, conteo, max, min.
- **Devuelve:** un DataFrame con los resultados.

Sigue el patrón **split-apply-combine**:
- **Split:** divide el DataFrame en grupos según valores de la columna.
- **Apply:** calcula la estadística o función agregada en cada grupo.
- **Combine:** une los resultados en un nuevo DataFrame.

```python
df.groupby("materia")["calificacion"].mean()
# Series con promedio por materia
```

Es la operación más poderosa de pandas para análisis exploratorio.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Piensa en `groupby` como el organizador de un salón de clases:
- **Split:** Separa a los alumnos en esquinas distintas por materia.
- **Apply:** Le pide a cada grupo que sume o promedie sus calificaciones.
- **Combine:** Anota los promedios juntos en la pizarra.

En Excel, esto exige varios clics y configurar tablas dinámicas.
En pandas, es una línea reproducible y escalable a millones de datos.
Si dominas `groupby`, dominas el 80% del análisis exploratorio.
::/interioriza

## Sintaxis básica

```python
# Promedio por categoría
df.groupby("materia")["calificacion"].mean()

# Múltiples agregaciones
df.groupby("materia")["calificacion"].agg(["mean", "min", "max", "count"])

# Por múltiples columnas
df.groupby(["nivel", "materia"])["calificacion"].mean()

# Múltiples columnas a agregar
df.groupby("materia")[["calificacion", "asistencia"]].mean()

# Agregaciones diferentes por columna
df.groupby("materia").agg({
    "calificacion": "mean",
    "asistencia": "mean",
    "horas_estudio": "sum"
})
```

## Funciones de agregación más usadas

- `mean()`, `median()`: Promedio y mediana.
- `sum()`: Suma.
- `count()`, `size()`: Conteo sin NaN / con NaN.
- `min()`, `max()`: Mínimo y máximo.
- `std()`, `var()`: Desviación y varianza.
- `nunique()`: Valores únicos.
- `first()`, `last()`: Primero y último.
- `quantile(0.5)`: Percentil (ej. mediana).

## Función personalizada con apply

```python
# Tasa de aprobación por materia
def tasa_aprobacion(grupo):
    return (grupo["calificacion"] >= 6).mean()

df.groupby("materia").apply(tasa_aprobacion)
```

## Transformaciones (no agregadas)

`transform` devuelve algo del mismo tamaño que el grupo, útil para crear columnas comparativas:

```python
# Diferencia con promedio del grupo
df["promedio_materia"] = df.groupby("materia")["calificacion"].transform("mean")
df["diff_promedio"] = df["calificacion"] - df["promedio_materia"]

# Ranking dentro del grupo
df["rank_en_materia"] = df.groupby("materia")["calificacion"].rank(ascending=False)
```

## Aplicado al caso escolar

```python
# Estadísticas completas por materia
resumen = df.groupby("materia").agg(
    n_alumnos=("id", "count"),
    cal_promedio=("calificacion", "mean"),
    cal_mediana=("calificacion", "median"),
    cal_std=("calificacion", "std"),
    asistencia_promedio=("asistencia", "mean"),
    tasa_aprobacion=("calificacion", lambda x: (x >= 6).mean())
).round(2)

print(resumen)
```

Una sola línea.
Resultado idéntico a 30 minutos de tabla dinámica de Excel.

## Multi-nivel (group por varias columnas)

```python
df.groupby(["nivel", "materia"])["calificacion"].mean()
```

Resultado: índice multi-nivel. Para "aplanar":

```python
df.groupby(["nivel", "materia"])["calificacion"].mean().reset_index()
```

## Anti-patrones

- **Usar lambdas innecesarias:** `groupby().apply(lambda x: x.mean())` es lento; prefiere `groupby().mean()`.
- **Olvidar `.reset_index()`:** Guarda índices multi-nivel molestos si exportas a CSV.
- **Agregaciones superpuestas:** `agg({"y": "mean", "y": "sum"})` choca; usa nombres `agg(media=("y", "mean"))`.
- **Grupos con NaN:** `groupby` ignora filas si su columna agrupadora tiene NaN.
