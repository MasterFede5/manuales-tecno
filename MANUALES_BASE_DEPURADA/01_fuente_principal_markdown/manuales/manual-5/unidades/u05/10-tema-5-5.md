# 5.5 Regresión lineal

- Es **el modelo más simple posible**.
- Ajusta una línea recta (o un plano multidimensional) a tus datos.
- Su principal ventaja es que es 100% interpretable matemáticamente.

::concepto{titulo="Regresión lineal"}
Un modelo de regresión lineal toma la forma de una ecuación:
`y = β₀ + β₁·x₁ + β₂·x₂ + ... + βₙ·xₙ`

- **β₀ (Intercepto):** El valor base de `y` si todas las variables `x` valen cero.
- **βᵢ (Coeficientes):** Cuánto aumenta (o disminuye) `y` por cada unidad adicional de `xᵢ`.

```python
from sklearn.linear_model import LinearRegression

# Crear y entrenar
modelo = LinearRegression()
modelo.fit(X_train, y_train)

# Extraer interpretación
print(f"Intercepto base: {modelo.intercept_:.3f}")
print(f"Coeficientes de impacto: {modelo.coef_}")
```
::/concepto

::interioriza{titulo="El Ticket del Súper"}
Piensa en la regresión lineal como el ticket de compra del supermercado.
- El **Intercepto** (`β₀`) es como el costo fijo del envío a domicilio. Siempre lo pagas.
- Los **Coeficientes** (`βᵢ`) son los precios unitarios de cada producto. 
Al final, sabes exactamente por qué te cobraron lo que te cobraron. No hay magia ni cajas negras.
::/interioriza

## Aplicación al caso escolar

```python
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression

# 1. Separar variables explicativas (X) del objetivo (y)
X = df[["horas_estudio", "asistencia", "cal_anterior"]]
y = df["cal_final"]
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# 2. Entrenar el modelo con los datos
modelo = LinearRegression()
modelo.fit(X_train, y_train)

# 3. Mostrar la ecuación matemática descubierta
print(f"cal_final = {modelo.intercept_:.2f} + {modelo.coef_[0]:.3f}*horas + ...")
```
- **Output esperado:**
`cal_final = 0.42 + 0.085*horas_estudio + 1.234*asistencia + 0.481*cal_anterior`

## ¿Qué nos dicen los coeficientes?

- **0.085 × horas:** 1 hora extra de estudio suma 0.085 a la calificación final.
- **1.234 × asistencia:** Subir del 0% al 100% de asistencia suma 1.23 puntos.
- **0.481 × cal_anterior:** 1 punto extra en el curso previo equivale a medio punto extra hoy (+0.48).

## Realizando predicciones

```python
# Un estudiante nuevo con sus métricas
nuevo = pd.DataFrame([{"horas_estudio": 5.0, "asistencia": 0.92, "cal_anterior": 7.5}])

# Pasarlo por el modelo
prediccion = modelo.predict(nuevo)
print(f"Predicción cal_final: {prediccion[0]:.2f}") # Resultado: 7.31
```

## ¿Cuándo podemos confiar en esto?

La regresión lineal asume cuatro reglas de oro:
- **Linealidad:** La relación entre X e Y se puede trazar con una línea.
- **Homocedasticidad:** El error o dispersión es constante, no crece en embudo.
- **Independencia:** Un dato no afecta al resultado de otro dato consecutivo.
- **Normalidad:** Los errores del modelo (residuos) forman una campana de Gauss.

## Cuándo usar y cuándo NO usar

- **SÍ ÚSALO cuando:** Necesitas predecir un número (continuo), las relaciones parecen lineales y exigen máxima interpretabilidad legal o de negocio.
- **NO LO USES cuando:** Quieres predecir categorías (A/B), la relación tiene formas de U, o tienes miles de variables.

## Anti-patrones comunes

- **Multicolinealidad:** Meter variables repetidas o equivalentes (ej. edad en años y edad en meses). Destruye los coeficientes.
- **Extrapolación ciega:** Intentar predecir la calificación de alguien que estudió 1000 horas, si tu dataset solo llegaba hasta 20.
- **Ceguera de formatos:** Alimentar texto directo o categorías ordinales sin codificarlas (One-Hot) primero.

::/pausa
