# 8.6 Despliegue con Streamlit

Tu código en notebook funciona perfectamente, pero solo para ti.
Para que la coordinadora lo use sin abrir Colab, lo convertiremos en **app web** con Streamlit.
Lograrás esto sin usar HTML, CSS, ni configurar un servidor.

::concepto{titulo="Streamlit"}
**Streamlit** es un framework de Python para construir aplicaciones web interactivas.
Está diseñado para data science, usando código Python puro.
No necesitas saber JavaScript, HTML o frontend.

```python
import streamlit as st

st.title("Mi App")
nombre = st.text_input("Tu nombre")
if st.button("Saludar"):
    st.write(f"Hola, {nombre}")
```

- Corres la app con `streamlit run app.py` y se abre en tu navegador.
- Opciones de despliegue: Streamlit Community Cloud, Hugging Face Spaces, Render, Heroku.
::/concepto

::interioriza{titulo="Baja Curva: El Traductor Universal"}
Imagina que Streamlit es como un intérprete en tiempo real en una reunión internacional. 
Tu modelo de ML habla un idioma que solo tú (y tu IDE) entienden. 
Streamlit traduce ese código complejo en una interfaz visual amigable (botones y selectores).
Así, la coordinadora académica puede interactuar con la predicción de los alumnos sin saber programar.
::/interioriza

## App básica

```python
# app.py
import streamlit as st
import joblib
import pandas as pd
import os
from anthropic import Anthropic

# Cargar modelo
model = joblib.load("modelo_predictor_v1.pkl")

claude = Anthropic(api_key=st.secrets["ANTHROPIC_API_KEY"])

st.title("🎓 Predictor de Rendimiento Albatros")
st.markdown("Predicción + explicación con IA")

# Inputs
col1, col2, col3 = st.columns(3)
with col1:
    horas = st.slider("Horas de estudio/semana", 0.0, 10.0, 4.0)
with col2:
    asistencia = st.slider("Asistencia", 0.0, 1.0, 0.85)
with col3:
    cal_ant = st.slider("Calificación anterior", 0.0, 10.0, 7.0)

if st.button("Predecir y explicar"):
    # Predecir
    X = pd.DataFrame([[horas, asistencia, cal_ant]], 
                     columns=["horas_estudio", "asistencia", "cal_anterior"])
    pred = model.predict(X)[0]
    
    # Mostrar
    st.metric("Calificación predicha", f"{pred:.2f}")
    
    # Explicar con IA
    riesgo = "ALTO" if pred < 6 else ("MEDIO" if pred < 7 else "BAJO")
    
    prompt = f"""Eres asesor académico. Un alumno con:
- Horas estudio: {horas}/semana
- Asistencia: {asistencia*100:.0f}%
- Calificación anterior: {cal_ant}

Tiene predicción de calificación final: {pred:.2f}. Riesgo: {riesgo}.

Explica en 3 partes:
1. Por qué la predicción (factores principales).
2. Qué cambiar para mejorar.
3. Acción concreta sugerida al coordinador."""
    
    with st.spinner("Generando explicación..."):
        response = claude.messages.create(
            model="claude-sonnet-4-6",
            max_tokens=600,
            messages=[{"role": "user", "content": prompt}]
        )
        explicacion = response.content[0].text
    
    st.markdown("### Análisis")
    st.write(explicacion)
```

- **Run:** `streamlit run app.py` → abre `localhost:8501`.

## Componentes Streamlit comunes

- **Textos y Métricas:**
  - `st.title("...")`, `st.header("...")`, `st.write("...")`
  - `st.markdown("**bold**")`, `st.metric("Etiqueta", valor, delta)`
- **Inputs:**
  - Textos/Números: `st.text_input()`, `st.number_input()`
  - Opciones: `st.slider()`, `st.selectbox()`, `st.multiselect()`
  - Booleanos/Archivos: `st.checkbox()`, `st.file_uploader()`
- **Layout (Diseño visual):**
  - Columnas: `st.columns(2)`
  - Pestañas/Desplegables: `st.tabs([...])`, `st.expander("...")`
- **Visualizaciones:**
  - `st.dataframe(df)`, `st.line_chart(data)`, `st.plotly_chart(fig)`

## Manejo de secrets y Anti-patrones

- **Manejo Seguro de Keys:** Nunca subas keys a GitHub. Usa `st.secrets` y guárdalas en `.streamlit/secrets.toml`.
- **Carga de Modelos:** Usa `@st.cache_resource` al cargar el modelo de ML. Así evitas recargarlo en cada clic.
- **Manejo de Errores:** Añade bloques `try/except` para que la app no colapse si la API falla.
- **Diseño Personalizado:** Evita el aspecto "clon de ChatGPT por defecto". Usa columnas y layouts para darle un toque profesional.
