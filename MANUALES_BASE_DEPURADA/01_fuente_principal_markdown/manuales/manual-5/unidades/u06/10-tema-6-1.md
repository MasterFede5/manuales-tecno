# 6.1 Concepto de clustering

- **¿Qué es?** Agrupar elementos similares sin instrucciones humanas.
- **Tu computadora** encuentra estructura latente en los datos.

::concepto{titulo="Clustering"}
- **Clustering**: divide un conjunto de datos en grupos (**clusters**).
- **Similitud**: elementos del mismo cluster son muy parecidos.
- **Diferencia**: elementos de distintos clusters son diferentes.
- **No hay etiquetas**: el algoritmo descubre grupos (usando distancia euclidiana).

**Algoritmos comunes**:
- **K-Means**: el más simple y popular (cubierto en este manual).
- **DBSCAN**: detecta formas raras y outliers.
- **Hierarchical / Gaussian Mixtures**: ver glosario.
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
- Sin etiquetas, la clasificación supervisada NO sirve.
- Clustering descubre estructura latente de forma automática.
- **Analogía**: Como ordenar una caja de legos por color sin saber qué colores hay dentro, solo viendo cuáles se parecen.
::/interioriza

## Aplicaciones canónicas

- **Negocios**: segmentación de clientes en marketing.
- **Tecnología**: compresión de imágenes (agrupando píxeles similares).
- **Seguridad**: detección de anomalías (puntos lejos de grupos).
- **Nuestro Case**: perfiles de aprendizaje escolar (horas, asistencia, notas).

## Anatomía conceptual

```text
Datos (Alumnos):
  1: (5h, 0.9, 8) | 2: (2h, 0.6, 5) | 3: (5h, 0.85, 8.5)

Clustering descubre:
  - Cluster A (estudiosos): alumnos 1, 3...
  - Cluster B (en riesgo): alumno 2...
```

## Distancia: el corazón del clustering

Para decir "similar", necesitas una **métrica de distancia**:
- **Euclidiana**: línea recta (K-Means la usa por default).
- **Manhattan**: suma de diferencias absolutas (como caminar cuadras).
- **Coseno**: ángulo entre vectores (para procesar texto).

## Pre-procesamiento crítico: scaling

- **El problema**: si asistencia es 0-1 y notas 0-100, las notas dominarán la distancia.
- **La solución**: siempre **normaliza** tus datos primero.

```python
from sklearn.preprocessing import StandardScaler
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X) # Media 0, Std 1
```

## Anti-patrones

- **Ignorar scaling**: arruinarás las distancias.
- **Asumir que K-Means todo lo puede**: solo encuentra clusters esféricos.
- **Buscar la verdad**: clusters son descubrimientos, no validación final.
