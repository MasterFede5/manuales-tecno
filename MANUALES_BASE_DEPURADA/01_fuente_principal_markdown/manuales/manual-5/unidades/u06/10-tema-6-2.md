# 6.2 K-Means paso a paso

K-Means es el algoritmo de clustering más popular.
Destaca por ser simple, rápido y muy efectivo.
Aquí lo aprenderás en 4 pasos y con código.

::concepto{titulo="K-Means"}
Divide los datos en **K** clusters con un proceso iterativo:

* **Inicializar**: elige K puntos (centroides) al azar.
* **Asignar**: cada dato va a su centroide más cercano (distancia euclidiana).
* **Recalcular**: el centroide se mueve al promedio de sus datos.
* **Repetir**: vuelve al paso 2 hasta que nada cambie (convergencia).

Garantía: Siempre converge, pero a un mínimo local.
Depende fuertemente de una buena inicialización.

```python
from sklearn.cluster import KMeans
kmeans = KMeans(n_clusters=4, random_state=42, n_init=10)
kmeans.fit(X_scaled)
labels = kmeans.labels_
centroids = kmeans.cluster_centers_
```
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
K-Means es como organizar tu armario por color:
* Empiezas con cajas en lugares al azar.
* Pones cada prenda en la caja del color más cercano.
* Mueves cada caja al "centro" de las prendas que tiene.
* Repites hasta estabilizar. Esto es exactamente K-Means.
::/interioriza

## Aplicado al caso escolar

```python
import pandas as pd
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler
import matplotlib.pyplot as plt
import seaborn as sns

df = pd.read_csv("dataset_escolar_limpio.csv")
features = ["horas_estudio", "asistencia", "cal_anterior", "cal_final"]
X = df[features]

# 1. Scaling (¡crucial!)
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# 2. Aplicar K-Means
kmeans = KMeans(n_clusters=4, random_state=42, n_init=10)
df["cluster"] = kmeans.fit_predict(X_scaled)

# 3. Caracterizar cada cluster
print(df.groupby("cluster")[features].mean().round(2))
```

Output típico:
```
         horas_estudio  asistencia  cal_anterior  cal_final
cluster
0                 6.5         0.92          8.5        8.7
1                 4.5         0.85          7.0        7.2
2                 2.0         0.55          5.0        4.5
3                 5.5         0.75          6.5        6.8
```

Interpretación rápida de los grupos:
* **Cluster 0**: Excelentes (estudian 6.5h, asisten 92%, venían bien).
* **Cluster 1**: Buenos consistentes (4.5h, 85%, promedio 7-7.5).
* **Cluster 2**: En riesgo (2h, 55% asistencia, reprueban).
* **Cluster 3**: Medio-bajo (mejorable con apoyo adicional).

## Detalles Clave

**Inicialización Importante:**
K-Means converge a un mínimo local. Una mala inicialización da malos clusters.

```python
# n_init=10: prueba 10 inicializaciones, elige la mejor.
KMeans(n_clusters=4, random_state=42, n_init=10) 
```
* Nota: `n_init=10` es el default reciente; antes era 1 y causaba bugs.

**Visualizar clusters (PCA):**
Para ver en 2D, aplicamos PCA (ver 6.4):

```python
from sklearn.decomposition import PCA
import matplotlib.pyplot as plt

pca = PCA(n_components=2)
X_2d = pca.fit_transform(X_scaled)

plt.scatter(X_2d[:,0], X_2d[:,1], c=df["cluster"], cmap="viridis", s=30)
cent_2d = pca.transform(kmeans.cluster_centers_)
plt.scatter(cent_2d[:,0], cent_2d[:,1], marker="X", s=200, c="red", edgecolor="black")
plt.title("Clusters de estudiantes (PCA 2D)")
plt.show()
```

## Evaluación y Uso Práctico

**Inertia (criterio de calidad):**
* Es la suma de distancias al cuadrado de cada punto a su centroide.
* **Menor es mejor**, pero siempre baja al aumentar K. No confíes solo en esto.

```python
print(f"Inertia: {kmeans.inertia_:.2f}")
```

**Predecir alumnos nuevos:**
```python
nuevo = scaler.transform([[5.0, 0.9, 7.0, 0]]) # 0 es placeholder
cluster_predicho = kmeans.predict(nuevo)
print(f"Cluster: {cluster_predicho[0]}")
```

**Limitaciones:**
* **K fijo**: No descubre el número óptimo por sí solo.
* **Formas**: No funciona bien con clusters alargados, asume esféricos.
* **Outliers**: Un valor atípico mueve el centroide drásticamente.
* **Tamaños**: Asume clusters de tamaños similares.

**Anti-patrones comunes:**
* **Omitir scaling**: La variable más grande dominará.
* **K arbitrario**: Usar K=3 "porque sí" daña tu análisis (usa métodos codo/silueta).
* **Falsa verdad absoluta**: Son descubrimientos, no verdades irrefutables.
