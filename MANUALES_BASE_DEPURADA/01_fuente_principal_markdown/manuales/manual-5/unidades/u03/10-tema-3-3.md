# 3.3 Seaborn para gráficos rápidos y bonitos

Si `matplotlib` es la estructura básica, **Seaborn** es la capa de decoración y diseño. 
Esta librería te permite generar visualizaciones con calidad de publicación escribiendo una sola línea de código.

::concepto{titulo="Seaborn"}
**Seaborn** (sns) es la librería de alto nivel para gráficas estadísticas en Python.

Sus principales gráficos son:
- `sns.histplot` — Histograma con línea de densidad (KDE).
- `sns.scatterplot` — Gráfico de dispersión categorizado por colores.
- `sns.boxplot` — Diagrama de caja y bigotes para comparar grupos.
- `sns.violinplot` — Mezcla de boxplot y gráfico de densidad.
- `sns.heatmap` — Matriz de calor para observar correlaciones.
- `sns.pairplot` — Matriz que cruza todas las variables de un dataset.

**Tip:** Seaborn se alimenta directamente de DataFrames: `sns.histplot(data=df, x="cal_final")`.
::/concepto

::interioriza{titulo="El traje a la medida"}
Imagina que `matplotlib` es como comprar tela e hilos para tejer tu propia camisa desde cero. Requiere tiempo y mucha precisión. 
Por otro lado, `seaborn` es como ir a una boutique exclusiva: pides una talla y te entregan un traje elegante, pulido y listo para usar en tu presentación de negocios.
::/interioriza

## Gráficos esenciales de Seaborn

### Histogramas

```python
import seaborn as sns
sns.histplot(data=df, x="cal_final", bins=20, kde=True)
```
- Esto genera un histograma con su respectiva curva de densidad en una sola línea.

### Histogramas por grupos

```python
sns.histplot(data=df, x="cal_final", hue="materia", bins=20, kde=True)
```
- El parámetro `hue="materia"` colorea los datos según la materia automáticamente.

### Dispersión (Scatter) por grupos

```python
sns.scatterplot(data=df, x="horas_estudio", y="cal_final", hue="materia", size="asistencia")
```
- Aquí analizamos 4 dimensiones: 2 ejes (x, y), un color por materia (`hue`) y un tamaño por asistencia (`size`).

### Comparación con Boxplot

```python
sns.boxplot(data=df, x="materia", y="cal_final")
```
- Perfecto para comparar medianas y rangos entre varias categorías al mismo tiempo.

### Violinplot (Boxplot + Densidad)

```python
sns.violinplot(data=df, x="materia", y="cal_final")
```
- Muestra el mismo rango que un boxplot, pero revela la forma exacta en la que se distribuyen los datos.

### Matriz de correlación (Heatmap)

```python
correlaciones = df[["horas_estudio", "asistencia", "cal_anterior", "cal_final"]].corr()
sns.heatmap(correlaciones, annot=True, cmap="coolwarm", center=0)
```
- Pinta con colores las relaciones más fuertes. `annot=True` imprime los números dentro de los recuadros.

### Análisis cruzado (Pairplot)

```python
sns.pairplot(df[["horas_estudio", "asistencia", "cal_final"]], hue="materia")
```
- Cruza todas las variables numéricas entre sí. 
- **Consejo:** Es la herramienta ideal para la exploración inicial de un nuevo dataset.

## Personalización rápida

```python
# Aplica estilos predefinidos a todos los gráficos siguientes:
sns.set_theme(style="whitegrid")  # Opciones: darkgrid, ticks, white
sns.set_palette("Set2")            # Opciones: viridis, husl, Set1
```

## Anti-patrones comunes

- **Reinventar la rueda:** No intentes programar manualmente en `matplotlib` lo que `seaborn` ya hace en una línea.
- **Uso ciego de colores:** Ten cuidado con paletas que no son amigables para daltónicos. Usa `viridis` como opción segura.
- **Pairplots gigantes:** Aplicar `pairplot` a 20 variables saturará el gráfico y tardará mucho en procesarse. Límitalo a 5 variables.
