# 3.1 Matplotlib esencial: línea, barras, dispersión

- Matplotlib es **la base** de visualización en Python.
- Casi todas las otras librerías construyen sobre ella.
- Aprender los 3 gráficos básicos te abre las puertas al resto.

::concepto{titulo="Matplotlib"}
**Matplotlib** es la librería 2D líder. Su API principal es **`pyplot`** (`plt`).
Tres gráficos cubren el 80% de tus necesidades:
- **Línea** (`plt.plot`): Tendencias en el tiempo.
- **Barras** (`plt.bar`): Comparar categorías.
- **Dispersión** (`plt.scatter`): Relaciones entre 2 variables numéricas.

```python
import matplotlib.pyplot as plt
plt.plot([1, 2, 3], [4, 5, 6])
plt.show()
```

- Hay dos estilos: **funcional** (pyplot, más simple) y **orientado a objetos** (fig/ax).
- Empieza con pyplot para lo básico.
::/concepto

## Gráfico de línea (tendencia)

```python
import matplotlib.pyplot as plt

semanas = [1, 2, 3, 4, 5, 6]
promedio = [7.2, 7.5, 7.8, 7.6, 8.0, 8.2]

plt.plot(semanas, promedio, marker="o", color="blue")
plt.title("Promedio por semana")
plt.show()
```

- **Cuándo usar:** Para ver evolución y tendencias temporales.

## Gráfico de barras (comparar categorías)

```python
materias = ["Mate", "Física", "Química"]
promedios = [6.95, 7.42, 7.65]

plt.bar(materias, promedios, color=["#ff6b6b", "#4ecdc4", "#ffe66d"])
plt.title("Promedio por materia")
plt.show()
```

- **Cuándo usar:** Para contrastar valores entre categorías discretas.

## Gráfico de dispersión (relación de 2 variables)

```python
import pandas as pd
df = pd.read_csv("dataset_escolar.csv")

plt.scatter(df["horas_estudio"], df["cal_final"], alpha=0.5)
plt.title("Horas vs Calificación")
plt.show()
```

- **Cuándo usar:** Para explorar correlación numérica.
- El `alpha=0.5` añade transparencia si los puntos se solapan.

## Anatomía: figure y axes

- Para proyectos más complejos, usa `fig` y `ax`.
- `fig` es tu lienzo completo, `ax` es el área específica del gráfico.

```python
fig, ax = plt.subplots(figsize=(10, 6))
ax.bar(materias, promedios)
ax.set_title("Promedio por materia")
plt.show()
```

## Guardar y exportar

```python
plt.savefig("grafico.png", dpi=150, bbox_inches="tight")
```

- `dpi=150` para digital, `300` para imprimir.
- `bbox_inches="tight"` evita recortar los ejes.

## Anti-patrones

- **Falta de etiquetas:** Sin contexto, el gráfico no sirve.
- **Exceso de colores:** Usa 3-5 máximo para no confundir.
- **Eje Y sin cero en barras:** Distorsiona la percepción visual.
