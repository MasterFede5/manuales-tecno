# 2.3 Claude: artifacts, projects, styles y Computer Use

Claude es el modelo de **Anthropic**, creado por ex-miembros de OpenAI enfocados en seguridad.
Se distingue por:
- **Redacción superior** y un tono más natural y humano.
- **Gran ventana de contexto** capaz de procesar libros enteros.
- **Artifacts y Projects** para estructurar tu espacio de trabajo.

::concepto{titulo="Claude como plataforma"}
**Modelos (2025):** Opus 4.7 (top), Sonnet 4.6 (balanceado), Haiku 4.5 (rápido).

**Niveles principales:**
- **Free:** Sonnet 4.6 con límites de uso diario.
- **Pro ($20 USD):** Opus 4.7 + Projects + Artifacts.

**Diferenciadores clave:**
1. **Artifacts:** Panel lateral con recursos editables en vivo (código, tablas, SVG).
2. **Projects:** Espacios con memoria y documentos compartidos.
3. **Styles:** Perfiles de personalidad persistentes.
4. **Computer Use:** Claude controla tu mouse y teclado.
::/concepto

::interioriza{titulo="¿Por qué te importa? El escritorio ordenado"}
Imagina a ChatGPT como un erudito brillante que trabaja en una sola libreta donde anota todo mezclado. 
Claude es como tener **un escritorio con cajones separados** (Projects). 
En un cajón tienes tu "Tutor de Cálculo" con sus temarios y en otro tu "Tesis".
Además, su panel *Artifacts* es como una pizarra: no te explica cómo dibujar el problema, te lo dibuja ahí mismo para que lo modifiques en vivo.
::/interioriza

## Artifacts: el documento que cobra vida

Cuando Claude genera contenido complejo (código, páginas web, diseños), abre un **panel lateral**.
Aquí interactúas directamente:
- **Visualiza** resultados (como HTML renderizado o gráficas SVG).
- **Edita** a mano el texto o código generado.
- **Pide ajustes** al chat para modificar partes específicas.

> **Caso de uso:** Es la diferencia entre recibir "instrucciones para programar" y recibir una **página funcional** que puedes probar al instante.

## Projects: conocimiento por contexto

Un **Project** agrupa chats bajo un mismo *system prompt* y documentos base.
- **Ejemplo Tesis:** Prompt académico + PDFs de investigaciones.
- **Ejemplo Tutor:** Prompt pedagógico + programa del curso.

Cada chat nuevo en el proyecto hereda esta memoria automáticamente.

> **Diferencia clave:** Los Projects de Claude son **espacios de trabajo privados**. Los GPTs de OpenAI se enfocan en **distribución pública**.

## Styles: cambia el tono al instante

Modifica la personalidad de Claude sin alterar tu prompt cada vez:
- **Predefinidos:** Formal, Conciso, Explicativo.
- **Custom:** Define reglas globales (ej. que siempre responda en viñetas).

> **Recomendación:** Configura un Style "Tutor pedagógico" que te obligue a deducir antes de entregarte respuestas.

## Computer Use: el modelo toma el mouse

Claude puede ver tu pantalla, mover el cursor y escribir en tus apps.
Es un **agente activo**, no solo un chatbot conversacional.
- **Potencial:** Puede navegar una web, buscar un restaurante y llenar la reserva por ti.
- **Precaución:** Aún comete errores. Requiere supervisión y no es ideal para pagos críticos.

## Claude Code y Plan Recomendado

**Claude Code:** Interfaz de terminal que edita tu repositorio y corre tests autónomamente.

**Plan para tu tutor (Pro $20/mes):**
- Ideal si valoras la **redacción cuidada**, manejas **documentos gigantes** o necesitas interactuar con **Artifacts**.
- La versión gratuita basta para comenzar, pero limitará tus pruebas complejas.
