# 2.6 Comparativa práctica: cuándo usar cada uno

Existen cuatro plataformas excelentes, pero muy diferentes entre sí.
Si las conoces solo de oídas, es muy probable que elijas mal.

La "mejor IA" depende de dos factores críticos:
- **Qué quieres hacer** (escribir, codificar, dibujar).
- **Dónde vives digitalmente** (Google Workspace, Microsoft 365).

::concepto{titulo="Criterios de selección"}
Para elegir tu herramienta ideal, evalúa estas cinco dimensiones:

1. **Tarea principal** — ¿escritura larga, código, análisis, multimedia?
2. **Ecosistema actual** — ¿Google Workspace, Microsoft 365, ninguno?
3. **Presupuesto** — ¿gratis, $20/mes, $30/mes?
4. **Privacidad** — ¿cuánta información sensible vas a compartir?
5. **Funciones distintivas** — ¿necesitas Artifacts, Voice o Computer Use?

Recuerda: no hay un ganador absoluto en todo.
Solo existe un **ganador por caso de uso**.
::/concepto

::interioriza{titulo="🚗 Analogía: Comprar un auto"}
Elegir tu IA principal es como elegir un vehículo.
No comprarías un deportivo biplaza para hacer mudanzas, ni un camión de carga para correr en una pista.

Tomarte 30 minutos hoy para elegir bien te ahorrará meses de frustración.
Tu tutor virtual será una relación a **largo plazo**.
¡Un mal ajuste temprano crea fricciones que se acumulan con el tiempo!
::/interioriza

## Comparativa por tarea concreta

| Tarea | Mejor opción | Por qué |
|---|---|---|
| **Resumen de PDF largo** | Claude Opus | Ventana 1M, retiene cada detalle. |
| **Generar imagen** | ChatGPT o Gemini | Integraciones DALL·E 3 e Imagen 3. |
| **Correo formal** | El más cercano a tu inbox | Cero copy/paste (Gemini o Copilot). |
| **Código/Debugging** | Claude o ChatGPT o3 | Modelos razonadores especializados. |
| **Lluvia de ideas** | ChatGPT Voice | Menor latencia, voz más natural. |
| **Análisis de Excel** | Copilot Pro | Modifica celdas directamente. |
| **Investigación** | Perplexity | Especializado en citas (ver U5). |
| **Tutor de matemáticas**| Claude o ChatGPT o3 | Razonamiento lógico paso a paso. |

## Estrategia recomendada para este manual

**Mínimo requerido (Costo $0):**
- **Claude** (claude.ai) — para usar Artifacts y Projects.
- **ChatGPT** (chatgpt.com) — para usar GPTs en la Unidad 7.

**Recomendado (Si hay presupuesto):**
- **Claude Pro** ($20/mes) si vas a escribir documentos largos.
- O **ChatGPT Plus** ($20/mes) si buscas herramientas multimodales.

No necesitas pagar las cuatro opciones.
Empieza con dos versiones gratuitas y explora.
Luego invierte en la que mejor resuelva tu día a día.

Para el caso práctico **"Mi tutor IA personal"**, te sugiero:
- **Claude Pro** como cerebro principal (por su redacción superior).
- **ChatGPT Free** como apoyo para usar GPTs puntuales.

Esta dupla resuelve el 95% de tus necesidades por $20 al mes.
