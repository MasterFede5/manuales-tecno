# 2.2 ChatGPT: interfaz, modelos, voz, Canvas y GPTs

ChatGPT es mucho más que un simple chat de texto.
Es la plataforma con mayor integración de funciones del mercado.
Saber cómo aprovechar sus herramientas te ahorrará horas.
Te abrirá opciones avanzadas para crear tu tutor virtual.

::concepto{titulo="ChatGPT como plataforma"}
**ChatGPT** es la interfaz web/móvil de OpenAI.
Opera sobre sus distintos modelos (GPT-4o, 5, o1, o3).
Niveles de servicio:
- **Free:** GPT-4o limitado.
- **Plus ($20 USD/mes):** GPT-5 + o3 + DALL·E + voz avanzada.
- **Pro ($200 USD/mes):** Uso ilimitado + modelos de razonamiento superior.

**7 Funciones Clave (2025):**
1. **Selector:** Eliges el modelo según la tarea.
2. **Voz:** Conversación bidireccional real.
3. **DALL·E 3:** Generación de imágenes.
4. **Browsing:** Búsqueda web en vivo.
5. **GPTs:** Versiones personalizadas tuyas/de terceros.
6. **Canvas:** Editor lateral de texto/código.
7. **Memoria:** Persistencia de preferencias.
::/concepto

::interioriza{titulo="Analogía: La Caja de Herramientas del Tutor"}
Imagina que vas a construir una casa (tu tutor virtual).
Usar ChatGPT solo para texto es como usar un destornillador para todo (clavar, medir, cortar).
Si integras *Voz*, *Canvas* y *GPTs*, abres toda la caja de herramientas.
Cada función es una herramienta experta para acelerar tu aprendizaje.
::/interioriza

## Selector de modelos: Elige bien

No siempre necesitas el modelo más caro y potente.
- **GPT-4o:** Conversación rápida y multimodal.
- **GPT-5:** Tareas complejas y ensayos largos.
- **o1 / o3:** Problemas de lógica profunda (matemáticas o programación).
- **GPT-4o mini:** Consultas rápidas y económicas.

**Trampa común:** Usar *o3* para un simple resumen es un desperdicio. 
Usa *mini* para lo básico y reserva *o3* para el razonamiento paso a paso.

## Modo voz (Voice Mode)

Toca el ícono de auriculares en la app móvil.
- **Práctica de idiomas:** Habla en inglés y recibe correcciones al instante.
- **Brainstorming:** Genera ideas y dialoga mientras caminas (sin manos).
- **Tutor de audio:** Pide explicaciones orales de un problema escrito en papel.

> **Tip pro:** El *Advanced Voice Mode* (Plus) incluye interrupciones y tono emocional. Es lo más parecido a interactuar con un humano.

## DALL·E 3 y Búsqueda Web

**Imágenes (DALL·E 3):**
- Pide descripciones exactas: *"diagrama celular estilo neón"*.
- Excelente para crear *moodboards* y recursos visuales.

**Búsqueda (Browsing):**
- Usa "🌐 Search" para obligarlo a buscar datos actuales.
- **Limitación:** Si el modelo no busca automáticamente, dáselo explícito: *"Busca en internet..."*.

## GPTs personalizados y Canvas

**GPTs (Tu asistente a medida):**
- Son configuraciones guardadas con prompts y archivos predefinidos.
- No entrenan al modelo base; solo le ponen "un uniforme y reglas".
- Lo veremos a fondo al construir tu tutor en la Unidad 7.

**Canvas (Editor lateral):**
- Panel estilo Google Docs para textos largos o código.
- Modifica secciones que selecciones sin regenerar todo el documento.
- Ofrece atajos: "acortar", "traducir", "agregar comentarios".
