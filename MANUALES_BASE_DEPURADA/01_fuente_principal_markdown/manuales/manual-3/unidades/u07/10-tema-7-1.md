# 7.1 GPTs personalizados: instrucciones, knowledge files y actions

Un **GPT** es una versión personalizada de ChatGPT.
No reentrenas el modelo, pero le das instrucciones detalladas, archivos como knowledge y capabilities específicas.
Es la forma más accesible de construir un asistente IA propio.

::concepto{titulo="GPT personalizado"}
Un **GPT** es una configuración guardada en ChatGPT que combina:
- **Identidad:** Nombre, descripción y avatar.
- **Instructions:** System prompt detallado (qué debe hacer, qué no, tono, formato).
- **Knowledge:** Hasta 20 archivos que el GPT puede consultar (PDFs, docx, txt, csv).
- **Capabilities:** Web browsing, DALL·E (genera imágenes), Code Interpreter (corre Python).
- **Actions:** Conexiones a APIs externas (avanzado).
- **Distribución:** Privado (solo tú), link compartido, o público en GPT Store.

**Acceso:** Requiere ChatGPT Plus ($20/mes).
El Free tier solo permite usar GPTs ajenos.
::/concepto

::interioriza{titulo="Baja curva: El empleado de nuevo ingreso"}
Crear un GPT es como contratar a un empleado de nuevo ingreso.
No le enseñas a hablar español o a usar una computadora (el modelo base ya sabe eso).
Le das un gafete (Identidad), un manual de inducción (Instructions), sus carpetas de proyectos (Knowledge) y las llaves de ciertas oficinas (Capabilities/Actions).
::/interioriza

### Capa 1 — Identidad

- Name: Tutor Física Albatros
- Description: Tu tutor personal de física para bachillerato mexicano. Especializado en mecánica, termodinámica y electromagnetismo.
- Avatar: [genera con DALL·E o sube imagen]

### Capa 2 — Instructions (la más importante)

Aquí escribes el **system prompt completo**.
Se recomienda usar entre 500-2000 palabras estructuradas así:

- **Rol y experticia:** Quién es y qué sabe (ej. Tutor experto en física bachillerato).
- **Personalidad y tono:** Paciente, usa analogías, no es condescendiente.
- **Metodología pedagógica:** Preguntar antes de explicar, analogía -> concepto -> ejemplo.
- **Casos de excepción:** Qué hacer si preguntan cosas fuera de tema o piden la tarea hecha.
- **Formato de respuesta:** Longitud máxima, uso de Markdown, negritas.
- **Lo que no hace:** No inventar datos, no hacer tareas completas.

### Capa 3 — Knowledge

Carga hasta 20 archivos (cada uno hasta 512 MB).
Formatos soportados: PDF, docx, xlsx, csv, txt.

- Carga **lo esencial** del curso, no toda la biblioteca.
- Documentos bien estructurados son consultados mejor.
- Evita PDFs escaneados de baja calidad (OCR malo).

### Capa 4 — Capabilities

Activa según necesidad:
- **Web Browsing:** Útil si tu GPT necesita info actualizada.
- **DALL·E Image Generation:** Si necesitas generar diagramas.
- **Code Interpreter:** Python para cálculos numéricos, gráficos.

> **Tip:** Si tu GPT es un tutor de física, activa **Code Interpreter**. Podrá graficar funciones y resolver ecuaciones.

### Capa 5 — Actions (avanzado)

Conecta APIs externas a tu GPT.
- Wolfram Alpha para cálculo avanzado.
- Tu propia API de sistema externo.
- APIs públicas como clima o traducción.

> **Tip:** Para principiantes, omite las actions hasta dominar lo básico.

### Conversation Starters

Son 4 frases predefinidas al iniciar el chat.
Sirven para que el usuario sepa qué puede preguntar.
Ejemplo: "Explícame el principio de Bernoulli" o "Hazme un quiz del tema".

## Cómo construir paso a paso

1. Ve a chatgpt.com → Explore GPTs → Create.
2. Usa el Modo "Configure" para mayor control.
3. Pega tus Instructions.
4. Sube tus archivos en Knowledge.
5. Activa las Capabilities necesarias.
6. Agrega tus 4 Conversation starters.
7. Genera un avatar y prueba en el preview.
8. Guarda y elige visibilidad.

## Distribución y limitaciones

**Tres niveles de distribución:**
- **Only me:** Privado, ideal para empezar.
- **Anyone with link:** Compartir por enlace con tu equipo.
- **GPT Store:** Público, indexado y descubrible.

**Limitaciones:**
- Solo usuarios con ChatGPT Plus pueden usarlo libremente.
- El Knowledge no se actualiza solo.
- No mantiene memoria entre distintos usuarios.
- **Privacidad:** Cualquiera que use el GPT puede extraer tu Knowledge.
