# Unidad 03

## Prompt Engineering Fundamental

> El **mismo modelo** de IA, con dos prompts distintos, da respuestas que parecen venir de dos asistentes diferentes —uno brillante y uno mediocre. La diferencia está en cómo le hablas. Esta unidad te entrena a hablarle bien.
