# 3.1 Anatomía de un prompt profesional

Un prompt profesional **no es una pregunta**, es una **especificación**.
Posee hasta 6 capas claramente definidas para guiar al modelo.
Si omites capas, el modelo adivina y entrega resultados mediocres.

::concepto{titulo="Capas de un prompt profesional"}
Un prompt completo tiene seis capas:
- **Rol:** Quién quieres que el modelo sea.
- **Tarea:** Qué quieres que haga exactamente.
- **Contexto:** Quién eres tú y para qué lo necesitas.
- **Restricciones:** Límites de longitud, idioma o tono.
- **Formato:** Cómo quieres la estructura (tablas, listas, etc.).
- **Ejemplos:** Casos de uso ideales (few-shot).
> Mnemotécnico **R-T-C-R-F-E**: "Rol-Tarea-Contexto-Restricciones-Formato-Ejemplos".
::/concepto

::interioriza{titulo="¿Por qué te importa?"}
Imagina pedir una pizza diciendo "tengo hambre". El pizzero tendrá que adivinar el tamaño y los ingredientes.
Las 6 capas son como el pedido exacto: "Una pizza grande, de pepperoni, para recoger en 20 minutos".
Le ahorras al modelo tener que adivinar y obtienes exactamente lo que necesitas a la primera.
::/interioriza

## Capa 1 — Rol

Define **quién** debe ser el modelo en ese momento.
Cambia radicalmente su tono, vocabulario y profundidad técnica.
- Ejemplo: `Eres un tutor de física experto en mecánica clásica.`
- Resultado: Ajusta su nivel y deja de ser genérico.

## Capa 2 — Tarea

Indica **qué** quieres que haga usando verbos de acción explícitos.
Prefiere verbos como "explica", "analiza" o "compara".
- Ejemplo: `Explícame el principio de Bernoulli.`
- Evita: "Háblame de física" (es demasiado ambiguo).

## Capa 3 — Contexto

Explica **quién eres**, qué sabes y para qué necesitas la información.
Reduce alucinaciones y permite una personalización extrema.
- Ejemplo: `Soy estudiante y tengo examen el viernes. Ya conozco F=ma.`
- Regla: Incluye solo el contexto directamente relevante.

## Capa 4 — Restricciones

Establece lo que el modelo **NO** debe hacer y sus límites.
- `Máximo 500 palabras.`
- `Español neutro.`
- `No uses cálculo diferencial.`

## Capa 5 — Formato

Detalla la estructura exacta que esperas recibir en la respuesta.
Hace que el resultado final sea scaneable y reutilizable.
- `Estructura tu respuesta en: 1. Definición. 2. Analogía. 3. Fórmula. 4. Ejemplo.`

## Capa 6 — Ejemplos (few-shot)

Muestra 1-3 ejemplos del "input-output" ideal.
Vital cuando la tarea es repetitiva o el formato no es obvio.
(Se profundizará en el subtema 3.2.)

## Ejemplo de prompt completo

::visual{tipo="infografia" descripcion="Anatomía completa de un prompt profesional descompuesta en 6 capas con colores distintos: 1) ROL azul. 2) TAREA verde. 3) CONTEXTO amarillo. 4) RESTRICCIONES naranja. 5) FORMATO morado. 6) EJEMPLOS rosa. Cada capa muestra el texto literal del prompt de Bernoulli aplicado. Al final, flecha hacia 'PROMPT FINAL ENVIADO AL MODELO'." paginas=1}

```text
[ROL] Eres un tutor de física para bachillerato.
[TAREA] Explica el principio de Bernoulli en alerones de F1.
[CONTEXTO] Tengo 16 años, examen el viernes, conozco energía cinética.
[RESTRICCIONES] Máximo 500 palabras, sin cálculo diferencial.
[FORMATO] 1. Definición, 2. Analogía, 3. Ejemplo numérico.
```

## Comparativa rápida

::visual{tipo="cuadro-comparativo" descripcion="Tabla comparativa de Prompt Malo vs Bueno con dos columnas: PROMPT MALO 'ayúdame con la tarea de física, es Bernoulli' → respuesta esperada vaga, definición de Wikipedia genérica. PROMPT BUENO con las 6 capas estructuradas → respuesta esperada detallada, personalizada al case study, con ejemplo F1 numérico y pregunta de comprobación. - **Prompt Malo:** "Ayúdame con Bernoulli". Respuestas vagas. Mucha iteración.
- **Prompt Bueno:** Usa las 6 capas. Respuestas a medida. Funciona al primer intento.
> *"Si la respuesta falla, el problema suele estar en el prompt."*

::pausa{titulo="💭 Pausa para pensar"}
1. Si un modelo te da una respuesta muy técnica que no entiendes, ¿qué capa falló y cómo la arreglas?
2. Deduce: ¿Por qué agregar "ejemplos" reduce alucinaciones en tareas repetitivas?
3. Convierte esta petición en un prompt de 4 capas: "Quiero una rutina para el gimnasio".
::/pausa
