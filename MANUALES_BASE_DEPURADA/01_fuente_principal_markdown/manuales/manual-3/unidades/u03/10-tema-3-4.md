# 3.4 Refinamiento iterativo: cómo mejorar un prompt sin empezar de cero

La primera versión de un prompt **rara vez** es la definitiva.
Los profesionales **iteran**: prueban, observan, ajustan y repiten.
Esta sección te enseña el ciclo completo para iterar de manera eficiente.

::concepto{titulo="Loop de refinamiento de prompt"}
El proceso profesional tiene 5 pasos cíclicos:
* **Escribir:** Crea tu prompt v1 con la anatomía R-T-C-R-F-E.
* **Ejecutar:** Envía el prompt y observa la respuesta de la IA.
* **Evaluar:** Analiza si cumple con tus criterios definidos.
* **Refinar:** Identifica la causa del problema y ajusta solo eso.
* **Re-ejecutar:** Prueba de nuevo y compara con la v1.

Si mejora, lo guardas. Si no, vuelves a refinar.
::/concepto

::interioriza{titulo="Analogía: Afinando un instrumento"}
Iterar un prompt es como afinar un instrumento de cuerda.
No cambias todas las cuerdas si solo una desafina ligeramente.
Ajustas una variable a la vez, escuchas, y vuelves a probar.
::/interioriza

### Criterios de evaluación (Paso 3)

No te limites a decir "no me gustó". Mide objetivamente:
* **Precisión factual:** ¿Los datos son correctos?
* **Formato:** ¿Sigue la estructura pedida (JSON, tabla)?
* **Tono:** ¿Suena al rol asignado o es genérico?
* **Alucinaciones:** ¿Inventa citas o números?

### Refinamiento preciso (Paso 4)

Realiza ajustes como un cirujano, atacando la causa raíz:
* **Si es vaga:** Añade contexto específico.
* **Si inventa datos:** Limita su alcance o activa búsqueda.
* **Si es larga:** Añade una restricción de palabras.
* **Si el tono es frío:** Suma adjetivos de personalidad.

> **Regla de oro:** Nunca borres todo y empieces de cero.
> Cambia **una variable** por iteración para entender qué funciona.

## Meta-prompts

Truco experto: **la propia IA puede mejorar tu prompt**.

```text
Critica este prompt: [tu prompt v1].
Mira la respuesta que generaste: [tu output].
Sugiere una v2 mejorada que sea más precisa.
```

> Usa "prompt sobre prompt" cuando no sepas qué más ajustar.

## Versionado de Prompts

Mantén un registro de tus prompts ganadores:
* **v1:** Rol + tarea + contexto (Output genérico)
* **v2:** Añadí formato 5-pasos (Output estructurado)
* **v3:** Añadí ejemplo (Output aplicado al caso) -> **¡GUARDADO!**
