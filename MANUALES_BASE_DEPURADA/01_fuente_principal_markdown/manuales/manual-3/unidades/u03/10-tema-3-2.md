# 3.2 Zero-shot, few-shot, chain-of-thought y técnicas avanzadas

- Un modelo puede responder como **escolar promedio** o como **PhD**.
- Todo depende de la técnica de prompting que utilices.
- Esta sección te enseña las cinco estrategias más importantes.

::concepto{titulo="Técnicas de prompting"}
- **Zero-shot:** Pides directo sin ejemplos. Usa su conocimiento base.

- **Few-shot:** Das 2-5 ejemplos de formato, luego preguntas. Imita el patrón.

- **Chain-of-thought (CoT):** Pides que piense paso a paso. Mágico en lógica.

- **Self-consistency:** Preguntas varias veces y tomas el voto mayoritario.

- **ReAct (Reason + Act):** Alterna el razonamiento con uso de herramientas.
::/concepto

::interioriza{titulo="Analogía: Las herramientas del Chef"}
- Piensa en el prompting como utensilios de cocina.
- **Zero-shot** es un cuchillo básico: rápido y sirve para casi todo.
- **Chain-of-thought** es una olla de cocción lenta: extrae todo el sabor (precisión) pero toma tiempo.
- Usar "razona paso a paso" para preguntar la hora es como usar una licuadora industrial para remover el café. ¡Puro desperdicio de tokens!
::/interioriza

## Zero-shot

- Es el **modo por defecto**. No das ejemplos ni pasos previos.
- Funciona excelente para tareas comunes y bien definidas.

```
Clasifica el siguiente tweet como POSITIVO, NEUTRAL o NEGATIVO:

"El nuevo procesador es rapidísimo, pero la batería dura nada."
```

- **Falla** cuando el formato es ambiguo o no resulta obvio.

## Few-shot

- Das **ejemplos del input y output deseado** antes de tu instrucción final.
- El modelo aprende el patrón de inmediato sin necesidad de reentrenamiento.

```
Clasifica el sentimiento del tweet como POSITIVO, NEUTRAL o NEGATIVO.

Ejemplos:
Tweet: "Excelente servicio, lo recomiendo."
Sentimiento: POSITIVO

Tweet: "El producto llegó tarde y dañado."
Sentimiento: NEGATIVO

Tweet: "El paquete llegó hoy a las 3pm."
Sentimiento: NEUTRAL

Tweet: "El nuevo procesador es rapidísimo, pero la batería dura nada."
Sentimiento:
```

- **Ideal para:** clasificación, estilos de código o traducciones con tono específico.

- **Tip pro:** Usa 3 a 5 ejemplos. Más de 8 no mejora el resultado y gasta tokens.

## Chain-of-thought (CoT)

- Le pides que **razone paso a paso** antes de entregar la conclusión.
- Es vital para resolver **matemáticas, lógica y planeación**.

```
Pregunta: María tiene 23 manzanas. Compra 47 más, regala 18 y se le caen 5.
¿Cuántas manzanas tiene al final?

Piensa paso a paso antes de responder.
```

- Reduce drásticamente los errores por adelantarse a la respuesta.

- **Variantes mágicas de CoT:**
  - "*Let's think step by step.*" (Kojima, 2022).
  - "*Razona como un experto en el tema antes de responder.*"
  - "*Lista tus pasos numerados y luego concluye.*"

::visual{tipo="ilustracion" descripcion="Tres técnicas en columnas paralelas comparando aplicación a la misma pregunta de matemáticas: 1) ZERO-SHOT — pregunta directa, respuesta inmediata (a veces incorrecta en problemas complejos). 2) FEW-SHOT — usuario da 2 ejemplos resueltos primero, luego pregunta nueva; modelo imita el patrón paso a paso. 3) CHAIN-OF-THOUGHT — usuario añade 'piensa paso a paso'; modelo escribe razonamiento explícito antes de la respuesta final. Resultados: zero-shot 60% precisión, few-shot 75%, CoT 85% en problemas matemáticos." paginas=1}

## Self-consistency

- Haces la misma pregunta **5 veces** con una temperatura mayor a cero.
- Tomas la respuesta más frecuente (consenso).

```
[Mismo prompt repetido 5 veces]
→ Respuesta 1: 47
→ Respuesta 2: 47
→ Respuesta 3: 48
→ Respuesta 4: 47
→ Respuesta 5: 47

Mayoría: 47 (4 de 5).
```

- **Ventaja:** Elimina errores aleatorios del modelo.
- **Desventaja:** Cuesta 5 veces más. Úsalo solo para decisiones críticas.

## ReAct (Reason + Act)

- El modelo **alterna razonamiento lógico con acciones externas**.
- Piensa, usa un buscador o API, observa el resultado y vuelve a pensar.
- **Flujo:** Thought → Action → Observation → Thought → Final Answer.

```
Thought: Necesito averiguar la población actual de México.
Action: search("población de México 2025")
Observation: 130 millones según INEGI 2024.
Thought: Ahora dividir entre PIB. Necesito el PIB.
Action: search("PIB México 2024")
Observation: 1.79 trillones USD.
Thought: PIB per cápita = 1.79e12 / 130e6 = 13,769 USD.
Final Answer: PIB per cápita 2024 ≈ 13,769 USD.
```

- Es la arquitectura base de los **agentes autónomos**.

## Tree-of-thoughts (avanzado)

- El modelo explora **múltiples ramas de razonamiento** simultáneamente.
- Evalúa el progreso de cada rama y descarta los callejones sin salida.
- Útil en rompecabezas, juegos de estrategia y planificación compleja.

::visual{tipo="ilustracion" descripcion="Técnicas avanzadas en 3 paneles: 1) SELF-CONSISTENCY — mismo prompt enviado 5 veces, 5 respuestas, voto mayoritario destacado. 2) ReAct — diagrama de alternancia Thought→Action→Observation→Thought→...→Final Answer. 3) TREE-OF-THOUGHTS — árbol de decisión donde modelo explora 3 ramas de razonamiento, evalúa cada una, elige la mejor. Cada técnica con caso de uso ideal y costo aproximado en tokens." paginas=1}

## Cuándo usar cada una

| Tarea | Técnica recomendada |
|---|---|
| Pregunta simple o general | **Zero-shot** |
| Clasificación / Formato custom | **Few-shot** |
| Matemáticas y Lógica | **Chain-of-thought** |
| Decisiones de alto riesgo | **Self-consistency** |
| Requiere consultar datos | **ReAct** |
| Puzzles y múltiples caminos | **Tree-of-thoughts** |

- **Dato extra:** Las técnicas son combinables. Un "Few-shot CoT" es extremadamente poderoso.
