# 4.1 Imagen: DALL·E 3, Midjourney, Firefly, Stable Diffusion y Flux

- Hace 5 años, el "fotorrealismo" IA era pura ciencia ficción.
- Hoy, un prompt de 50 palabras crea un render comercial.
- Aprenderás a dominar las **cinco plataformas** principales de 2025.

::concepto{titulo="Generación de imagen por IA"}
- Los **generadores de imagen** usan arquitecturas de **diffusion**.
- Se entrenan analizando millones de pares imagen-texto.
- Su función es generar imágenes nuevas a partir de tus descripciones.

**Las 5 plataformas líderes (2025):**
- **DALL·E 3** (OpenAI): Integrado en ChatGPT, muy conversacional y fácil.
- **Midjourney v6**: Calidad estética superior, operado vía Discord.
- **Adobe Firefly**: Seguro para uso comercial (entrenado con Adobe Stock).
- **Stable Diffusion XL / Flux**: Código abierto, autohospedables y modificables.
- **Imagen 3** (Google): Integrado en la suite de Gemini Advanced.
::/concepto

::interioriza{titulo="Analogía: El Director de Fotografía"}
- Piensa en la IA como un director de fotografía con infinito talento pero ciego.
- Si le dices "quiero una foto bonita", improvisará algo genérico.
- Si le das instrucciones exactas (sujeto, iluminación, lente), creará la obra maestra para tu tutor.
::/interioriza

## Anatomía de un prompt visual

- A diferencia de los prompts de texto, aquí no usas la estructura tradicional.
- Un prompt visual efectivo se compone de **5 capas** específicas.

1. **Sujeto:** Qué objeto o persona quieres ver.
2. **Escenario:** En qué lugar o contexto sucede la escena.
3. **Estilo:** Si es fotorrealista, ilustración, render 3D, etc.
4. **Composición:** El ángulo de cámara, encuadre o perspectiva.
5. **Parámetros:** Instrucciones técnicas de iluminación, resolución o formato.

### Ejemplo concatenado

> *Una molécula de agua H₂O (Sujeto), flotando en el espacio profundo (Escenario), render 3D fotorrealista (Estilo), plano medio (Composición), iluminación cinematográfica suave (Parámetros).*

## Estilos y trucos de profesional

| Uso de tu tutor | Estilo recomendado |
|---|---|
| Ilustración científica | "render 3D estilo educativo, paleta limpia" |
| Infografías | "infografía vectorial estilo plana, fondo blanco" |
| Avatares | "ilustración de personaje estilizada, paleta plana" |

**Trucos clave:**
- **Negativos:** Usa comandos como `--no text` o `--no watermark` para excluir elementos.
- **Aspect Ratio:** Define el formato con `--ar 16:9` (video) o `--ar 9:16` (móvil).
- **Semilla (Seed):** Fija un valor (ej. `--seed 123`) para mantener coherencia en variaciones.
- **Iteración:** Empieza con un prompt simple y añade capas progresivamente.

## Limitaciones actuales

- **Texto en imagen:** Las IAs fallan al escribir palabras; DALL·E 3 suele ser el más preciso.
- **Anatomía humana:** Es común encontrar dedos extra o proporciones extrañas.
- **Coherencia temporal:** Mantener al mismo personaje en varias imágenes requiere técnicas avanzadas.

::pausa{titulo="💭 Pausa para deducción"}
1. Debes ilustrar un manual médico para un cliente corporativo que exige estricto cumplimiento de licencias comerciales. ¿Cuál de las 5 herramientas eliges y por qué?
2. Has generado un avatar para tu tutor virtual, pero al pedir una nueva postura cambió su rostro por completo. ¿Qué parámetro técnico de Midjourney o SD olvidaste configurar?
3. Un compañero usa el prompt: "Gato cyberpunk". Basado en las 5 capas, ¿qué información esencial le falta para evitar resultados aleatorios?
::/pausa
