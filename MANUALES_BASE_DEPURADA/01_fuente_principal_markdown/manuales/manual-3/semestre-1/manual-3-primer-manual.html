Historia conductora

Un estudiante escucha que "la IA lo puede hacer todo": resumir, explicar, programar, dibujar y hasta dar consejos. Quiere construir un tutor IA personal, pero primero debe entender que tipo de sistema esta usando.

El tutor no aparece de golpe. En esta unidad se construye la base mental: que es inteligencia artificial, como se diferencia de machine learning y deep learning, por que hubo inviernos de IA y como los LLM actuales cambiaron la forma de aprender.

  • Por que no toda IA es ChatGPT?
  • Que significa que un modelo "aprenda"?
  • Como se reconoce una respuesta util pero no verificada?

Definicion operativa de IA

Para este manual, IA es un sistema que realiza tareas que normalmente asociamos con inteligencia humana: clasificar, recomendar, reconocer patrones, generar texto o ayudar a decidir. No se define por magia, sino por tarea, datos, modelo y salida.

TerminoExplicacion breveEjemplo
IASistema que automatiza una tarea cognitiva.Responder preguntas.
MLModelo que mejora patrones a partir de datos.Predecir una calificacion.
DLModelos con redes profundas.Reconocer imagenes.
LLMModelo de lenguaje grande.Generar explicaciones.

Historia: avances, pausas y nuevos saltos

La IA no nacio con los chats. Hubo ideas tempranas, promesas exageradas, periodos de poco financiamiento e impulsos recientes por datos, computo y modelos fundacionales.

La lectura importante es que la IA avanza por ciclos: una idea necesita herramientas, datos y pruebas. Por eso conviene preguntar que tecnologia sostiene cada promesa.

Tipos de IA

La IA actual es estrecha: resuelve tareas especificas. Puede parecer general porque conversa de muchos temas, pero no tiene autonomia total ni comprension humana.

TipoEstadoCuidado
IA estrechaExiste.Hace tareas concretas.
IA generalNo disponible como humano general.No confundir fluidez con comprension total.
Super IAHipotetica.No usar como base para practicas escolares.

Como aprende un modelo

Un modelo no memoriza una clase como un alumno. Ajusta parametros a partir de ejemplos para producir salidas probables. La analogia del aprendiz ayuda: observa muchos ejemplos, practica y corrige patrones, pero aun puede equivocarse.

M3 IA BasicaPagina 6

Modelos fundacionales y LLM

Un modelo fundacional se entrena con gran cantidad de datos y despues se adapta a muchas tareas. Un LLM se especializa en lenguaje: predice, transforma y genera texto.

Nota de uso responsable

Un LLM puede sonar seguro aunque este equivocado. El tutor IA personal debe citar fuentes, pedir contexto y reconocer cuando no sabe.
