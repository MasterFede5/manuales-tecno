Cómo usar este manual

Este manual está diseñado para que avances de forma gradual: primero la teoría, después la práctica y al final la integración en proyectos.

Lee el objetivo y activa conocimientos previos.
Estudia los conceptos fundamentales.
Realiza mini actividades y ejercicios.
Desarrolla prácticas integradoras.
Finaliza con el reto digital.
Recomendación

Trabaja en equipo, documenta tus resultados y mantén un cuaderno de laboratorio.

Prólogo

La física es el lenguaje del universo. Este manual busca que cada estudiante domine ese lenguaje y lo utilice para crear soluciones reales. La curiosidad es la chispa; el método científico es el camino.

En Albatros creemos en el aprendizaje significativo: experimentar, analizar y construir. Cada unidad está pensada para conectar teoría y práctica con desafíos que inspiran a pensar como científicos e ingenieros.

Perfil de egreso y evaluación

Al concluir el manual, el estudiante será capaz de:

  • Aplicar principios físicos para resolver problemas reales.
  • Diseñar sistemas con enfoque en eficiencia energética.
  • Integrar sensores, actuadores y control en proyectos.
  • Argumentar decisiones técnicas con bases científicas.
Evaluación

La evaluación se basa en actividades, prácticas integradoras, retos digitales y participación en laboratorio. Se valora tanto el proceso como el resultado final.
